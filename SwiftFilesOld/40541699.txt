Need to change date string into "date string" as Date then you can change into your particular format. For every format you need to do this first. You can use the code below. Swift 3 extension String { func changeDate(_ mydate:String) -> String { let dateFormatter = DateFormatter() dateFormatter.dateStyle = DateFormatter.Style.long dateFormatter.dateFormat = "yyyy-MM-dd" let convertedDate = dateFormatter.date(from: mydate) dateFormatter.dateFormat = "dd-MM-yyyy" let date = dateFormatter.string(from: convertedDate!) return date } }
