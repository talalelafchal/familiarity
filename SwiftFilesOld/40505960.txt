Swift - Finding words in string and adding to array
I'm looking for a way to loop through a string and pull out words into an array where a certain prefix and suffix is contained. For example, lets take the following string; let str = "A Lorem @!ipsum!@ dolor sit amet, @!consectetur!@ adipiscing elit" I need a way to search the text and output both the words ipsum and consectetur into an array to be used within another function later on.
