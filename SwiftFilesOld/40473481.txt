You cannot use a table view cell initialized with the default initializer UITableViewCell() Reuse the cell, add an identifier (e.g. PeopleCell) in Interface Builder. let cell = tableView.dequeueReusableCell(withIdentifier: "PeopleCell", for: indexPath) And make sure that datasource and delegate of the table view are connected in Interface Builder, too.
