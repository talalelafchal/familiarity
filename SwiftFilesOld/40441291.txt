TableView Section Change Header Alignment
I have This tableView Here : With the default left text alignment which i would like to change to a right text alignment . How can i do this ? EDIT: Now it looks like this :
You have to do this in UITableView Datasource method viewForHeaderInSection: Swift 2.3: func tableView(tableView: UITableView, viewForHeaderInSection section: Int) -> UIView? { let headerText = UILabel() headerText.textColor = UIColor.lightGrayColor() headerText.adjustsFontSizeToFitWidth = true switch section{ case 0: headerText.textAlignment = .Center headerText.text = "This Header Will Be Centered" case 1: headerText.textAlignment = .Right headerText.text = "This Header Will Be Aligned Right" default: headerText.textAlignment = .Left headerText.text = "Default Will Be Left" } return headerText } EDIT: Modified the code above. You can use the section argument to identify the section you want to modify.
