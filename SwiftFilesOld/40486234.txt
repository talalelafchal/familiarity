For the first part, be sure to assign the class of the view controller in the storyboards, otherwise, Xcode doesn't allow you to create @IBOutlets by dragging the label. The second one, the function is expected to return a UITableViewCell, and you're not returning anything. Also, if you're using Swift 3, this is the new syntax. A quick fix for you to check would be the following: func tableView(_ tableView: UITableView, cellForRowAt indexPath: IndexPath) -> UITableViewCell { return UITableViewCell() }
