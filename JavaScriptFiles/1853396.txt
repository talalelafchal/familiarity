Javascript showing the function text instead of printing value on screen
<pre> function hex(x,y,side,isLast,color) {//Hex object constructor. this.x = x; this.y = y; this.side = side; this.isLast = isLast; this.color = color; function multiply() { return this.x * this.y; } this.multiply = multiply; } var hexagon = new hex(22,22,20,0,1); document.write(hexagon.multiply); </pre> <p>When loading index.htm, results that writes on screen the function instead of the returning value:</p> <p>function multiply() { return this.x * this.y; }</p> <p>:(</p>
<p>You forgot the ():</p> <pre><code>document.write(hexagon.multiply()); </code></pre> <p>If you don't use (), Javascript will treat <code>multiply</code> as a variable and write out it contents - in this case, the code of the function.</p>
<p>You have to make sure that your javascript code is in <code>&lt;script&gt;</code> and <code>&lt;/script&gt;</code> tags. So, it might read:</p> <pre><code>&lt;html&gt;&lt;head&gt;&lt;script type="text/javascript"&gt; function hex(x,y,side,isLast,color) {//Hex object constructor. this.x = x; this.y = y; this.side = side; this.isLast = isLast; this.color = color; function multiply() { return this.x * this.y; } this.multiply = multiply; } var hexagon = new hex(22,22,20,0,1); document.write(hexagon.multiply) &lt;/script&gt; &lt;body&gt; &lt;!--Content here--&gt; &lt;/body&gt; &lt;/html&gt; </code></pre>
