click links though javascript
<p>Am looking for how to click links on my page using javascript. In my page i show each record from my database . and a 'show' link to the right clicking it will expand and display the row in detail. i want to expand all the rows at once . Is it possible to click link in Javascript like this?</p>
<p>You can cycle through all of the anchors, and click them, yes:</p> <pre><code>var myLinks = document.getElementsByTagName("a"); for (var i = 0; i &lt; myLinks.length; i++) { myLinks[i].click(); } </code></pre> <p>Note that this will click <em>every</em> link on the page. You can focus more on a specific set of links by going through a parent container that sits just outside of them:</p> <pre><code>var myLinks = document.getElementById("container").getElementsByTagName("a"); </code></pre> <p>This would get all links within the element having the <code>id</code> value of "container".</p> <p><strong>Update2:</strong></p> <p>Within the comments, you wanted to know how to click only links whose <code>ID</code> begins with a specific string. Let's look at a couple ways we can implement that into our aforementioned code:</p> <p>Using jQuery, you could click all <code>id='reviews-show...'</code> links like this:</p> <pre><code>$("a[id^='reviews-show']").click(); // simple, eh? </code></pre> <p>The <code>^=</code> in this selector matches in a regular-expression style, asking if the string <em>starts</em> with "reviews-show," which will retrieve only the <code>a</code> tags for which this statement is true.</p> <p>If you want to stick with just regular javascript, we can modifiy our first code-block to check the substring:</p> <pre><code>var myLinks = document.getElementsByTagName("a"); for (var i = 0; i &lt; myLinks.length; i++) { currentlink = myLinks[i]; if (currentlink.id.substring(0,12) === "reviews-show") { currentlink.click(); } } </code></pre> <p><strong>Update:</strong></p> <p>Apparently <a href="http://jquery.com" rel="nofollow noreferrer">jQuery</a> is rotting my mind, and my solution doesn't work as-is. If you don't mind using jQuery (you'd love it), you could go this route:</p> <pre><code>$("a").click(); //clicks all links on page $("#container a").click();//clicks all links within element having id 'container' </code></pre> <p>If you want to stay away from a framework, try adding this prototype extension into the top of your code which will make my original answer sufficient:</p> <pre><code>HTMLElement.prototype.click = function() { if (document.createEvent) { var evt = this.ownerDocument.createEvent('MouseEvents'); evt.initMouseEvent('click', true, true, this.ownerDocument.defaultView, 1, 0, 0, 0, 0, false, false, false, false, 0, null); this.dispatchEvent(evt); } else if (this.fireEvent) { this.fireEvent("onclick"); } } </code></pre> <p>Source: <a href="http://www.barattalo.it/2009/11/18/click-links-with-javascript/" rel="nofollow noreferrer">http://www.barattalo.it/2009/11/18/click-links-with-javascript/</a></p>
