Javascript validation on multiple input controls
<p>i have arrays of input controls and want to check on submit that they should not be empty</p> <p> </p>
<p>Here's an example function:</p> <pre><code>function valid( array ) { if( typeof array != 'object' || array.length &lt; 1 ) return false; // some basic error checking var errors = []; var alen = array.length; for( var a = 0; a &lt; alen; a++ ) { if( array[ a ].value.length &lt; 1 ) errors.push( array[ a ] ); } if( errors.length &gt;= 1 ) return errors; return true; } </code></pre> <p>Usage:</p> <pre><code>function submitForm() { var inputs = document.getElementsByTagName( 'input' ); var errors = valid( inputs ); if( errors ) { var elen = errors.length; for( var e = 0; e &lt; elen; e++ ) { errors[ e ].className = "error"; } event.preventDefault(); return false; } } var form = document.getElementById( 'myForm' ).onsubmit = submitForm; </code></pre> <p>If you were using a framework, this would be even easier.</p>
