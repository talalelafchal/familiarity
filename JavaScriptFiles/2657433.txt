replace space with dash JavaScript
<pre><code>var html = "&lt;div&gt;"+title+"&lt;br/&gt;"; document.write(title.replace(/ /g,"-")); html+= '&lt;p&gt;&lt;a href="go.aspx?title=' + title + '"&gt;Details&lt;\/a&gt;&lt;/p&gt;&lt;/div&gt;'; </code></pre> <p>I want to replace <strong>title</strong> space with dash.</p>
<p>Try <code>title.replace(/\s/g , "-")</code> instead. (<code>/\s/</code> is the regex escape for whitespace).</p> <p>Also, do:</p> <pre><code>title = title.replace(/\s/g , "-"); var html = "&lt;div&gt;" + title + "&lt;/div&gt;"; // ... </code></pre>
<p>Calling <code>title.replace</code> will not change title, but return a string where the values have been replaced. You need to use the returned value:</p> <pre><code>var html = "&lt;div&gt;"+title+"&lt;br/&gt;"; var newTitle = document.write(title.replace(/ /g,"-")); html+= '&lt;p&gt;&lt;a href="go.aspx?title=' + newTitle + '"&gt;Details&lt;\/a&gt;&lt;/p&gt;&lt;/div&gt;'; </code></pre> <p>The regular expression is fine, but will only replace spaces and not all whitespace.</p>
