Splitting a text using javascript
<p>Hi I am having string called 'santhosh'. I wish to split this text as 's,a,n,t,h,o,s,h' using javascript. Is it possible? And want to save it in an array.</p> <p>And I wish to check whether the splitted character is string or Numeric is it possible?</p>
<p>You can use an empty string as the separator argument of the <a href="https://developer.mozilla.org/En/Core_JavaScript_1.5_Reference/Objects/String/Split" rel="nofollow noreferrer"><code>split</code></a> method:</p> <pre><code>var array = "santhosh".split(''); // ["s", "a", "n", "t", "h", "o", "s", "h"] </code></pre>
<pre><code>&lt;script type="text/javascript"&gt; var myString = "Hello"; var mySplitResult = myString.split(''); document.write("The first element is " + mySplitResult[0]); document.write("&lt;br /&gt; The second element is " + mySplitResult[1]); document.write("&lt;br /&gt; The third forth is " + mySplitResult[2]); document.write("&lt;br /&gt; The fourth element is " + mySplitResult[3]); document.write("&lt;br /&gt; The fifth element is " + mySplitResult[4]); &lt;/script&gt; </code></pre>
<p>The following code snippet returns an array object:</p> <pre><code>"santhosh".split('') // s,a,n,t,h,o,s,h </code></pre>
<p>You can use the following function:</p> <p>string.split(separator, limit)</p> <p>-just give the seperators as "" , it will split all charactors in the string.</p>
