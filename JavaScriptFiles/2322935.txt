<p>There is no int type in JavaScript. Only number. parseInt is fine if you have a non-decimal number, or you might have trailing garbage on the string like "5asdkfkasdk". But for the common case, when you know your string is only decimal digits, I prefer to use the Number() constructor. It looks more elegant and it's more familiar to C and Python programmers who do a lot of casting.</p> <pre><code>var numericValue = Number("100000"); var numericValue2 = parseInt("100000", 10); </code></pre> <p>These yield the same result of the same type.</p>
<p>If you are sure that your string contains int you can use just one +</p> <pre><code>var n = "455"; n = +n; // now n is integer </code></pre>
