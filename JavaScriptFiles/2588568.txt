How do I use JavaScript (not jQuery) to make something display:none?
<p>Suppose I have an element called <code>"#container-main"</code>. How do I make that <code>display:none</code> NOT using jQuery?</p>
<pre><code>document.getElementById('container-main').style.display = 'none'; </code></pre>
<p>I'm assuming that you mean the <code>id</code> attribute of the container is <code>container-main</code>. Then in JavaScript you do</p> <pre><code>document.getElementById('container-main').style.display = 'none'; </code></pre>
<p>edited: <code>document.getElementById('container-main').style.display = 'none';</code></p>
<p>You could also use</p> <pre><code>document.getElementById('container-main').style.visibility = "hidden"; </code></pre>
<p>Although the other answers are correct, for the sake of preventing a potential error, I would recommend getting a reference to your DOM element first, then making sure it exists and has properties, rather than encapsulating everything in a one liner. If, for some reason, the element with id 'container-main' no longer existed on the page, the suggested one-liner will create a JS error.</p> <pre><code>var myNode = document.getElementById('container-main'); // Attempt to get the reference first if ( myNode &amp;&amp; myNode.style ) { // make sure you did get an element, and if so... myNode.style.display = 'none'; // then operate on it's properties. } </code></pre> <p>Additionally, many people have the erroneous notion that each successive property alteration on the node requires another call to getElementById... this is not the case, and requires additional DOM traversal (which can become slow).</p> <pre><code>document.getElementById('container-main').style.display = 'none'; document.getElementById('container-main').style.color = 'black'; // getting by id a second time can be slow </code></pre> <p>-</p> <pre><code>var myNode = document.getElementById('container-main'); if ( myNode &amp;&amp; myNode.style ) { myNode.style.display = 'none'; myNode.style.color = 'black'; // but using a reference a second time costs nothing. } </code></pre>
<pre><code>document.getElementById('container-main').style.display = 'none'; </code></pre> <p><code>display='none'</code> hides the element so it doesn't take up any space, whereas <code>display='hidden'</code> still takes up space but is just not viewable.</p>
