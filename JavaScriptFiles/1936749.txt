<p>If I understand correctly, your question is simply: how do javascript functions receive multiple arguments? </p> <p>This is easy, just separate them by a comma in your function declaration, and pass multiple values, again separated by comma in the function call:</p> <pre><code>function myFunc(one, two) { alert(one); alert(two); } myFunc(1,2); </code></pre> <p>If you don't know in advance how many arguments to pass/receive, just declare the function without arguments, and use the built-in arguments array inside the function definition:</p> <pre><code>function myFunc(){ for (var i=0, numArgs = arguments.length; i&lt;numArgs; i++){ alert(arguments[i]); } } </code></pre> <p>The approach above is nice if you need to pass a list of values that are all equal, but when you have to deal with multiple arguments and some of them are optional, a better approach is to pass an object literal, and add the 'arguments' as properties:</p> <pre><code>function myFunc(obj){ if (typeof(obj.arg1)!="undefined") { .... } if (typeof(obj.arg2)!="undefined") { .... } ...more handling... } myFunc({ arg1: "one" , arg2: "two" , ...more properties... }); </code></pre>
