how this function work and how i can pass parameter that is param_url
<p>below is function i want to use </p> <pre><code>(function () { var url = param_url; })(); // what are these ending curly brackets for ? </code></pre>
<p>The ending parentheses (<code>()</code>) call the function. You can pass arguments to it by putting them within the parentheses.</p> <p>What you have there is a <em>function expression</em> which is then immediately called. The function expression is:</p> <pre><code>(function () { var url = param_url; }) </code></pre> <p>...and then the parens call it. it's the same as:</p> <pre><code>var v = function () { var url = param_url; }; v(); </code></pre> <p>...aside from the use of <code>v</code>, of course. So to pass an argument to it, just do this:</p> <pre><code>(function (argname) { var url = param_url; })(your_argument_here); </code></pre> <p><a href="http://stackoverflow.com/users/130652/kangax">kangax</a> has written up a <a href="http://yura.thinkweb2.com/named-function-expressions/" rel="nofollow noreferrer">useful article</a> on function expressions, including browser bugs related to naming the function in the expression (amongst other things), which you <em>should</em> be able to but sadly, can't currently.</p>
