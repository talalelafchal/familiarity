How to access radioButton individual items?
<p>I've got 2 radio button groups showed differently depending on the value selected on another radio button. The thing is that I would like to set the default value of the group to the first element in the radiobutton group, but when I change from the other radio button it preserves the value previously selected, when I want it to be reset to the default value. </p> <p>I've been searching in the javascript repository but apparently I cannot access like it says with <code>radio[1].checked = true</code>... It says that <code>radio[1]</code> is undefined, but radio is an HTML Input element object. I'm accessing it from a different file, but <code>var radio = document.getElementById("radio")</code> works, but I can't access the individual radiobuttons.</p> <p>Thank you in advance</p>
<p>You can access them if you give each radiobutton an individual ID. The "name" attribute defines the group.</p>
<p>I agree with Robusto that you should give your radio elements ID attributes, and reference them that way.</p> <p>However, you can also access the <code>form.elements[]</code> array, and access your radio button values that way:</p> <pre><code>for( i = 0; i &lt; document.formname.elements.length; i++ ) { if(document.formname.elements[i].type == "radio") { document.formname.elements[i].checked = true; } } </code></pre>
<p>You <strong>can</strong> access them the way you describe:</p> <pre><code>&lt;!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd"&gt; &lt;html&gt; &lt;head&gt;&lt;title&gt;&lt;/title&gt; &lt;/head&gt; &lt;body&gt; &lt;form action="" method="get"&gt; &lt;input type="radio" name="foo" value="One"&gt; &lt;input type="radio" name="foo" value="Two"&gt; &lt;input type="button" value="Show info" onclick="showInfo()"&gt; &lt;/form&gt; &lt;script type="text/javascript"&gt;&lt;!-- function showInfo(){ var info = ""; for(var i=0, len=document.forms[0].foo.length; i&lt;len; i++){ info += "Radio button " + document.forms[0].foo[i].name + " is " + (document.forms[0].foo[i].checked ? "checked" : "not checked") + " and has a value of '" + document.forms[0].foo[i].value + "'\n"; } alert(info); } //--&gt;&lt;/script&gt; &lt;/body&gt; &lt;/html&gt; </code></pre>
<p>If you want to refer to a specific radio button in the group, using the <code>id</code> attribute as per Robusto's answer is the way to go about it. However, if you want to do something with each of the radio buttons in the group, you could use <a href="https://developer.mozilla.org/en/DOM/document.getElementsByName" rel="nofollow noreferrer"><code>getElementsByName</code></a>, which returns a collection of all elements found with the given name:</p> <pre><code>var radio_buttons = document.getElementsByName("radio"); for (var i = 0; i &lt; radio_buttons.length; i++) { // do something with radio_buttons[i] } </code></pre>
