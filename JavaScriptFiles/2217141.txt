How can I find out the position of an item contained by an array?
<p>I know the name of the item and would like to know its position in the array, how would I do that?</p>
<pre><code>["a","b","c","d","e"].indexOf("c") 2 </code></pre>
<p>Use the <code>indexOf</code> method of Array. If it's not there add it (code example <a href="http://soledadpenades.com/2007/05/17/arrayindexof-in-internet-explorer/" rel="nofollow noreferrer">here</a> or other places).</p> <p>If you want to find a value by its name property, do this:</p> <pre><code>for (var i = 0; i &lt; this.length; i++) { if(this[i].name == name) { return i; } } return -1; </code></pre>
<p>You could loop through the array:</p> <pre><code>var array = ['item1', 'item2', 'item3']; function findIndex(array, item) { for (var i = 0; i &lt; array.length; i++) { if (array[i] === item) { return i; } } return -1; } alert(findIndex(array, 'item2')); </code></pre> <p>Or using a comparer function for more complex types:</p> <pre><code>var array = ['item1', 'item2', 'item3']; function findIndex(array, comparer) { for (var i = 0; i &lt; array.length; i++) { if (comparer(array[i])) { return i; } } return -1; } var index = findIndex(array, function(item) { return item === 'item2' ; }); </code></pre>
