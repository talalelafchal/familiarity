<p>Really difficult to understand what is being asked here. If you input the following into your address bar, you will output HTML onto the document:</p> <pre><code>javascript:document.write("&lt;b&gt;HelloWorld&lt;/b&gt;"); </code></pre> <p>This type of stuff is only really helpful when programming bookmarklets though. If you decide to do something like this, be sure to escape the inner quotes. For example, below I have unescaped double-quotes around my style rules.</p> <pre><code>javascript:document.write("&lt;b style="text-decoration:underline"&gt;foo&lt;/b&gt;"); </code></pre> <p>The appropriate way to write this would be:</p> <pre><code>javascript:document.write("&lt;b style=\"text-decoration:underline\"&gt;foo&lt;/b&gt;"); </code></pre> <p>Or</p> <pre><code>javascript:document.write("&lt;b style='text-decoration:underline'&gt;foo&lt;/b&gt;"); </code></pre>
<p>See the <a href="http://en.wikipedia.org/wiki/Data_URI_scheme" rel="nofollow noreferrer">data: URI scheme</a>.</p>
