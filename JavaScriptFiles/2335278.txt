Change field field value on click
<p>I want to supply the user with 3 images, each image will have an associated value:</p> <pre><code>1. left,middle,right 2. middle,left,right 3. left,right,middle </code></pre> <p>I then want to write that value to the text field in a form, so it gets saved when the user clicks submit.</p> <p>I already have a working form where the user can type in values, <code>1,2 or 3</code> but want the text field to be hidden and the user to select an image instead and that update the text field.</p>
<p>jQuery makes this a trivial task. You can bind the click event to the images.</p> <p>excerpt:</p> <pre><code> &lt;img src="/images/lmr.png" class="direction" name="left,middle,right" /&gt; &lt;img src="/images/mlr.png" class="direction" name="middle,left,right" /&gt; &lt;img src="/images/lrm.png" class="direction" name="left,right,middle" /&gt; &lt;input type="text" value="Click an image" /&gt; &lt;script&gt; $('.direction').click(function () { var text = $(this).attr('name'); $("input").val(text); }); &lt;/script&gt; </code></pre> <p>Tested locally, and the input type field gets its value set. Now you can make it hidden and it will be posted. <code>(&lt;input type="hidden" etc.)</code></p>
<p>if you're using jQuery:</p> <pre><code>&lt;img class="selector123" src="..." rel="1"/&gt; </code></pre> <p>in jQuery</p> <pre><code>$(".selector123").click(function(){ $("#value123input").val($(this).attr('rel')); }); </code></pre> <p>That's one example:)</p>
<p>Set <code>@id</code> attribute to your hidden inputs, and then at your images you can use javascript code, for example</p> <pre><code>&lt;img src="1.jpg" onclick="document.getElementById('inputID').value='your value'"&gt; &lt;input type="hidden" id="inputID" name="somename"&gt; </code></pre>
<p>Attach an onclick handler to the images, which sets the value attribute of the text field to whatever you want.</p> <p>Here is something for it in jQuery:</p> <pre><code>&lt;img class="clickable" src="something.jpg" alt="left,middle,right"/&gt; &lt;img class="clickable" src="something.jpg" alt="middle,left,right"/&gt; &lt;img class="clickable" src="something.jpg" alt="left,right,middle"/&gt; &lt;input type="hidden" id="clicked-image" name="something"/&gt; &lt;script&gt; $(document).ready(function() { $('.clickable').bind('click', function () { $('#clicked-image').val($(this).attr('alt')); }); }); &lt;/script&gt; </code></pre>
