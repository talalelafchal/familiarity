How do I know if a mouse pointer is over a particular element on the page?
<p>With the help of JavaScript, of course.</p>
<p>The onMouseover, onMouseout events fire.</p>
<p>The easiest way is to handle the <code>onmouseover</code> and <code>onmouseout</code> event for the elements:</p> <pre><code>element.onmouseover = function (e) { e = e || window.event; alert("You moused over "+element.outerHTML); } element.onmouseout = function (e) { e = e || window.event; alert("You moused out of "+element.outerHTML); } </code></pre> <p>You can also use <a href="https://developer.mozilla.org/en/DOM/document.elementFromPoint" rel="nofollow noreferrer"><code>document.elementFromPoint(x, y)</code></a> if the element is the topmost element at those co-ordinates:</p> <pre><code>var mouseX = 0, mouseY = 0; document.documentElement.onmousemove = function (e) { e = e || window.event; mouseX = e.clientX; mouseY = e.clientY; } function getElementUnderMouse() { return document.elementFromPoint(mouseX, mouseY); } </code></pre> <p>Obviously if you have certain elements that prevent propagation/bubbling of the <code>onmousemove</code> event the mouse position could be incorrect whilst hovering over those elements.</p>
<p>In the example below, if you mouse over a div, it will become blue.</p> <p>If you place the onmouseover to the full BODY, then you can then check on a condition(ID, className, tagName,...) if you want to do something on the element you are overing.</p> <pre><code>&lt;html&gt; &lt;head&gt; &lt;title&gt;so&lt;/title&gt; &lt;/head&gt; &lt;body&gt; &lt;div&gt;div 1&lt;/div&gt; &lt;div&gt;div 2&lt;/div&gt; &lt;div&gt;div 3&lt;/div&gt; &lt;script&gt; document.body.onmouseover = function(ev){ var elm = ev.target || ev.srcElement; if(elm.tagName === 'DIV'){ elm.style.backgroundColor = "#DEF"; } }; &lt;/script&gt; &lt;/body&gt; &lt;/html&gt; </code></pre>
