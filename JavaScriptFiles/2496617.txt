How do I verifying if site has javascript enabled and if user's browser is IE 8
<p>How do I verify if a site has Javascript enabled and if user's browser is IE 8?</p>
<p>Use conditional comments and some javascript code (it will work only if it's enabled, right? :))</p> <pre><code>&lt;!--[if IE 8]&gt; &lt;script type="text/javascript"&gt; /*do something in JS - it's enabled!*/ &lt;/script&gt; &lt;[endif]--&gt; </code></pre> <p>Or you may want to look at <a href="http://www.javascriptkit.com/javatutors/conditionalcompile.shtml" rel="nofollow noreferrer">conditional compilation in JS</a></p>
<p>If for whatever reason you need to know that you are dealing with IE8,</p> <p>you have to watch for the IE9 preview, which smells like IE8, but has some new objects:</p> <pre><code>&lt;!--[if IE 8]&gt; &lt;script type="text/javascript"&gt; if(document.addEventListener) alert('IE9'); if(document.documentMode) alert('IE8'+ running as '+document.documentMode); &lt;/script&gt; &lt;[endif]--&gt; </code></pre>
