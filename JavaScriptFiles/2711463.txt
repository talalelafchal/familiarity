How to use setAttribute in javascript
<p>Is this the correct format?</p> <pre><code>var title = new_element.setAttribute('jcarouselindex', "'items+1'"); alert(title); </code></pre>
<p>No need to put the second parameter in quotes.</p> <pre><code>var title = new_element.setAttribute('jcarouselindex', items+1); </code></pre> <p>If you want to set a custom attribute then you can use <strong><a href="http://ejohn.org/blog/html-5-data-attributes/" rel="nofollow noreferrer">HTML5 data-attributes</a></strong>, something like</p> <pre><code>var title = new_element.setAttribute('data-jcarouselindex', items+1); </code></pre>
<p>It is syntactically correct JS/DOM, but there is no 'jcarouselindex' attribute in HTML and using <code>setAttribute</code> (as opposed to setting properties that map to attributes) is generally bad practice as it is buggy in MSIE (although not in such a way that will cause a problem in this case).</p> <p>You might not be intending to have <code>&lt;foo jcarouselindex="&amp;#39;items+1&amp;#39;"&gt;</code> as your end result though.</p>
<p>Using jQuery?</p> <p>var title = $('#new_element').attr('jcarouselindex', "'items+1'"); alert(title);</p>
<p>You have too many quotes:</p> <pre><code>var index = items + 1; new_element.setAttribute('jcarouselindex', index); </code></pre> <p>Remark: there's no <code>jcarouselindex</code> attribute defined in HTML elements so this is not very clean code.</p>
