Whats the quickest way to get rid of all the properties of a an object in javascript?

<pre><code>for (p in obj) { if (obj.hasOwnProperty(p)) { delete obj[p]; } } </code></pre> <p>If you only have one reference to the object, then replacing it with a new one would be faster.</p> <pre><code>obj = {}; </code></pre>
