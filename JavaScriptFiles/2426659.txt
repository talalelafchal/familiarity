<p>You may use <code>setTimeout</code> to delay a function's execution:</p> <pre><code>window.setTimeout(doSomething, 1000); // 1000ms == 1 second </code></pre> <p>To hide an element, you may set its <code>display</code> property to <code>none</code>:</p> <pre><code>var element = document.getElementById('foo'); function doSomething() { element.style.display = 'none'; } </code></pre>
<p>Try this:</p> <pre><code>var timePeriodInMs = 4000; setTimeout(function() { document.getElementById("myDiv").style.display = "none"; }, timePeriodInMs); </code></pre>
<pre><code>setTimeout(function(){ document.getElementById('messageID').style.display = 'none'; }, 5000); //5secs </code></pre>
<ol> <li>add jquery to your html document</li> <li>add script tag to head of your document containing:</li> </ol> <pre><code>&lt;script type="text/javascript"&gt; $(document).ready(function(){ window.setTimeout(function(){ $('#id-of-div').hide(); }, 10000 /* delay time in milliseconds */ }); &lt;/script&gt;</code></pre>
<p>You are looking at the <code>setTimeout( expression, timeout );</code> where you need to give it an <code>expression</code> to run after the <code>timeout</code> in milliseconds that you allocate to it. then you would do <code>element.style.display="none"</code></p> <p>ex:</p> <pre><code>setTimeout( function(){element.style.display="none"}, 4000 ); </code></pre>
<p>If you want the element to disappear after a certain time, <strong>regardless of page reloads</strong> (that's the way I read your question), you would have to work with cookies.</p> <ul> <li><p>You would have to set a cookie with the starting point</p></li> <li><p>You would have to execute a JavaScript function on page load that compares the time set in the cookie with the current time, and shows/hides the element accordingly.</p></li> <li><p>For a perfect solution, you would additionally implement a timer that compares the current date to the date in the cookie every, say, half-second, and hides the element when the time has been reached.</p></li> </ul> <p>Using a framework like JQuery is certainly a good idea for this.</p>
