Using Javascript, How to get number of items of radio button with the same id in html
<p>Using Javascript, How to get number of items of radio button with the same id in html;</p> <p>I am aware if I can wrap a form on top of the radio button I can get the length like </p> <pre><code>document.formname.radiobtnobj.length; </code></pre> <p>Wondering if is there a way I can get the number of items if I do not have a parent form to that control.</p>
<p>All IDs must be unique. Duplicating IDs is BAD HTML.</p> <p>That being said, one way to do what you intend is to give all of the radio buttons a class name, and get all of the items that share that class name.</p> <p>In forms we usually group radio buttons by giving them the same NAME, not ID. In this case, the elements are an array.</p> <pre><code>var myRadioButtons = document.forms[0].yourRadioButtonName // this gives you back your array. alert(myRadioButtons[0].value) alert(myRadioButtons[0].checked) alert(myRadioButtons.length) </code></pre>
<p>As Diodeus mentioned, use the <em>name</em> attribute - do not duplicate the <em>id</em> attribute. Then you can use <code>document.getElementsByName()</code>. No <code>&lt;form&gt;</code> necessary.</p> <pre><code>var radioGrp = document.getElementsByName(radioButtonName)[0]; </code></pre> <p>Also, don't use <code>eval()</code> unless you have a good reason. evaling a dom object's ID isn't one.</p>
