Google liked drop down menu
<p><img src="http://sites.google.com/site/yanchengcheok/Home/google-drop-down-menu.png" alt="alt text"></p> <p>Hello, whenever we go to Google Page and click on the "more", a menu will be dropped down. I would like to have the following effect on my web site too. May I know which JavaScript library can help me to achieve the similar effect?</p>
<p>You just add listener to click event for a "more" element:</p> <pre><code>elementRef.addEventListener("click", function() { // listener code here }, false); </code></pre> <p>(you can do this in any JS library if you want to). This listener should now just display (change CSS property <code>display</code> from <code>none</code> to <code>block</code>) another element (ie. <code>&lt;div id="more" /&gt;</code>). Also you add another listener for click event, but this time for the <code>body</code> element (to hide menu).</p> <p>Final code could looks like following:</p> <p>JavaScript:</p> <pre><code>document.getElementById("toggle-more").addEventListener("click", function(e) { document.getElementById("more").style.display = "block"; e.stopPropagation(); }, false); document.body.addEventListener("click", function() { document.getElementById("more").style.display = "none"; }, false); </code></pre> <p>HTML:</p> <pre><code>&lt;span id="toggle-more"&gt;More...&lt;/span&gt; &lt;div id="more"&gt; &lt;ul&gt; ... &lt;/ul&gt; &lt;/div&gt; </code></pre> <p>CSS:</p> <pre><code>#more { display: none; position: absolute; top: 15px; left: 150px; } </code></pre>
