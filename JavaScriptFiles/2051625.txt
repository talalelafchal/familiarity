<pre><code>var b = 'I,am,a,JavaScript,hacker' var temp = new Array(); temp = b.split(','); </code></pre> <p>Now the string has been split into 5 strings that are placed in the array temp. The commas themselves are gone.</p> <pre><code>temp[0] = 'I'; temp[1] = 'am'; temp[2] = 'a'; temp[3] = 'JavaScript'; temp[4] = 'hacker.'; </code></pre> <p>Taken from <a href="http://www.quirksmode.org/js/strings.html" rel="nofollow noreferrer">QuirksMode</a>. </p> <p><strong>join</strong> is the "opposite" of split - use it to append the elements of an array together into a variable. </p> <pre><code>var j = temp.join(','); // sets j to 'I,am,a,JavaScript,hacker' </code></pre>
