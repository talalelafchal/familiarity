javascript: lightweight lightbox script
<p>i'm looking for a lightweight javascript for displaying images the way lightbox does. it should be small + simple, no jquery/mootools plugin, no animations.</p> <p>i just should be able to display an image when clicking on its thumbnail.</p> <p>thanks</p>
<p>You can use for example <a href="http://www.dolem.com/lytebox/" rel="nofollow noreferrer">Lytebox</a>. This script doesn't have any dependencies on external libraries and integrating it with your page is as easy as using Lightbox</p>
<p>If you're looking for something <em>really</em> light-weight, you might take a look at <a href="http://mercurial.intuxication.org/hg/js-hacks/raw-file/tip/minibox.js" rel="nofollow noreferrer">this script</a>.</p> <p>To convert all links whose URLs end in <code>.jpg</code>, <code>.png</code> or <code>.gif</code>, add the following lines to the bottom of your page:</p> <pre><code>&lt;script src="minibox.js" type="text/javascript"&gt;&lt;/script&gt; &lt;script type="text/javascript"&gt; minibox.borderSize = 5; minibox.borderType = 'solid red'; minibox.img.title = 'click to close image window'; minibox(); &lt;/script&gt; </code></pre> <p>There is no documentation, but the code is reasonably clear.</p>
