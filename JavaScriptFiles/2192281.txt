Javascript pass variable by reference
<p>How can I pass variables by reference to setInterval's call back function?<br> I don't want to define a global variable just for the counter. Is it possible?</p> <pre><code> var intervalID; function Test(){ var value = 50; intervalID = setInterval(function(){Inc(value);}, 1000); } function Inc(value){ if (value &gt; 100) clearInterval(intervalID); value = value + 10; } Test(); </code></pre>
<p>If you create a closure for it, you won't have to pass the value at all, it'll just be available in the internal scope, but not outside the <code>Test</code> function:</p> <pre><code>function Test() { var value = 50; var intervalID = setInterval(function() { // we can still access 'value' and 'intervalID' here, altho they're not global if(value &gt; 100) clearInterval(intervalID); value += 10; }, 1000); } Test(); </code></pre>
