Are you sure you want to navigate?
<p>What is the JavaScript code/event(s) that is used by sites like stackoverflow and Gmail to test for the user exiting the page once they have begun editing and try to navigate away?</p> <pre><code>"Are you sure you want to navigate away from this page?" </code></pre>
<p><a href="https://developer.mozilla.org/en/DOM/window.onbeforeunload" rel="nofollow noreferrer">onbeforeunload</a> event. Mozilla provides useful example code. you just want to have a function that:</p> <ol> <li>Returns a string</li> <li>Sets e.returnValue to that string, where e is the argument or window.event.</li> </ol> <p>The string will be used as your custom message.</p>
<p>The event used is called <code>onbeforeunload</code>.</p> <pre><code>&lt;html&gt; &lt;head&gt; &lt;script type="text/javascript" src="jquery.js"&gt;&lt;/script&gt; &lt;/head&gt; &lt;body&gt; &lt;input id="foo"&gt;&lt;/input&gt; &lt;script type="text/javascript"&gt; function unloadMessage() { return "Are you sure you want to leave?"; } function setConfirmUnload(enabled) { window.onbeforeunload = enabled ? unloadMessage : null; } $(document).ready(function() { $("#foo").keypress(function() { setConfirmUnload(true); }); }); &lt;/script&gt; &lt;/body&gt; &lt;/html&gt; </code></pre>
