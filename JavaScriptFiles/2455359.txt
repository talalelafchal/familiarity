Javascript - Get query string from a string?
<p>I'm trying to get the query string from a string (not the current URL).</p> <p>For example, I have a URL 'www.google.com/?query=string', I'd like to be able to run a function on it and get '?query=string' back from it. How would I go about doing this?</p> <p>Thanks</p>
<p>Well, you can use a quick regexp that gets you the part you need:</p> <pre><code>myString.match(/(\?.*)/)[1] </code></pre> <p>Example: </p> <pre><code>'www.google.com/?query=string'.match(/(\?.*)/)[1] // evaluates to '?query=string' </code></pre>
<p>There's <a href="http://plugins.jquery.com/project/query-object" rel="nofollow noreferrer">a jQuery plugin</a> for that.</p>
<p>If you're using jQuery, use this plugin: <a href="http://projects.allmarkedup.com/jquery_url_parser/" rel="nofollow noreferrer">http://projects.allmarkedup.com/jquery_url_parser/</a> This one lets you operate on the document's url, or any URL string</p> <p>Then you can do:</p> <pre><code>$.url.setUrl("www.google.com/?query=string").attr("query") // returns 'query=string' </code></pre> <p>Or also get a specific parameter:</p> <pre><code>$.url.setUrl("www.google.com/?query=string").param("query") // returns 'string' </code></pre> <p>But if you really just need the whole query string, a quick regex like Alsciende suggested is the way to go.</p>
<p>Window.location.search will evaluate to this.</p> <p><a href="http://www.w3schools.com/jsref/prop_loc_search.asp" rel="nofollow noreferrer">http://www.w3schools.com/jsref/prop_loc_search.asp</a></p>
