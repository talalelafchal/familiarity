Strings split in javascript?
<p>i am using my_colors.split(" ") method, but i want to split or divide string in fixed number of words e.g each split occurs after 10 words or so ...how to do this in javascript?</p>
<p>You can split(" ") then join(" ") the resulting array 10 elements at a time.</p>
<p>You can try something like</p> <pre><code>console.log("word1 word2 word3 word4 word5 word6" .replace(/((?:[^ ]+\s+){2})/g, '$1{special sequence}') .split(/\s*{special sequence}\s*/)); //prints ["word1 word2", "word3 word4", "word5 word6"] </code></pre> <p>But you better do either <code>split(" ")</code> and then <code>join(" ")</code> or write a simple tokenizer yourself that will split this string in any way you like.</p>
<p>You might use a regex like <code>/\S+/g</code> to split the string in case the words are separated by multiple spaces or any other whitespace.</p> <p>I am not sure my example below is the most elegant way to go about it, but it works.</p> <pre><code>&lt;html&gt; &lt;head&gt; &lt;script type="text/javascript"&gt; var str = "one two three four five six seven eight nine ten " + "eleven twelve thirteen fourteen fifteen sixteen " + "seventeen eighteen nineteen twenty twenty-one"; var words = str.match(/\S+/g); var arr = []; var temp = []; for(var i=0;i&lt;words.length;i++) { temp.push(words[i]); if (i % 10 == 9) { arr.push(temp.join(" ")); temp = []; } } if (temp.length) { arr.push(temp.join(" ")); } // Now you have an array of strings with 10 words (max) in them alert(" - "+ arr.join("\n - ")); &lt;/script&gt; &lt;/head&gt; &lt;body&gt; &lt;/body&gt; &lt;/html&gt; </code></pre>
<p>Try this - this regex captures groups of ten words (or less, for the last words):</p> <pre><code>var groups = s.match(/(\S+\s*){1,10}/g); </code></pre>
