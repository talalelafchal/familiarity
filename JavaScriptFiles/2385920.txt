Return statements at the end of a JavaScript function
<p>I have seen in many times in JavaScript code people add a <code>return true</code> at the end although not necessary. Does anyone know why? </p> <pre><code>var _globalString; function doSomething() { _globalString= _globalString +' do something'; //some codes to do something more //finally adding a return true return true; } </code></pre>
<p>The thing that may have gotten some people into the habit was event handlers for forms, if you have, say:</p> <pre><code>&lt;form onsubmit="return myfunction();"&gt; </code></pre> <p>and <code>myfunction()</code> returns true, the form submits, else if it returns false it doesn't. People doing it in general could've got the habit from this. Some languages require return values from functions, Javascript doesn't; and having <code>return true</code> at the end of most functions serves no purpose.</p>
<p>Actually if you are calling the function in onsumbit event </p> <p>Example </p> <pre> &lt;input type=sumit value=click Onsumbit='return function_name();"&gt; </pre> <p>While you are calling like , if the function return true only, form will be submit </p> <p>If it return false , it wont submit the form </p>
<p>In addition to Erik's answer I would like to add</p> <p><strong>return true</strong> / <strong>return false</strong> are also used when you want boolean value as a return. And based on that return you execute some other function.</p>
<p>It's hard to say why some programmers do certain things.</p> <p>Maybe it's intended to indicate success/failure, but they haven't added any failing branches yet?</p>
<p>A "return" inside a function automatically stops further execution of that function so for example:</p> <pre><code>function myFunc(){ if(foo == 'bar'){ /* do something */ }else{ /* do something */ } } </code></pre> <p>is the same as:</p> <pre><code>function myFunc(){ if(foo == 'bar'){ /* do something */ return true; } /* if foo != 'bar' then whatever follows is executed... */ } </code></pre>
<p>Also you do not need to use return true or false in this case below</p> <pre><code>var newPage = "http://www.google.com"; function redirectURL(){ window.location.href= newPage; return true; //no return required } </code></pre>
