<p>Is your input XHTML? If so, to validate it as XML you would need a CDATA block to contain your javascript code.</p>
<p>If you are "using" XHTML then you need to wrap the code in CDATA:</p> <pre><code>&lt;script type="text/javascript"&gt; //&lt;![CDATA[ // rest of your javascript goes here //]]&gt; &lt;/script&gt; </code></pre>
