Splitting in string in JavaScript
<p>How to split a string in JavaScript with the "<code>,</code>" as seperator?</p>
<p>var expression = "h,e,l,l,o";</p> <p>var tokens = expression.split("\,");</p> <p>alert(tokens[0]);// will return h</p>
<p>Use <a href="https://developer.mozilla.org/En/Core_JavaScript_1.5_Reference/Objects/String/Split" rel="nofollow noreferrer"><code>split</code></a> to split your string:</p> <pre><code>"foo,bar,baz".split(",") // returns ["foo","bar","baz"] </code></pre>
