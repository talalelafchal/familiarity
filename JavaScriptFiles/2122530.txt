Is there any way to make a date object in 12 hour mode in javascript
<p>Hii,</p> <p>I have to make date time in 12hr mode i.e. 23 January 2010 1:30 PM in java script</p>
<p>Well, you can always use a wrapper on getHours() and convert to the 12 hr format. </p> <pre><code>Date.prototype.getHoursIn12HrMode = function() { var h = this.getHours(); if(h == 0) return 12; return (h &gt; 12) ? h - 12 : h; } </code></pre> <p><a href="http://www.datejs.com/" rel="nofollow noreferrer">Datejs</a> is a nice library to use if more customizations are needed.</p>
<p>The built-in Date object will allow you to format the date in the format defined by the user's system - this may or may not include AM/PM (it may not be in English either). You can format it yourself, of course, using something like <a href="http://www.webdevelopersnotes.com/tips/html/formatting_time_using_javascript.php3" rel="nofollow noreferrer">this</a>. For complete control, use a <a href="http://hacks.bluesmoon.info/strftime/" rel="nofollow noreferrer"><code>strftime()</code></a> library.</p>
<p>Steven Levithan has published a JS script which implements all sorts of date-time masks. It includes a 12-hour clock element. <a href="http://blog.stevenlevithan.com/archives/date-time-format" rel="nofollow noreferrer" title="JavaScript Date Format">Find out more</a>. </p>
