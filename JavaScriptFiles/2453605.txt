<p>You can try this :</p> <pre><code>var i=1; // starting index while(document.getElementById('mystickytooltip'+i)) { stickytooltip.init("*[data-tooltip]", "mystickytooltip"+(i++)); } </code></pre> <p>In english : as long as you can find an element with id <code>'mysticktooltip'+i</code> (+ is string concatenation in js), call <code>stickytooltip.init</code> with this id as second parameter (and increment <code>i</code>).</p>
<p>In your previous (now deleted) question, it looked like you were using jQuery. If that's the case, you should just be able to use a class selector:</p> <pre><code>var myToolTips = $("div.mystickytooltip"); </code></pre> <p>Or you could use the <a href="http://api.jquery.com/attribute-starts-with-selector/" rel="nofollow noreferrer"><code>Attribute Starts With</code></a> selector:</p> <pre><code>var myToolTips = $("div[id^='mystickytooltip']"); </code></pre> <p>The only possible drawback I could forsee with the attr-starts-with selector is if you have any other div elements whose id begins with "mystickytooltip" (ie, "mystickytooltipcontainer"). In that case you could combine the class and attr-starts-with selectors:</p> <pre><code>var myToolTips = $("div.mystickytooltip[id^='mystickytooltip']"); </code></pre>
