Need help in OnChange event Javascript
<p>I have five text fields which are mark1, mark2, mark3, mark4 and total. I have a problem to make the total automatically display in the text field. I have no problem with the calculation part. But I can't make the total to be display automatically in the text field. How can I make the total to be automatically display in the textfield once the user have finish entered all the four marks? I know it have to do with the OnChange event using Javascript. But I did not get the right codes. Does anyone can help me with this? Really need your advise.</p> <p>Thank you.</p>
<p>Daisy, something like this should work. You can <a href="http://jsbin.com/uyifa" rel="nofollow noreferrer">view a live demo</a> of this code on JSBin.</p> <pre><code>&lt;input type="text" name="mark1" id="mark1" value="" /&gt; &lt;input type="text" name="mark2" id="mark2" value="" /&gt; &lt;input type="text" name="mark3" id="mark3" value="" /&gt; &lt;input type="text" name="mark4" id="mark4" value="" /&gt; &lt;input type="text" name="total" id="total" value="" /&gt; &lt;!-- Script block must come AFTER the input elements --&gt; &lt;script type="text/javascript"&gt; var m1 = document.getElementById('mark1'), m2 = document.getElementById('mark2'), m3 = document.getElementById('mark3'), m4 = document.getElementById('mark4'), total = document.getElementById('total'); function calculate_total(){ // Use your real calculation here total.value = Number(m1.value) + Number(m2.value) + Number(m3.value) + Number(m4.value); } if( window.addEventListener ) { m1.addEventListener('change', calculate_total, false); m2.addEventListener('change', calculate_total, false); m3.addEventListener('change', calculate_total, false); m4.addEventListener('change', calculate_total, false); } else if(window.attachEvent){ m1.attachEvent('onchange', calculate_total); m2.attachEvent('onchange', calculate_total); m3.attachEvent('onchange', calculate_total); m4.attachEvent('onchange', calculate_total); } &lt;/script&gt; </code></pre> <p><strong>UPDATE:</strong> Since you want a letter grade entered (A, B, C, F, etc) I would recommend a <code>select</code> control instead of an <code>input[type='text']</code>. One of the grade fields would look like this:</p> <pre><code>&lt;select name="mark1" id="mark1"&gt; &lt;option value=""&gt;Select Grade&lt;/option&gt; &lt;option value="100"&gt;A&lt;/option&gt; &lt;option value="90"&gt;B&lt;/option&gt; &lt;option value="80"&gt;C&lt;/option&gt; &lt;option value="70"&gt;D&lt;/option&gt; &lt;option value="50"&gt;F&lt;/option&gt; &lt;/select&gt; </code></pre> <p>You put the number value in the <code>value=</code> portion, but you can display the friendlier A, B, C, etc.</p> <p>And anywhere there was <code>.value</code> replace with <code>.selectedValue</code> (Except <code>total.value</code> of course).</p>

Note the <code>grades</code> array of values and the small change to Doug's function:</p> <pre><code>// Add grades and corresponding value below var grades = {A:100,B:70}; function calculate_total(){ // Use your real calculation here total.value = grades[m1.value] + grades[m2.value] + grades[m3.value] + grades[m4.value]; } </code></pre>
