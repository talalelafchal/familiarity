how do I automatically execute javascript?
<p>how do I automatically execute javascript?</p> <p>I know of <code>&lt;body onLoad=""&gt;</code>, but I just thought maybe there is another way to do it?</p> <p>html:</p> <pre><code>&lt;html&gt;&lt;head&gt;&lt;/head&gt;&lt;body&gt;&lt;div id="test"&gt;&lt;/div&gt;&lt;/body&gt;&lt;/html&gt; </code></pre> <p>javascript:</p> <pre><code>&lt;script&gt;(function(){var text = document.getElementById('test').innerHTML;var newtext = text.replace('', '');return newtext;})();&lt;/script&gt; </code></pre> <p>I wanna get the text within "test", replace certain parts, and then output it to the browser.</p> <p>Any ideas on how to do it? I'd appreciate any help. Thanks.</p>
<p>If you don't want to use <code>&lt;body onload&gt;</code> which is good choice in terms of <strong>obtrusive</strong> javascript, you can separate that and put you code like this:</p> <pre><code>window.onload = function(){ // your code here }; </code></pre> <p><strong>Alternative:</strong></p> <p>Place your javascript code at the bottom of the page.</p>
<p>Place the script at the bottom of the page, outside the closing body tag..</p>
<p>A quick way, if you just want to debug, would be move what you want to execute outside of a function.</p> <pre><code>&lt;script type="text/javascript"&gt; var text = document.getElementById('test').innerHTML; var newtext = text.replace('', ''); alert(newtext); &lt;/script&gt; </code></pre> <p>NB. I'm not sure what you hope to achieve with <code>text.replace('', '')</code> ?</p>
<p>If you don't want to use jQuery, use native window.onload method:</p> <pre><code>&lt;script type="text/javascript"&gt; function ReplaceText() { document.getElementById('test').innerHTML = document.getElementById('test').innerHTML.replace(/abc/g, "def"); } window.onload = ReplaceText; &lt;/script&gt; </code></pre> <p>Used on the code:</p> <pre><code>&lt;div id="test"&gt;abc abc&lt;/div&gt; </code></pre> <p>Will give this output:</p> <pre><code>def def </code></pre>
<p>It's REALLY easy! If you have a script in your "head" block, with no function id, it will run automatically as soon as the web page loads. For example:</p> <p><code>&lt;head&gt;</code> <code>&lt;meta charset="UTF-8"&gt;</code> <code>&lt;title&gt;Redirection to www.mywebsite.org&lt;/title&gt;</code></p> <pre><code>&lt;!-- This script initiates an automatic web page redirection, as the page is loaded --&gt; &lt;script type="text/javascript"&gt; window.location = "http://www.mywebsite.com/" &lt;/script&gt; </code></pre> <p><code>&lt;/head&gt;</code></p>
