setInterval is not working
<p>Suppose I have a function:</p> <pre><code>function chk(){ alert("Welcome"); } window.onload = chk(); setInterval("chk();", 5000); </code></pre> <p>but it is not working but when I refresh the page it works for me. How can I fix that?</p>
<p>This works just fine for me. Note the use of the function reference instead of calling the function and assigning the return value. SetInterval need not use a string -- which forces an eval of the argument. You can also use a function reference (or an anonymous function) as the argument.</p> <pre><code>function chk() { alert('checking'); } window.onload = chk; setInterval(chk,5000); </code></pre>
<p>If you want the alert to display once, after 5 seconds, use:</p> <pre><code>function chk(){ alert("Welcome"); } setTimeout("chk()", 5000); </code></pre> <p>If you want the alert to appear every 5 seconds (extremely annoying, but there is other legitimate for setInterval)</p> <pre><code>function chk(){ alert("Welcome"); } setInterval("chk()", 5000); </code></pre>
<pre><code>function chk(){ alert("Welcome"); } window.onload = chk(); setInterval("chk();", 5000); </code></pre> <p><code>window.onload</code> becomes <code>undefined</code> here -- <code>chk</code> doesn't return anything. And you need to rewrite your setInterval like this: <code>setInterval(chk, 5000);</code></p>
