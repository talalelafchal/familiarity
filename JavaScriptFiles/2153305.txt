Derive window.self from object
<p>is there a way to get window.self of an unknown iframe (same domain) with the only reference: an object within the iframe (a div). holding thumbs for this one</p>
<p><code>element.ownerDocument.defaultView</code> in Firefox.</p> <p><code>element.ownerDocument.parentWindow</code> in IE.</p> <p>But Webkit (Safari) apparently doesn't support either of those things, so to get your page to work in Safari, you have to use <a href="http://code.google.com/p/doctype/wiki/ArticleParentWindow" rel="nofollow noreferrer">this amazing hack</a>.</p>
<p>If the script is running within the iframe, then just do <code>window</code>. If the script is running outside of the iframe, then follow Jason's advice.</p> <p>If you need a reference take a look at the <a href="https://developer.mozilla.org/En/Gecko_DOM_Reference" rel="nofollow noreferrer">Gecko DOM reference</a> and <a href="http://msdn.microsoft.com/en-us/library/ms533050%28VS.85%29.aspx" rel="nofollow noreferrer">MS Msdn</a>.</p>
