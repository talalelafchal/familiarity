<p>To pass data when you submit a form, you have to include that data in a form input field (hidden, visible, doesn't matter).</p>
<p>You can add hidden fields in your HTML form like this</p> <pre><code>&lt;input type="hidden" id="Data_ID"&gt; &lt;input type="hidden" id="Data_Type"&gt; </code></pre> <p>and then set the values in your javascript function and then submit (?)</p> <pre><code>&lt;script type="text/javascript"&gt; function Service_Add(Data_ID,Data_Type) { document.getElelementByID("Data_ID").value=Data_ID; document.getElelementByID("Data_Type").value=Data_Type; document.miformulario.submit(); } &lt;/script&gt; </code></pre>
