how to add array element values with javascript?
<p>I am NOT talking about adding elements together, but their values to another separate variable.</p> <p>Like this:</p> <pre><code>var TOTAL = 0; for (i=0; i&lt;10; i++){ TOTAL += myArray[i] } </code></pre> <p>With this code, TOTAL doesn't add mathematically element values together, but it adds them next to eachother, so if <code>myArr[1] = 10</code> and <code>myArr[2] = 10</code> then <code>TOTAL</code> will be <code>1010</code> instead of <code>20</code>.</p> <p>How should I write what I want ?</p> <p>Thanks</p>
<p>Sounds like your array elements are Strings, try to convert them to Number when adding:</p> <pre><code>var total = 0; for (var i=0; i&lt;10; i++){ total += +myArray[i]; } </code></pre> <p>Note that I use the unary plus operator (<code>+myArray[i]</code>), this is one common way to make sure you are adding up numbers, not concatenating strings.</p>
<p>Make sure your array contains numbers and not string values. You can convert strings to numbers using parseInt(number, base)</p> <pre><code>var total = 0; for(i=0; i&lt;myArray.length; i++){ var number = parseInt(myArray[i], 10); total += number; } </code></pre>
<p>A quick way is to use the unary plus operator to make them numeric:</p> <pre><code>var TOTAL = 0; for (var i = 0; i &lt; 10; i++) { TOTAL += +myArray[i]; } </code></pre>
<p>Use <code>parseInt</code> or <code>parseFloat</code> (for floating point)</p> <pre><code>var total = 0; for (i=0; i&lt;10; i++) total+=parseInt(myArray[i]); </code></pre>
