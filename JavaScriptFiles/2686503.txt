<p>In the below code, I get the two dialog boxes, but the bgColor of the page is never changed. Is there any standard that font / color changes are not honored in JavaScript ?</p> <pre><code> &lt;html&gt; &lt;head&gt; &lt;/head&gt; &lt;body bgColor="GRAY"&gt; &lt;script type="text/javascript"&gt; document.write("This message is written by JavaScript"); alert('Am here'); alert('Am here again'); document.bgcolor="WHITE"; &lt;/script&gt; &lt;/body&gt; &lt;/html&gt; </code></pre>
<p><code>bgColor</code> is really, really old. Better use CSS values. And you need to address the <code>body</code> element, not the document.</p> <p>This should work: <code>document.body.style.backgroundColor = '#ffffff'</code></p>
<p>That's because it's <code>bgColor</code> and not <code>bgcolor</code> (note the case).</p> <p>Also, using <code>document.body.style.backgroundColor</code> might be a better idea. And I would use hexadecimal instead of named colors :)</p> <p>So, use <code>#ffffff</code> instead of <code>WHITE</code>.</p>
