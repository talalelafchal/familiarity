What does the i--&gt; opeator do in JavaScript?
<p>So I took a look at the code that controls the counter on the SO advertising page. Then I saw the line where this occured <code>i--&gt;</code>. What does this do?</p> <p>Here is the full code:</p> <pre><code>$(function(){ var visitors = 5373891; var updateVisitors = function() { visitors++; var vs = visitors.toString(), i = Math.floor(vs.length / 3), l = vs.length % 3; while (i--&gt;0) if (!(l==0&amp;&amp;i==0)) // &lt;-------- Here it is!!! vs = vs.slice(0,i*3+l) + ',' + vs.slice(i*3+l); $('#devCount').text(vs); setTimeout(updateVisitors, Math.random()*2000); }; setTimeout(updateVisitors, Math.random()*2000); }); </code></pre>
<p>Other answers have explained that it's two operators. I'll just add that in the example, it's unnecessary. If you're counting down from a positive integer to zero, you can miss out the greater-than-zero test and your code will be shorter and, I think, clearer:</p> <pre><code>var i = 10; while (i--) { // Do stuff; } </code></pre>
<p><code>i--&gt;0</code> is the same as <code>i-- &gt; 0</code>, so the comparison expression if the evaluated value of <code>i--</code> is greater than <code>0</code>.</p>
<p>it is not an operator.
