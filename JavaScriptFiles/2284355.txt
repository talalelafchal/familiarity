How to get the count of the DIVs with almost the same ID property?
<p>I have many DIVs on my page with the same ID</p> <p>eg:</p> <pre><code>&lt;div id="myDiv1"&gt; ... &lt;/div&gt; &lt;div id="myDiv2"&gt; ... &lt;/div&gt; &lt;div id="myDiv3"&gt; ... &lt;/div&gt; ... &lt;div id="myDiv20"&gt; ... &lt;/div&gt; ... </code></pre> <p>As You see, the ID property looks almost the same - the only diffrence is that there is a number in each ID.</p> <p>How to get the count of that DIVs? I thought I can do something like that:</p> <pre><code>var myDivs= document.getElementById('myDiv'); </code></pre> <p>but returns null</p>
<p>You can do this using <a href="http://jquery.com" rel="nofollow noreferrer">jQuery</a> like this:</p> <pre><code>$('div[id^=myDiv]') </code></pre> <p>If you can't use jQuery, you'll need to call <code>getElementsByTagName</code> and loop through the values checking the ID property.</p>
<p>you can use jquery </p> <pre><code>//this will give you all divs start with myDiv in the id var divs = $("div[id^='myDiv']"); </code></pre>
<pre><code>var divs = document.getElementsByTagName('div'); var counter = 0; for(var i in divs) { if(divs[i].id.indexOf('myDiv') === 0) { counter++; } } </code></pre> <p>or just</p> <pre><code>document.querySelectorAll('[id^=myDiv]').length </code></pre>
