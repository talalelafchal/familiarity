JavaScript text manipulation
<p>Using JavaScript and I want to replace any text between @anytext@ with some text. I want to make it generic so I am thinking to make use of regular expression. How do I do it?</p> <p>Example:replace('@hello@','Hi')</p>
<p>You can use the regex function of jquery to accomplish that... So find the @'s with a regular expression and afterwards use the replace function with the text you want.</p>
<p>Try this:</p> <pre><code>str.replace(/@[^@]+@/g, 'Hi') </code></pre> <p>This will remove any sequences of <code>@ … @</code> globally with <code>Hi</code>.</p> <hr> <p><strong>Edit</strong>    Some explanation:</p> <ul> <li><code>/…/</code> is the <a href="https://developer.mozilla.org/en/Core_JavaScript_1.5_Reference/Global_Objects/RegExp" rel="nofollow noreferrer">regular expression literal syntax in JavaScript</a></li> <li><code>@[^@]+@</code> describes any sequence of a literal <code>@</code>, followed by one or more (<code>+</code> quantifier) characters that is not a <code>@</code> (negated charcater class <code>[^@]</code>), followed by a literal <code>@</code></li> <li>the <code>g</code> flag in <code>/…/g</code> allows global matches; otherwise only the first match would be replaced</li> </ul>
