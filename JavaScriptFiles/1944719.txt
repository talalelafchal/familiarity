<pre><code>var txt = document.getElementById('ID-OF-FIELD'); var value = txt.value; </code></pre>
<p>Your Javascript should look similar to this:</p> <pre><code>var elm = document.getElementById('myElement'); var value = elm.value; </code></pre> <p>And your HTML:</p> <pre><code>&lt;input type="text" id="myElement" value="My Value" /&gt; </code></pre>
<p>Try this:</p> <pre><code>&lt;input type="text" id="testid" value="" /&gt; </code></pre> <p>Now you can get the value of above text box like this:</p> <pre><code>document.getElementById('testid').value; </code></pre> <p>Other way:</p> <pre><code>document.form_name_here.element_name.value; </code></pre> <p>Simply run an alert to check if value comes:</p> <pre><code>alert(document.getElementById('testid').value); </code></pre>
