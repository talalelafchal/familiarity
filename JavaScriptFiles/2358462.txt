how to find out how many items are selected in select box
<p>I have a select box which has more than 200 items. </p> <p>Below the select box is the submit button which submits the form using javascript. </p> <p>I want to restrict the user to only select 100 items from the select box. If they select box and try to submit the form then it will say <code>Select 100 Only</code></p> <p>How can I do this in Javascript?</p>
<p>You can either:</p> <p>Loop through all of the options elements and check the selected attribute:</p> <pre><code>var cnt = 0; for (var i = 0; i &lt; selectbox.options.length; i++) { if (selectbox.options[i].selected === true) { cnt++; } } return cnt; </code></pre> <p>Or you can get the <code>value</code> of the select element and split the value by comma and base it on the number of items in the array.</p> <pre><code>var vlu = selectbox.value; var vlus = vlu.split(','); return vlus.length; </code></pre>
