How to handle a clicked link in JavaScript?
<p>How can I handle a link which doesn't have an id? It just has a classname like "classbeauty".</p> <p>Now I need to know if a user has clicked the link. If the link is clicked I just need to call <code>alert("yes link clicked");</code>.</p> <p>I don't know how to handle events in JavaScript. </p> <p>How can I do that? </p>
<p>If jQuery is an option: </p> <pre><code> $("a.classbeauty").click(function(){ alert("yes link clicked"); }); </code></pre> <p>If you need pure JavaScript:</p> <pre><code>var elements = document.getElementsByTagName("a"); for(var i=0; i&lt;elements.length; i++){ if (elements[i].className == 'classbeauty') { elements[i].onclick = function(){ alert("yes link clicked); } } } </code></pre> <p>If you need it in <a href="http://en.wikipedia.org/wiki/Greasemonkey" rel="nofollow noreferrer">Greasemonkey</a>:</p> <pre><code>function GM_wait() { if(typeof unsafeWindow.jQuery != 'function') { window.setTimeout(wait, 100); } else { unsafeWindow.jQuery('a.classbeauty').click(function(){ alert('yes link clicked'); }); } } GM_wait(); </code></pre>
<pre><code>function getElementsByClassName(class_name) { var elements = document.getElementsByTagName('*'); var found = []; for (var i = 0; i &lt; elements.length; i++) { if (elements[i].className == class_name) { found.push(elements[i]); } } return found; } getElementsByClassName("YourClass")[0].onclick = function () { alert("clicked"); }; </code></pre> <p>This is pure javascript, by the way. Everyone here loves jQuery (including me).</p>
<pre><code>for (var i= document.links.length; i--&gt;0;) { if (document.links[i].className==='classbeauty') { document.links[i].onclick= function() { alert('yes link clicked'); return false; // if you want to stop the link being followed } } } </code></pre>
<pre><code>function handleLinks(className, disableHref) { var links = document.body.getElementsByTagName("a"); for (var i = 0; i &lt; links.length; i++) { if (links[i].className == className) { links[i].onclick = function(){ alert('test') if (disableHref) { return false; //the link does not follow } } } } } handleLinks('test', true) </code></pre> <p>you can test it at this link: <a href="http://jsfiddle.net/e7rSb/" rel="nofollow noreferrer">http://jsfiddle.net/e7rSb/</a></p>
<p>I'm not sure if this is an option, but I would flip a flag when the link is clicked, and then just test for that variable.</p> <pre><code>//javascript var _cbClicked = false; function checkClicked() { if (_cbClicked) alert("link was clicked"); return false; } //html &lt;a href="#" class="classbeauty" onclick="_cbClicked = true; return false;"&gt;Link Text&lt;/a&gt; &lt;a href="#" onclick="checkClicked();"&gt;Check Link&lt;/a&gt; </code></pre> <p>Very simple, and pure JS :)</p>
