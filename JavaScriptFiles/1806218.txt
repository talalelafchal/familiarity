JavaScript: event triggering
<p>How would I go about developing cross-browser event triggering for mouse buttons (left, middle and right) in <b>plain-vanalla</b> JavaScript?</p>
<p>It would be strongly recommended you use a Javascript Library for this purpose.</p> <p>Most popular being <a href="http://jquery.com" rel="nofollow noreferrer">JQuery</a> (recommended)</p> <p>For additional libraries you can view the following <a href="http://stackoverflow.com/questions/74048/what-is-the-best-lightweight-javascript-framework">what-is-the-best-lightweight-javascript-framework</a></p> <p>Do you have a valid reason for not using any libraries...?</p> <p>Regards</p> <p>Gordon</p>
<p>A few starting points</p> <ul> <li><a href="https://developer.mozilla.org/en/DOM/document.createEvent" rel="nofollow noreferrer"><code>document.createEvent</code></a></li> <li><a href="https://developer.mozilla.org/en/DOM/element.dispatchEvent" rel="nofollow noreferrer"><code>element.dispatchEvent</code></a></li> <li><a href="http://msdn.microsoft.com/en-us/library/ms536423(VS.85).aspx" rel="nofollow noreferrer"><code>fireEvent</code> Method</a></li> <li><a href="http://msdn.microsoft.com/en-us/library/ms536390(VS.85).aspx" rel="nofollow noreferrer"><code>createEventObject</code> Method</a></li> <li><a href="http://github.com/kangax/protolicious/blob/5b56fdafcd7d7662c9d648534225039b2e78e371/event.simulate.js" rel="nofollow noreferrer">event.simulate.js</a></li> <li>A look into how known big Javascript Frameworks do it (e.g. <a href="http://ajax.googleapis.com/ajax/libs/jquery/1.3.2/jquery.js" rel="nofollow noreferrer"><code>jQuery</code></a>)</li> <li>And don't forget <ul> <li>grab a cup of coffee (or better two)</li> <li>be ready to get many headaches</li> <li>be ready to encounter subtle bugs which will make you want to rip your hear out</li> <li>to make it crossbrowser you will need to do a lot(!) of testing</li> </ul></li> </ul>
