how to get hidden element value in javascript
<p>how to get hidden element value in javascript</p>
<p>If the element has an id attribute and if it has a value attribute then you can use value property</p> <pre><code>document.getElementById( "hidElem" ).value; </code></pre> <p>for a hidden input element like</p> <pre><code>&lt;input type="hidden" id="hidElem" /&gt; </code></pre> <p>otherwise you can use <code>textContent</code> property</p> <pre><code>document.getElementById( "hidElem" ).textContent; </code></pre> <p>for a hidden div element like</p> <pre><code>&lt;div style="display: none;" id="hidElem"&gt;value of hidden element&lt;/div&gt; </code></pre>
<p>using the following document function</p> <pre><code>document.getElementById("elementId").value; </code></pre> <p>elementId-> id defined for the hidden element</p>
<p>Putting ids on all your elements is overkill in my opinion. You can access them by their name attribute - all you need is a reference to the form object.</p> <pre><code>&lt;form id="blah"&gt; &lt;input type="hidden" name="inputOne" value="1" /&gt; &lt;input type="hidden" name="inputTwo" value="2" /&gt; &lt;/form&gt; var formObj = document.getElementById('blah'); alert("Input one value: " + formObj.inputOne.value); alert("Input two value: " + formObj.inputTwo.value); </code></pre> <p>This applies to all types of form input.</p>
