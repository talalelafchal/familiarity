Unit conversion in Javascript
<p>I have a question concerning number conversion i JS. I have a number like 1613841.93424 (meters) and I would like it to convert to 1.6 km instead. What JS function should I use for that? Thanks.</p>
<p>To display with one number to the right of the decimal (if that is what you're asking..I'm guessing you know how to convert m to km) :</p> <pre><code>.toFixed(1) </code></pre>
<p>To convert from metres to kilometres, simply divide by 1000. To format a number using fixed-point notation, you can use the <a href="https://developer.mozilla.org/en/Core%5FJavaScript%5F1.5%5FReference/Global%5FObjects/Number/toFixed" rel="nofollow noreferrer">toFixed()</a> method:</p> <pre><code>var km = 1613841.93424 / 1000; alert(km.toFixed(1) + " km"); // 1613.8 km </code></pre> <p><strong>Note:</strong> 1613841.93424 meters != 1.6 km (<a href="http://en.wikipedia.org/wiki/Kilometre" rel="nofollow noreferrer">Source</a>)</p>
<p>i usually deal with float padding numbers with a function of my own</p> <pre> var zeroPad = function(num, pad){ var pd = Math.pow(10,pad); return Math.floor(num*pd)/pd; } alert(zeroPad(1.32878,3)); // outputs 1,328 alert(zeroPad(1.32878,1)); // outputs 1,3 </pre> <p>Then finally to convert to kms divide by 1000, apply zeroPad and solved<br /> example :</p> <pre> var m = 1613841.93424; var km = zeroPad(m/1000,3); alert(km); // outputs 1613.841 </pre>
<p>1613841.93424 m = 1613,841.93424 km</p> <p>Simply divide by 1000.</p>
<p>Use Math.round:</p> <pre><code>m = 1613841.93424; var km = Math.round(m / 100) / 10; // km = 1613.8 </code></pre> <p>This will give a result rounded to 1/10th km</p>
