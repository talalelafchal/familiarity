It seems to be not working, how can I do this?</p> <p>Edit: Sorry folks It says undefined.</p> <pre><code> var lol=new Array( "test", "test2" ); var x = 0; while( x == 4 ) { number = parseInt(document.getElementById(lol[x]).value); x++; } </code></pre> <p>And i have inputs id named test and test2.</p>
<p>Try taking your array out of the quotes...</p> <pre><code>document.getElementById(lol[x]).value </code></pre> <p>The quotes turn it into a static string "lol[x]", when you want the value of the lol array at x index.</p> <p>This replaces my earlier, less informed answer.</p> <p>Hope this helps</p>
<p>Your while loop only works if x==4. Change this to:</p> <pre><code>while(x &lt; lol.length) </code></pre> <p>To loop through all the elements in the array. Better yet, this will condense your loop:</p> <pre><code>var lol=new Array( "test", "test2" ); for( var x = 0; x &lt; lol.length; x++ ) { number = parseInt(document.getElementById(lol[x]).value); } </code></pre>
<p>You say you have <code>number = parseInt(document.getElementById("lol[x]").value);</code></p> <ol> <li><p><code>"lol[x]"</code> is a string with that literal value, not the value that <code>lol</code> holds at index <code>x</code>. Use <code>getElementById(lol[x])</code></p></li> <li><p><code>parseInt</code> may do unexpected things when you don't pass a radix. Use something like <code>parseInt(document.getElementById(lol[x]).value, 10)</code></p></li> </ol> <p>Finally, you aren't checking whether the element exists. Do something like:</p> <pre><code>var element = document.getElementById(lol[x]); if (element) { number = parseInt(element.value, 10); } else { // handle error or throw exception } </code></pre>
