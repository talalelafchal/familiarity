date to timestamp in javascript
<p>Is it possible in javascript to convert some date in timestamp ?</p> <p>i have date in this format <code>2010-03-09 12:21:00</code> and i want to convert it into its equivalent time stamp with javascript.</p>
<p>The <code>getTime()</code> method of <code>Date</code> object instances returns the number of milliseconds since the epoch; that's a pretty good timestamp.</p>
<pre><code>+(new Date()) </code></pre> <p>Does the job.</p>
<p>In response to your edit:</p> <p>You need to parse the <em>date string</em> to build a <code>Date</code> object, and then you can get the timestamp, for example:</p> <pre><code>function getTimestamp(str) { var d = str.match(/\d+/g); // extract date parts return +new Date(d[0], d[1] - 1, d[2], d[3], d[4], d[5]); // build Date object } getTimestamp("2010-03-09 12:21:00"); // 1268158860000 </code></pre> <p>In the above function I use a simple regular expression to extract the digits, then I build a new Date object using the <a href="https://developer.mozilla.org/en/Core_JavaScript_1.5_Reference/Objects/Date" rel="nofollow noreferrer">Date constructor</a> with that parts (<em>Note:</em> The Date object handles months as 0 based numbers, e.g. 0-Jan, 1-Feb, ..., 11-Dec).</p> <p>Then I use the unary plus operator to get the timestamp.</p> <p>Note also that the timestamp is expressed in milliseconds.</p>
