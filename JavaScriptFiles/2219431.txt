How do I use regular javascript to look through every &lt;a&gt; tag and change the href?
<p>A page has many tags. How do I loop through all of them and replace their "href" with "<a href="http://example.com" rel="nofollow noreferrer">http://example.com</a>"?</p> <p>(do not use jQuery)</p>
<pre><code>for(var i=0,L=document.links.length;i &lt;L; i++) { document.links[i].href = "http://example.com"; } </code></pre> <p>Or load a 20 kb library and write a little less code.</p>
<p>You must use <a href="https://developer.mozilla.org/en/DOM/element.getElementsByTagName" rel="nofollow noreferrer"><code>getElementsByTagName()</code></a> to fetch all the links, and then loop through them to change the <code>href</code> property.</p> <pre><code>var links = document.getElementsByTagName('a'); if(links) { // if none are found, do not continue for(var i = 0; i &lt; links.length; i++) { links[i].href = 'http://example.com/'; } } </code></pre>
<pre><code>var links = document.getElementsByTagName("a"); for (i=0;i&lt;links.length;i++) links[i].href = "http://example.com"; </code></pre>
<pre><code>var links = document.getElementsByTagName("a"); for (var i = 0; i &lt; links.length; i++) { links[i].href = "http://example.com"; } </code></pre>
<p>You can use the <code>document.links</code> collection. It's defined by the W3C and supported by all common browsers.</p> <p>Moreover you get access not only to <code>&lt;a&gt;</code> elements, but to <code>&lt;area&gt;</code> tags too (which are commonly used in client image maps).</p> <pre><code>for(var i=0; i &lt; document.links.length; i++) { document.links[i].href = "http://example.com"; } </code></pre>
