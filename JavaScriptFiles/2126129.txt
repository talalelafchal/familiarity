<p>First, you need your <code>id</code> values to be different. You should never have the same ID twice on the same page. So lets use this as the example HTML:</p> <pre><code>&lt;input type="text" id="name_0" name="name" /&gt; &lt;input type="text" id="phone_0" name="phone" /&gt; &lt;input type="text" id="email_0" name="email" /&gt; </code></pre> <p>You could use this JavaScript function:</p> <pre><code>&lt;script type='text/javascript'&gt; function clearRow(id){ var name = document.getElementById('name_' + id), phone = document.getElementById('phone_' + id), email = document.getElementById('email_' + id); // Clear values name.value = phone.value = email.value = ""; } &lt;/script&gt; </code></pre> <p>And your <code>img</code> tag would remain unchanged:</p> <pre><code>&lt;img src="images/delete_row.png" width="25" onClick="clearRow(0);" /&gt; </code></pre>
<blockquote> <p>I have three input text fields that all have the id of "0".</p> </blockquote> <p>This is entirely wrong. In a document you can't have element with the same id. Either use a name or a classname for these textfields and make their ids different.</p> <pre><code>&lt;script type="text/javascript"&gt; function Change() { var elems = document.getElementsByName ( "myfields"); for ( var i = 0;i &lt; elems.length; i++) { elems[i].value = ""; } } &lt;/script&gt; &lt;input name="myfields" type="text" id="txt1" /&gt; &lt;input name="myfields" type="text" id="txt2" /&gt; &lt;input name="myfields" type="text" id="txt3" /&gt; &lt;img onclick="Change();" alt="test" src="yourimagpath" /&gt; </code></pre>
