JScript runtime error: 'System' is undefined&quot;
<p>I have JScript method that is :</p> <pre><code>function init() { var userid = System.Environment.getEnvironmentVariable("USERNAME"); setUserName(userid); } </code></pre> <p>When I debug it , at first line of the init function, I'll get "javascript Microsoft JScript runtime error: 'System' is undefined".</p> <p>I'd be thankful if some one let me know how can I solve it.</p>
<p>You're using code specific to Windows Sidebar/Desktop Gadgets. The host for Windows Desktop Gadgets provides the <code>System</code> namespace for access to computer properties that are not normally available to Javascript.</p> <p>Gadgets can be coded in Javascript which is most likely how you picked up that source code. Using this namespace outside of a Windows Desktop Gadget (ie in a web page or shell script) wouldn't work because it is provided by the host (sidebar.exe).</p> <p>See <a href="http://msdn.microsoft.com/en-us/library/ms723688(VS.85).aspx" rel="nofollow noreferrer">http://msdn.microsoft.com/en-us/library/ms723688(VS.85).aspx</a>.</p>
