What's the principle inside this js tree?
<p><a href="http://jsbin.com/idala" rel="nofollow noreferrer">http://jsbin.com/idala</a></p> <p>How is it implemented?</p>
<p>Javascript making use of x, y coordinates, arrays and bit of css. You got to analyze it closely to get an idea of how it works. Of course you should have good understanding of javascript. :)</p>
<p>The creator used <a href="http://www.packtpub.com/article/using-prototype-property-in-javascript" rel="nofollow noreferrer">javascript prototype feature</a> (not the popular <a href="http://www.prototypejs.org/" rel="nofollow noreferrer">Prototype framework</a>) to create a Christmas Tree class named chrisTree.</p> <p>There are methods implemented to perform the 'drawing' of the tree as well as the animation by:</p> <ul> <li>Dynamically creating DOM elements to represent the trees </li> <li>Manipulation of CSS styles </li> <li>Using <code>setTimeout</code> method for animation effect triggering </li> <li>etc. etc.</li> </ul> <p>Nice work! </p>
<ol> <li><p>Creating div elements which are absolutely positioned so that it appears as a tree.</p></li> <li><p>Settimeout used to light up divs</p></li> <li><p>Colors are stored in an array</p></li> </ol> <p>You can view the code using the following link and understand what is being done</p> <p><a href="http://jsbin.com/idala/edit" rel="nofollow noreferrer">http://jsbin.com/idala/edit</a></p>
