How could I delete an item from an array by the item's name?
<p>Like I have an array <b>Array('one', 'three', 'two');</b>, and, say, I want to delete the item three, how would I do it?</p>
<pre><code>var a = ['one', 'three', 'two']; a.splice(a.indexOf('three'), 1); alert(a); </code></pre> <p>for browsers that don't support indexOf there's a workaround <a href="https://developer.mozilla.org/En/Core_JavaScript_1.5_Reference/Objects/Array/IndexOf#Compatibility" rel="nofollow noreferrer">https://developer.mozilla.org/En/Core_JavaScript_1.5_Reference/Objects/Array/IndexOf#Compatibility</a></p>
<pre><code>function RemoveArrayValue(arr, val) { var result = [], //empty array j = 0, i, len = arr.length; for(i = 0; i &lt; len; ++i) if(arr[i] != val) result[j++] = val; return result; } </code></pre> <p>BTW:</p> <p>the indexOf method is not part of the Javascript Array object under IE (also not part of IE8), but of the String object. If you want the indexOf method to be part of the JS Array object you must declare this in your code:</p> <pre><code>if (!Array.prototype.indexOf) { Array.prototype.indexOf = function(val) { for(var i = 0, len = this.length; i &lt; len; ++i) if(this[i] === val) return i; return -1; } } </code></pre>
