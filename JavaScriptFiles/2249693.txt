how to count total number of divs inside another div using javascript
<p>How to count the total number of <code>div</code> elements that are contained in another <code>div</code> using javascript?</p>
<p>The <code>getElementsByTagName()</code> is not only a <code>document</code> method, but one that can run on any DOM element.</p> <blockquote> <p><strong>element</strong>.<em>getElementsByTagName</em> is similar to <strong>document</strong>.<em>getElementsByTagName</em>, except that its search is restricted to those elements which are descendants of the specified element</p> </blockquote> <p>see more at <a href="https://developer.mozilla.org/en/DOM/element.getElementsByTagName" rel="nofollow noreferrer">https://developer.mozilla.org/en/DOM/element.getElementsByTagName</a></p> <hr> <p>So the actual code that does what you ask is</p> <pre><code>var container_div = document.getElementById('id_of_container_div'); var count = top_level_div.getElementsByTagName('div').length; </code></pre>
<p>There are many way to count divs element using jquery.</p> <p>But most popular and simple way are:</p> <pre><code>$(document).ready(function(){ var divCount = $("div").size(); alert(divCount); }); </code></pre> <p>AND</p> <pre><code>$(document).ready(function(){ var divCount = $("div").length; alert(divCount); }); </code></pre> <p>Its helpful for you</p>
<p>You can use @davidcmoulton's handy Gist: <a href="https://gist.github.com/davidcmoulton/a76949a5f35375cfbc24" rel="nofollow">https://gist.github.com/davidcmoulton/a76949a5f35375cfbc24</a></p> <p>I find it quite useful that it doesn't only count DIVs but also lists the count of all element types of your page.</p> <p>Here is a copy of the Gist for further reference:</p> <pre><code>(function (window, undefined) { // Counts all DOM elements by name &amp; logs resulting object to console. var forEach = Array.prototype.forEach, counter = {}, incrementElementCount = function (elementName) { if (counter.hasOwnProperty(elementName)) { counter[elementName] += 1; } else { counter[elementName] = 1; } }, processNode = function (node) { var currentNode = node; if (currentNode.nodeType === currentNode.ELEMENT_NODE) { incrementElementCount(currentNode.nodeName); if (currentNode.hasChildNodes) { forEach.call(currentNode.childNodes, function (childNode) { if (childNode.nodeType === currentNode.ELEMENT_NODE) { processNode(childNode); } }); } } }; processNode(window.document.firstElementChild); console.log(counter); }(this)); </code></pre>
