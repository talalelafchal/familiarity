<p>There is a good example <a href="http://www.php-help.ro/mootools-12-javascript-examples/javascript-drop-down-menu-using-mootools-12/" rel="nofollow noreferrer">here</a> including all of the necessary HTML, Javascript and CSS.</p> <p>HTML:</p> <pre><code>&lt;div id="menu-container"&gt; &lt;ul id="drop_down_menu"&gt; &lt;li class="menu"&gt;Menu 1 &lt;ul class="links"&gt; &lt;li&gt;&lt;a href="#"&gt;Link 1&lt;/a&gt;&lt;/li&gt; &lt;li&gt;&lt;a href="#"&gt;Link 2&lt;/a&gt;&lt;/li&gt; &lt;/ul&gt; &lt;/li&gt; &lt;li class="menu"&gt;Menu 2 &lt;ul class="links"&gt; &lt;li&gt;&lt;a href="#"&gt;Link 1&lt;/a&gt;&lt;/li&gt; &lt;li&gt;&lt;a href="#"&gt;Link 2&lt;/a&gt;&lt;/li&gt; &lt;li&gt;&lt;a href="#"&gt;Link 3&lt;/a&gt;&lt;/li&gt; &lt;/ul&gt; &lt;/li&gt; &lt;/ul&gt; </code></pre> <p></p> <p>The Javascript:</p> <pre><code>window.addEvent('domready', function(){ $('drop_down_menu').getElements('li.menu').each( function( elem ){ var list = elem.getElement('ul.links'); var myFx = new Fx.Slide(list).hide(); elem.addEvents({ 'mouseenter' : function(){ myFx.cancel(); myFx.slideIn(); }, 'mouseleave' : function(){ myFx.cancel(); myFx.slideOut(); } }); }) }); </code></pre>
