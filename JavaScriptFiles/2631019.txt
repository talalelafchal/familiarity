how to print entire HTML element in JavaScript?
<p>I want to print the entire element including tag name, attribute name/value pairs and innerHTML. How can I do it in JavaScript (jQuery)?</p> <p>for example:</p> <pre><code>var elArr = document.getElementsByTagName('link'); alert(elArr[0].printEntireElement()); //expected output might be &lt;link href="/css/common.css" rel="stylesheet" type="text/css"&gt;` </code></pre> <p>Note that for link element outerHTML is not defined!</p>
<p>Use an outerHTML jQuery plugin, like <a href="http://dbj.org/dbj/?p=91" rel="nofollow noreferrer">this one</a>, or <a href="http://brandonaaron.net/blog/2007/06/17/jquery-snippets-outerhtml/" rel="nofollow noreferrer">this one</a>.</p>
<p>you can do something like this:</p> <pre><code>function dump(element) { var a = ["Element dump:"]; for (var k in element) { if (element.hasOwnProperty(k)) { a.push(k + ": " + element[k]); } } a.push("HTML: " + element.innerHTML); alert(a.join('\n')); } </code></pre> <p>Whether you want the "hasOwnProperty" test or not, I don't know.</p>
<p>If you can't do outerHtml, clone it, create a new div, place the clone in the div, and grab the innerHTML of the container. </p>
<pre><code>document.getHTML= function(who){ var txt, ax, el= document.createElement("div"); el.appendChild(who.cloneNode(false)); txt= el.innerHTML; ax= txt.indexOf('&gt;')+1; txt= txt.substring(0, ax)+who.innerHTML+ txt.substring(ax); el= null; return txt.replace(/&gt;/g,'&gt;\n'); } </code></pre>
<p>If you want to print an entire tag from a jQuery element you could do something like this:</p> <pre><code>var el = $('&lt;div id="my_div"&gt;test div 123&lt;/div&gt;'); alert(el[0].outerHTML); </code></pre>
