<p>please can some one tell me what is this window name in <code>window.open()</code> is it title name or some Id in java script language?</p>
<p>From <strong><a href="https://developer.mozilla.org/En/DOM/Window.open" rel="nofollow noreferrer">MDC on window.open()</a></strong></p> <pre><code>window.open(strUrl, strWindowName [, strWindowFeatures]); </code></pre> <blockquote> <p><strong>strWindowName</strong><br> This is the string that just names the new window. Such string can be used to be the target of links and forms when the target attribute of an <code>&lt;a&gt;</code> element or of a <code>&lt;form&gt;</code> is specified. This string parameter should not contain any blank space. <code>strWindowName</code> does not specify the title of the new window.</p> </blockquote>
<pre><code>windowName </code></pre> <p>A name to be given to the new window. The name can be used to refer this window again. </p> <p>After opening the window you will want to do all sorts of things with it e.g. move it it then you can do</p> <pre><code>&lt;html&gt; &lt;head&gt; &lt;title&gt;Window Example&lt;/title&gt; &lt;/head&gt; &lt;SCRIPT language="JavaScript1.2"&gt; function poponload() { testwindow= window.open ("", "mywindow"); alert('I will move window to 0,0'); testwindow.moveTo(0,0); } &lt;/SCRIPT&gt; &lt;body onload="javascript: poponload()"&gt; &lt;H1&gt;Window Example&lt;/H1&gt; &lt;/body&gt; &lt;/html&gt; </code></pre> <p>And <strong>NO</strong> its not <code>window title</code> its different.</p> <p><a href="http://www.javascript-coder.com/window-popup/javascript-window-open.phtml" rel="nofollow noreferrer">-source</a></p>
<p>From the <a href="http://msdn.microsoft.com/en-us/library/ms536651(VS.85).aspx" rel="nofollow noreferrer">documentation</a>:</p> <p><strong>sName</strong></p> <pre><code>Optional. String that specifies the name of the window. This name is used as the value for the TARGET attribute on a form or an anchor element. _blank The sURL is loaded into a new, unnamed window. _media The url is loaded in the Media Bar in Microsoft Internet Explorer 6. Windows XP Service Pack 2 (SP2) and later. This feature is no longer supported. By default the url is loaded into a new browser window or tab. _parent The sURL is loaded into the current frame's parent. If the frame has no parent, this value acts as the value _self. _search Disabled in Windows Internet Explorer 7, see Security and Compatibility in Internet Explorer 7 for details. Otherwise, the sURL is opened in the browser's search pane in Internet Explorer 5 or later. _self The current document is replaced with the specified sURL. _top sURL replaces any framesets that may be loaded. If there are no framesets defined, this value acts as the value _self. </code></pre>
