How do we prevent default actions in JavaScript?
<p>What is the cross-browser method? I need to prevent any default action on an image, so that neither dragging nor anything else will fire on a default bases.</p>
<p>You can only cancel specific events. You cannot "globally cancel" default actions.</p> <p>To specifically cancel <code>dragging</code> an image (which is only a default function in some browsers), return <code>false</code> to the <code>mousedown</code> event. </p>
<p>You can register the events you want to cancel, and then either <code>return false</code> from them or use <code>Event.preventDefault()</code>, depending on the browser and event.</p>
<pre><code>(function() { var onmousedown; if('onmousedown' in document &amp;&amp; typeof document.onmousedown == 'function') { onmousedown = document.onmousedown; } document.onmousedown = function(e) { if(typeof e == 'undefined') { e = window.event; } if(!e.target) { e.target = e.srcElement || document; } if('nodeName' in e.target &amp;&amp; e.target.nodeName.toLowerCase() == 'img') { if(e.preventDefault) { e.preventDefault(); } // If you want to register mousedown events for // elements containing images, you will want to // remove the next four lines. if(e.stopPropagation) { e.stopPropagation(); } e.cancelBubble = true; e.returnValue = false; return false; } if(onmousedown !== undefined) { onmousedown(e); } }; })(); </code></pre> <p>You may need to do something similar to other events you'd like to prevent, if this doesn't do what you want.</p> <p>Also it's worth noting that if you're trying to prevent people from downloading images from a page to their computer, you will not succeed. If a browser can download an image, so can the user. Using JavaScript to block (some) attempts is easily circumvented by simply disabling JavaScript.</p>
