Find the number in line. Javascript
<p>How do I find the number in line number may not be in the beginning. For example: "d: \ \ 1.jpg"</p> <p>Thank you.</p>
<p>You can use a <a href="https://developer.mozilla.org/en/Core_JavaScript_1.5_Guide/Regular_Expressions" rel="nofollow noreferrer">regexp</a>:</p> <pre><code>var match = "testing 123".match(/\d/); if (match) { alert(match.index); // alerts 8, the index of "1" in the string } </code></pre> <p>That uses <a href="https://developer.mozilla.org/En/Core_JavaScript_1.5_Reference/Objects/String/Match" rel="nofollow noreferrer">String#match</a>, passing in a literal regexp using the "digit" class (<code>\d</code>).</p> <p>And/or you can grab all contiguous digits starting with the first digit found:</p> <pre><code>var match = "testing 123".match(/\d+/); if (match) { alert(match.index); // alerts 8, the index of "1" in the string alert(match[0]); // alerts "123" } </code></pre> <p>Those links are to the Mozilla documentation because it's fairly good, but these are not Mozilla-specific features.</p>
<p>You use a Regular Expression with the <a href="http://www.regular-expressions.info/javascript.html" rel="nofollow noreferrer"><code>RegExp</code></a> object:</p> <pre><code>var myRegEx = /\d+/; // RegEx to find one or more digits var myMatch = myRegEx.exec("d:\\1.jpg") </code></pre>
