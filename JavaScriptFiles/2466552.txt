Name an id dynamically with javascript?
<p>How can I dynamically name id's using javascript?</p> <p>Something like this:</p> <p>js:</p> <pre><code>var idName = "fruit"; </code></pre> <p>html:</p> <pre><code>&lt;img id="javascript:idName" src="banana.jpg"&gt; </code></pre>
<p>Using the jQuery framework you could do something like:</p> <pre><code>&lt;img class="idName" src="banana.jpg"/&gt; &lt;img class="idName" src="cherry.jpg"/&gt; </code></pre> <p>The script ...</p> <pre><code>var idName = 'fruit'; $(function() { $('img.idName').each(function(i) { $(this).attr({id: idName+i}); }); }); </code></pre> <p>... which results in:</p> <pre><code>&lt;img id="fruit0" class="idName" src="banana.jpg"/&gt; &lt;img id="fruit1" class="idName" src="cherry.jpg"/&gt; </code></pre>
<p>or you can get the element any other way IE</p> <pre><code>imgs = container.getElementsByTagName("img"); foreach(imgs as img){ if(imgs.src == "banana.jpg") img.id = "fruit"; } </code></pre> <p>Note: foreach doesn't work in JS you'll need a for loop, but I'm too lazy :P</p>
<p>I think this is what your looking for. You need some sort of event to trigger the JavaScript. Since IMG doesnt have it, except for user events, you need something like:</p> <pre><code>&lt;head&gt; &lt;script language="JavaScript" type="text/javascript"&gt; function nameMyIds() { var idToName = "firstImageId"; var theImage = getImageBySrc("banana.jpg"); if (theImage!=null) theImage.id = idToName; } function getImageBySrc(src) { // using DOM, using jQuery would make this easier var allImages = document.body.getElementsByTagName("IMG"); for (var i=0; i&lt;allImages.length; i++ ) { var img = allImages[i]; var i = img.src.indexOf(src); if (img.src == src || img.src.indexOf(src) &gt; 0) { return img }; } } &lt;/script&gt; &lt;/head&gt; &lt;body onload="nameMyIds()"&gt; &lt;img src="banana.jpg"&gt; &lt;/body&gt; </code></pre>
