How to call JS to click a table element
<p>Assume I get a table element with ID="emTab", how do I call JS to click it?</p> <p>Thanks.</p>
<p><strike> document.getElementById("emTab").onclick = function() { // your code goes here };</p> <p>See <strong><a href="https://developer.mozilla.org/en/DOM/element.onclick" rel="nofollow noreferrer">element.onclick</a></strong></p> <p></strike></p> <p>To trigger click event</p> <pre><code>document.getElementById("emTab").click(); </code></pre> <p>See <strong><a href="https://developer.mozilla.org/en/DOM/element.click" rel="nofollow noreferrer">element.click</a></strong></p> <blockquote> <p>The click method is intended to be used with INPUT elements of type button, checkbox, radio, reset or submit. Gecko does not implement the click method on other elements that might be expected to respond to mouse–clicks such as links (A elements), nor will it necessarily fire the click event of other elements.</p> <p>Non–Gecko DOMs may behave differently.</p> <p>When a click is used with elements that support it (e.g. one of the INPUT types listed above), it also fires the element's click event which will bubble up to elements higher up the document tree (or event chain) and fire their click events too. However, bubbling of a click event will not cause an A element to initiate navigation as if a real mouse-click had been received.</p> </blockquote> <p><strong><em>Cross browser way</em></strong></p> <p>If you can use jQuery then it would be</p> <pre><code>$("#emTab").trigger("click"); </code></pre>
<p>its simple using JQuery $('#emTab').click(functionToCall);</p> <p>while in JS document.getElementById('emTab').onclick = function() {};</p> <p>for details on DOM events: <a href="http://www.howtocreate.co.uk/tutorials/javascript/domevents" rel="nofollow noreferrer">http://www.howtocreate.co.uk/tutorials/javascript/domevents</a></p>
