How to access a numeric property?
<p>I have an object like:</p> <pre><code>var myObject = { '0' : 'blue' }; </code></pre> <p>Now, when I try to access the value of the key '0' like:</p> <pre><code>myObject.0 </code></pre> <p>...I am getting an error. (Maybe this is not the proper way?)</p> <p>How can I access the value of a key that is a number (like the above)?</p>
<p>This should work:</p> <pre><code>myObject["0"] </code></pre> <p>(<code>myObject["propertyName"]</code> is an alternative syntax for <code>myObject.propertyName</code>.)</p> <p>You're getting the error because, in JavaScript, identifiers can't begin with a numeral. From the <a href="https://developer.mozilla.org/en/Core_JavaScript_1.5_Guide/Variables#Variables" rel="nofollow noreferrer">Variables</a> page at the <a href="https://developer.mozilla.org/" rel="nofollow noreferrer">Mozilla Developer Centre</a>:</p> <blockquote> <p>A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($); subsequent characters can also be digits (0-9). Because JavaScript is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase).</p> </blockquote>
<p><code>myObject["0"]</code></p>
<p>if you have data like </p> <pre><code> `"rain": { "3h": 0 },` </code></pre> <p>then you can simply access it <code>rain['3h']</code></p>
