Adding SVG element to an html file?
<p>I'd like to add some SVG elements to an html file. I'm finding links which say this is not supported, the document type must be xhtml.</p> <p>But I am using some javascript libraries with html that allow me to use SVG in them (Raphael, SVG Web, etc).</p> <p>So is there a way to add a script to an html file which can dynamically add an SVG element to the document at run time?</p> <p>Thanks</p>
<pre><code>var svgnode = document.createElementNS('http://www.w3.org/2000/svg','svg'); var circle = document.createElementNS('http://www.w3.org/2000/svg','circle'); circle.setAttribute("cx", "100"); circle.setAttribute("cy", "100"); circle.setAttribute("r", "40"); circle.setAttribute("fill", "blue"); svgnode.appendChild(circle); mydiv.appendChild(svgnode); </code></pre>
<p>You might want to take a look at <a href="http://code.google.com/p/svgweb/" rel="nofollow noreferrer">svgweb</a>, which allows you to put SVG directly into a <code>&lt;script&gt;</code> tag in HTML. It will also emulate SVG support using Flash for browsers which do not have native SVG support.</p> <p><strong>edit</strong>: Hmm. I just noticed that you mentioned svgweb already.
