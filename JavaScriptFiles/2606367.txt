limit the choices of acceptable inputs using javascript
<p>Is it possible to prevent user from writing letters to a textbox (i.e. force user to enter only numbers in textbox) using javascript?</p>
<p>Sure, you put an event handler on keydown events and cancel those for non-digits when the relevant text box has the focus. See <a href="https://developer.mozilla.org/en/DOM/element.onkeydown" rel="nofollow noreferrer"><code>element.onkeydown</code> event</a>.</p> <p>You can of course do this in vanilla Javascript but like many things, it's easier with a library (like jQuery).</p> <p>For example, assuming:</p> <pre><code>&lt;input id="one" type="text"&gt; &lt;input id="two" type="text"&gt; </code></pre> <p>try:</p> <pre><code>document.getElementById("one").onkeydown = numbers_only; function numbers_only(evt) { return evt.keyCode &gt;= 48 &amp;&amp; evt.keyCode &lt;= 57; } </code></pre> <p>The first should only allow digits.</p>
<p>If you don't fancy writing it from scratch you can use the following jQuery plugin: <a href="http://www.itgroup.com.ph/alphanumeric/" rel="nofollow noreferrer">http://www.itgroup.com.ph/alphanumeric/</a> and then writing:</p> <pre><code>$('#id').numeric(); </code></pre> <p>And its been asked before here: <a href="http://stackoverflow.com/questions/895659/how-do-i-block-or-restrict-special-characters-from-input-fields-with-jquery">how do i block or restrict special characters from input fields with jquery?</a></p> <p><strong>ADDITIONAL</strong>: And make sure you validate on the server regardless!</p>
<p>the exact code that i was looking for is:</p> <pre><code> $('#id').bind('keypress', function (e) { return (e.which != 8 &amp;&amp; e.which != 0 &amp;&amp; (e.which &lt; 48 || e.which &gt; 57)) ? false : true; }); </code></pre>
<p>If you want to catch this on a submit event then you can use regex to do that in the submit event hanlder.</p> <p>But if you want to do this when the user interaction with the text box is going on then you will have to manipulate <strong><em>keydown</em></strong> and <strong><em>focus</em></strong> events. By wiring only keypress or keydown events will not prevent the user from entering other characters. He can also drag text into the text box. So focus event will prevent that from happening.</p>
<pre><code>$('#input-id').bind('keypress', function(e) { return !((e.which &lt; 48 || e.which &gt; 57) &amp;&amp; e.which != 8 &amp;&amp; e.which != 0); }) </code></pre>
