Write breakline in HTML from JavaScript
<p>I have some <a href="http://en.wikipedia.org/wiki/JavaScript" rel="nofollow noreferrer">JavaScript</a> code that does some stuff, but at some point I need to do this:</p> <pre><code>$('#mySpan').text('Hello\nWorld'); </code></pre> <p>Here is my HTML code:</p> <pre><code>&lt;span id="mySpan"&gt;&lt;/span&gt; </code></pre> <p>So I can finally get this:</p> <p>Hello<br> World</p> <p>The problem is that HTML is not detecting the breakline inserted from the JavaScript. I also tried this:</p> <pre><code>$('#mySpan').text('Hello&lt;BR/&gt;World'); </code></pre> <p>But it doesn't work either.</p> <p>How should I do it?</p>
<p>What about</p> <pre><code>$('#mySpan').html('Hello&lt;br/&gt;World'); </code></pre> <p>?</p> <p><a href="http://docs.jquery.com/Attributes/text" rel="nofollow noreferrer">text</a> only sets the text contents whereas <a href="http://docs.jquery.com/Attributes/html" rel="nofollow noreferrer">html</a> sets the HTML contents.</p>
<p>Use this</p> <pre><code>$('#mySpan').html('Hello&lt;br/&gt;World'); </code></pre>
<p>Why don't use try this initially on:</p> <pre><code>$('#mySpan').html('Hello&lt;br /&gt;World'); </code></pre>
<p>Maybe use <a href="http://docs.jquery.com/Manipulation/append" rel="nofollow noreferrer"><code>.append()</code></a> instead of <code>.text()</code>, if should break the line with your <code>&lt;br /&gt;</code> properly.</p>
