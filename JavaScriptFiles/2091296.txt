<p>That's browser based.</p> <p>As long as you implement <code>&lt;form&gt;</code> tag, and you type in something in the form, and in Google Chrome, it will prompt you that message.</p>
<p>You want to catch the <code>onbeforeunload</code> event of <code>window</code>. Then if you return a string, the browser will display your message in a prompt. Here's <a href="http://www.vijayjoshi.org/2009/07/07/faqjavascript-user-confirmation-on-close-button-of-browser/" rel="nofollow noreferrer">an article with sample code</a>.</p>
<p>In javascript, attach a function to <code>window.onbeforeunload</code>, like so:</p> <pre><code>window.unbeforeunload = function (e) { // return (prompt ('Are you sure you want to exit?')); // EDIT: Amended version below: var e = e || window.event; if (e) e.returnValue = 'Your exit prompt'; return 'Your exit prompt'; } </code></pre> <p>EDIT: As the comments below indicate, I had misunderstood the working of the event. It should now work as intended.</p> <p>Ref: <a href="https://developer.mozilla.org/en/DOM/window.onbeforeunload" rel="nofollow noreferrer">window.onbeforeunload (MDC)</a></p>
<p>You can use the <strong><a href="https://developer.mozilla.org/en/DOM/window.onbeforeunload" rel="nofollow noreferrer">window.onbeforeunload</a></strong> event to catch this. If there is any value inside the textarea then an alert message is shown.</p>
