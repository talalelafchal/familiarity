<p>A common way to do this is with a Javascript support framework like jquery. Using jQuery you can select elements in your html and change their attributes. For example if you wanted to set the background color for all your div elements, you would do:</p> <pre><code>$("div").css("background-color", "blue"); </code></pre> <p>Here the $ function lets you select DOM elements (html elements that are part of your loaded page), and the css function applies some css rule to them. </p>
<p>Basically as darren says you should change your <code>body</code> color with jQuery selector. A simple UI can be found here, for example <a href="http://www.eyecon.ro/colorpicker/" rel="nofollow noreferrer">jQuery ColorPicker</a>.</p>
