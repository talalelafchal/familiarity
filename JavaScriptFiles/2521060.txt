Regular Expression for finding font-style in html style
<p>Any idea how to find and replace the HTML font-size in style attribute? </p> <pre><code>eg. &lt;span style="font-size:12px"&gt;hello world&lt;/span&gt; </code></pre> <p>I would like to remove all font-size using javascript.</p> <p>Thank you</p>
<p>OH GOD NO NOT REGULAR EXPRESSIONS AAAAAAHHHHHH ;-) You should do this using the DOM manipulation methods of Javascript - that's what they're there for.</p> <pre><code>var theSpan = ...; theSpan.style.removeProperty("font-size"); </code></pre> <p>Here's <a href="http://www.java2s.com/Tutorial/JavaScript/0440__Style/Removeastylevaluebypropertyname.htm" rel="nofollow noreferrer">one reference</a>.</p>
<pre><code>$("span").each( function() { $(this).css("font-size", ""); }); </code></pre> <p>You can loop through the desired elements using jquery.</p>
<p>Here's an example of a Javascript function to find all spans and replace their font size.</p> <pre><code>function changeFontSize(newSize) { var spans = document.getElementsByTagName('span'); for(var i=0;i&lt;spans.length;i++) { spans[i].style.fontSize=newSize ; } } </code></pre> <p>The argument <code>newSize</code> is a string. If you set it to "50px", all spans will have a 50px font size. If you set it to "", it will have the same effect as removing the fontsize.</p>
