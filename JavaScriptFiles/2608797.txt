How to open new browser window for popup?
<p>Following Javscript can open popup:</p> <pre><code>window.open('http://www.google.com','mywindow','width=400,height=200'); </code></pre> <p>If call it two times like:</p> <pre><code>window.open('http://www.google.com','mywindow1','width=400,height=200'); window.open('http://www.google.com','mywindow2','width=400,height=200'); </code></pre> <p>The later will override previous only one popup. I want to two separated windows for popup at sametime with above code. even I set different reference name like mywindow1, mywindow2.</p> <p>How to fix this problem?</p> <p>More info: </p> <p>The way I call js in code-behind is:</p> <pre><code>string js = "&lt;script language=javascript&gt; window.open("myurl1","_blank","DialogWidth=700;DialogHeight=750;left=30;top=30;");&lt;/script&gt;"; Response.Write(sUrl); js = "&lt;script language=javascript&gt; window.open("myurl2","_blank","DialogWidth=700;DialogHeight=750;left=30;top=30;");&lt;/script&gt;"; Response.Write(sUrl); </code></pre> <p>It looks like later close the previous window. </p>

<p><a href="https://developer.mozilla.org/en/DOM/window.open" rel="nofollow noreferrer">It shouldn't</a>, you should get a new window because the window names are different ("mywindow1" vs. "mywindow2"). Do you really need to name the windows? If not, try <a href="http://www.w3.org/TR/html4/types.html#h-6.16" rel="nofollow noreferrer">"_blank"</a> instead. If you're not putting the digits on the end, that's your problem&nbsp;&mdash; using the same window name reuses the window (by design).</p>
