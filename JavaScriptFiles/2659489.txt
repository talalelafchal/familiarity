div slide effect
<p>i need to slide a div down and then up on an event, how can i do this in java script</p>
<p>I would recommend that you check out <a href="http://docs.jquery.com/UI/Effects/Slide" rel="nofollow noreferrer">jQuery/UI/Effects/Slide</a>.</p>
<p><strong>Update</strong> <strong><a href="http://jsbin.com/ideme" rel="nofollow noreferrer">I created this demo for you :)</a></strong></p> <p>You can do that <strong>easily</strong> with <strong><a href="http://api.jquery.com/slideToggle/" rel="nofollow noreferrer">slideToggle()</a></strong> function of JQuery:</p> <pre><code>&lt;div id="hello"&gt;Hello World&lt;/div&gt; &lt;a href="#" id="link"&gt;Toggle Div&lt;/a&gt; </code></pre> <p><strong>JQuery:</strong></p> <pre><code>$(function(){ $('#link').click(function(){ $('#hello').slideToggle(); return false; }) }); </code></pre>
<p>Highly recommend look at what JQuery or another javascript library does, (maybe even AJAX control toolkit for .NET, which has JS components) and examine the JS file, and replicate the feature yourself...</p> <p>I will say it probably isn't that easy to replicate yourself, meaning that the logic is pretty complex.</p> <p>HTH.</p>
