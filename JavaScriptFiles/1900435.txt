How to remove li from ul within a div
<p>I have a div, where in i do have a ul, where the ul has some li.</p> <p>i just want to find some text and remove the li from the ul.</p> <p>suppose eg: </p> <pre><code>&lt;div id="div1"&gt; &lt;ul&gt; &lt;li&gt;Hello&lt;/li&gt; &lt;!--i need to remove this.--&gt; &lt;/ul&gt; &lt;/div&gt; </code></pre>
<p>With jQuery, you can do this:</p> <pre><code>$("#div1 ul li:contains('Hello')").remove(); </code></pre>
<pre><code>var div = document.getElementById('div1'), ul = div.getElementsByTagName('ul')[0], li = ul.getElementsByTagName('li'), len = li.length; // Go backwards so that removing items has no effect while (len--) { if ( /Hello/.test(getText(li[len])) ) { ul.removeChild(li[len]); } } function getText(node) { var s = ''; node = node.firstChild; if ( node ) do { if ( node.nodeType === 3 ) { s += node.data; } if ( node.nodeType === 1 ) { s += getText(node); } } while ( node = node.nextSibling ); return s; } </code></pre>
