How to add and run a javascript for only on firefox not on others?
<p>How to add and run a javascript for only on firefox not on others?</p> <p>I've to add a javascript on a page and it should only work on firefox. not on IE</p>
<p>use <strong><a href="https://developer.mozilla.org/En/DOM/Window.navigator" rel="nofollow noreferrer">navigator</a></strong> object</p> <p><strong><a href="https://developer.mozilla.org/en/Navigator.appName" rel="nofollow noreferrer">navigator.appName</a></strong> </p> <blockquote> <p>Returns the official name of the browser.</p> </blockquote>
<p>Aside from using <code>navigator.appName</code>, you could use the <a href="http://www.quirksmode.org/js/detect.html" rel="nofollow noreferrer">browser detection script</a> from Quirksmode, then use an if statement to check if it's Firefox. This will nicely prepare browser information in an object for you, so you can pick out individual information such as version number, etc.</p>
<p>Never use browser detection, use <a href="http://www.quirksmode.org/js/support.html" rel="nofollow noreferrer"><strong>object detection</strong></a> (if your intent was using some objects/methods available only on some browsers.)</p>
