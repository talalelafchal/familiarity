Is contentDocument.location.reload compatible with IE 6
<p>I'm busy writing a application that refreshes a IFrame completely using javascript, now my issue is that I'm getting the following error in IE 6</p> <pre><code>'document.getElementById(...).contentDocument.location' is null or not an object </code></pre> <p>My code looks like this</p> <pre><code>document.getElementById('mobi').contentDocument.location.reload(true); </code></pre> <p>and mobi exist. mobi is the id of an iframe</p>
<p>use contentWindow for old IE :)</p> <p>document.getElementById('mobi').contentWindow.location.reload(true);</p> <p>(read more about <a href="http://msdn.microsoft.com/en-us/library/ms533692%28VS.85%29.aspx" rel="nofollow noreferrer">contentWindow</a>)</p>
