<p>We couldn't see the issue because of your formatting on the question, but after editing it to add code tags for you, I noticed that you are defining your function <strong>outside</strong> of your <code>&lt;script&gt;</code> tag. The following should work:</p> <pre><code>&lt;script&gt; function doit(){ makeWhizzyWig("edited", "all"); } doit(); &lt;/script&gt; </code></pre>
<p>Functions are just a way of grouping statements of code together in a way that makes them easier to call and reuse.</p> <p>In your example doit() only contains one statement and contains no parameters so the value of it is questionable, you may as well just call makeWhizzyWig directly. However, in most cases functions are a vital part of structuring and organising code.</p>
<p>The code you are showwing it should work. Or at least, the method doit() should get called when you click on the doit link. If you add an alert call at the beginning of the doit method you should see the popup window appear when you click on the link. Do you ?</p> <pre><code>function doit(){ alert('should be reached'); makeWhizzyWig("edited", "all"); } </code></pre> <p>If you do not see the alert window, what browser are you using ?</p> <p>Edit</p> <p>After seeing that it worked, I created a js file: </p> <pre><code>test.js function makeTest(param1, param2) { alert("first: " + param1 + " second: " + param2); } </code></pre> <p>and modified my html file:</p> <pre><code>&lt;head&gt; &lt;script type="text/javascript" src="test.js"&gt;&lt;/script&gt; &lt;/head&gt; &lt;body&gt; &lt;script&gt; function doit() { makeTest("edited", "all"); } &lt;/script&gt; &lt;a href=javascript:doit()&gt; doit &lt;/a&gt; &lt;/body&gt; </code></pre> <p>This shows that the problem is not in the way you are calling the doit function but that there is a problem inside the makeWhizzyWig() when it is called from a global context. Can't say much more since I do not have the source to makeWhizzyWig but you could try changing </p> <pre><code>&lt;a href=javascript:doit()&gt; doit &lt;/a&gt; </code></pre> <p>to </p> <pre><code>&lt;a href="#" onClick="doit()"&gt; doit &lt;/a&gt; </code></pre> <p>and see if it works </p>
<p>makeWhizzyWig("edited", "all"); creates a new Whizzywig using document.write. It has to be called inline before the page has loaded. Your problem is that you are not calling your doit() function until after the page has finished loading, so the document.write inside makeWhizzyWig won't work because the document is already complete.</p> <p>See <a href="http://unverse.net/WYSIWYG" rel="nofollow">http://unverse.net/WYSIWYG</a> and click "Dynamic Loading" for an example of how to dynamically create editors with Whizzywig.</p> <p>Or try <a href="http://unverse.net/Whizzywig-2011" rel="nofollow" title="Whizzywig 2011">Whizzywig 2011</a>, which solves this problem by not using document.write</p>
