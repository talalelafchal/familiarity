JavaScript, getting value of a td with id name
<p>Originally I was using input with an id and grabbing it's value with getElementById. Currently, I'm playing with <code>&lt;td&gt;</code>s. How can I grab the values of the <code>&lt;td&gt;</code>?</p> <p>Originally I used:</p> <pre><code>&lt;input id="hello"&gt; document.getElementById("hello").value; </code></pre> <p>Now I want to get value of <code>&lt;td&gt;</code>, but I don't know how;</p> <pre><code>&lt;td id="test"&gt;Chicken&lt;/td&gt; &lt;td&gt;Cow&lt;/td&gt; </code></pre> <p><strong>Edit:</strong> What's the difference between <code>textContent</code>, <code>innerHTML</code>, and <code>innerText</code>?</p>
<p>use the</p> <pre><code>innerHTML </code></pre> <p>property and access the <code>td</code> using <code>getElementById()</code> as always.</p>
<p>If by 'td value' you mean text inside of td, then:</p> <pre><code>document.getElementById('td-id').innerHTML </code></pre>
<p>Have you tried: <code>getElementbyId('ID_OF_ID').innerHTML</code>?</p>
<p>To get the text content</p> <pre><code>document.getElementById ( "tdid" ).innerText </code></pre> <p>or</p> <pre><code>document.getElementById ( "tdid" ).textContent var tdElem = document.getElementById ( "tdid" ); var tdText = tdElem.innerText | tdElem.textContent; </code></pre> <p>If you can use jQuery then you can use </p> <pre><code>$("#tdid").text(); </code></pre> <p>To get the HTML content</p> <pre><code>var tdElem = document.getElementById ( "tdid" ); var tdText = tdElem.innerHTML; </code></pre> <p>in jQuery</p> <pre><code>$("#tdid").html(); </code></pre>
<p>Again with getElementById, but instead .value, use .innerText</p> <pre><code>&lt;td id="test"&gt;Chicken&lt;/td&gt; document.getElementById('test').innerText; //the value of this will be 'Chicken' </code></pre>
<p>For input you must have used <code>value</code> to grab its text but for td, you should use <code>innerHTML</code> to get its html/value. Example:</p> <pre><code>alert(document.getElementById("td_id_here").innerHTML); </code></pre> <p>To get its text though, use:</p> <pre><code>alert(document.getElementById("td_id_here").innerText); </code></pre>
<p><code>.innerText</code> doesnt work in Firefox. </p> <p><code>.innerHTML</code> works in both the browsers.</p>
