How to find input text box is exist in iframe window?
<p>How to find the input text box is exit or no in my iframe html window using javascript</p> <p>Now i using below code</p> <pre><code>if(document.getElementById('iframe').contentWindow.document.getElementById("INPUTBOX").value) { alert("Exist") } else { alert("NOT Exist") } </code></pre> <p>But it gives me error "Object required"</p>
<p>You need to remove the <code>.value</code> from the end of your if statement.</p> <pre><code>if(document.getElementById('iframe') .contentWindow.document.getElementById("INPUTBOX")) { alert("Exist"); } else { alert("NOT Exist"); } </code></pre> <p>The reason you are getting the "Object required" error is the input box doesn't exist on the page and you're still trying to access the <code>value</code> property. Accessing a property on a non-existent object will throw that error.</p>
