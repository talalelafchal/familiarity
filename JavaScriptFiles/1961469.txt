javascript: get end key value?
<p>how do i get the end key value for a javascript array?</p> <p>i want to compare it with another value.</p> <p>EDIT: and how do you declare a value directly with a declaration of an array. i have to use 2 lines for it.</p> <p>var arrayname = new Array(); arrayname[] = 'value';</p> <p>could one do that in one line?</p>
<p>You can initialize an array at one go using:</p> <pre><code>var arrayname = [ 'value' ]; </code></pre> <p>Accessing the last value uses:</p> <pre><code>arrayname[arrayname.length-1]; </code></pre> <p>The previous assumes that the array has a length greater than zero. If the array can be empty you should check that length is greater than zero before trying to access the last element.</p>
<p>You can create arrays in one of many ways.</p> <pre><code>var foo = new Array(); foo[0] = 'a'; foo[1] = 'b'; foo[2] = 'c'; //is the same as: var foo = new Array(); foo.push('a'); foo.push('b'); foo.push('c'); //is the same as: var foo = []; foo.push('a'); foo[1] = 'b'; foo[foo.length] = 'c'; //is the same as: var foo = ['a','b','c']; //is the same as: var foo = 'a|b|c'.split('|'); </code></pre> <p>Of course, if you only want to build an array to pass the array to another function, you can build it anonymously on the fly:</p> <pre><code>doSomething('param1', ['a','b','c'], 'param3'); </code></pre>
