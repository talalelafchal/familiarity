how to access element whose id is variable
<p>I need to access elements in html file using javascript, their names are like <code>arr_1</code>, <code>arr_2</code>, <code>arr_3</code>, I wish to use a loop to dynamically create the id then to access them like below:</p> <pre><code>for(var i=0; i&lt; 10; i++) { var id = "arr_" + i; $document.getElementById('id').... } </code></pre> <p>But it doesn't work. I remember there is an function to allow me do that, anyone know what that is?</p>
<p>change</p> <pre><code>$document.getElementById('id') </code></pre> <p>to</p> <pre><code>$document.getElementById(id) </code></pre>
<pre><code>for (var i = 0; i &lt; 10; i++) { var obj = document.getElementById("arr_" + i); obj.style.border = "1px solid red"; } </code></pre>
<p>You don't need the dollar sign preceding <code>document</code>, and you should pass your <code>id</code> variable to the <code>getElementById</code> function, not a string containing <code>'id'</code>:</p> <pre><code>for(var i=0; i&lt; 10; i++) { var id = "arr_" + i; var element = document.getElementById(id); // work with element } </code></pre> <p>You might also want to check if <code>getElementById</code> actually found your element before manipulating it, to avoid run-time errors:</p> <pre><code>if (element) { element.style.color = '#ff0000'; } </code></pre>
