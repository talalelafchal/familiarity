<p>In Swift, appending strings is as easy as:</p> <pre><code>let stringA = "this is a string" let stringB = "this is also a string" let stringC = stringA + stringB </code></pre> <p>Or you can use string interpolation.</p> <pre><code>let stringC = "\(stringA) \(stringB)" </code></pre> <p>Notice there will now be whitespace between them.</p> <p>Note: I see the other answers are using <code>var</code> a lot. The strings aren't changing and therefore should be declared using <code>let</code>. I know this is a small exercise, but it's good to get into the habit of best practices. Especially because that's a big feature of Swift.</p>
<p>Its very simple: </p> <p><strong>For ObjC:</strong></p> <pre><code> NSString *string1 = @"This is"; NSString *string2 = @"Swift Language"; </code></pre> <p><strong>ForSwift:</strong></p> <pre><code> let string1 = "This is" let string2 = "Swift Language" </code></pre> <p><strong>For ObjC AppendString:</strong> </p> <pre><code> NSString *appendString=[NSString stringWithFormat:@"%@ %@",string1,string2]; </code></pre> <p><strong>For Swift AppendString:</strong> </p> <pre><code> var appendString1 = "\(string1) \(string2)" var appendString2 = string1+string2 </code></pre> <p><strong>Result:</strong> </p> <pre><code> print("APPEND STRING 1:\(appendString1)") print("APPEND STRING 2:\(appendString2)") </code></pre> <p><strong>Complete Code In Swift:</strong></p> <pre><code> let string1 = "This is" let string2 = "Swift Language" var appendString = "\(string1) \(string2)" var appendString1 = string1+string2 print("APPEND STRING1:\(appendString1)") print("APPEND STRING2:\(appendString2)") </code></pre>

<p>Add this extension somewhere:</p> <pre><code>extension String { mutating func addString(str: String) { self = self + str } } </code></pre> <p>Then you can call it like:</p> <pre><code>var str1 = "hi" var str2 = " my name is" str1.addString(str2) println(str1) //hi my name is </code></pre> <p>A lot of good Swift extensions like this are in my repo here, check them out: <a href="https://github.com/goktugyil/EZSwiftExtensions" rel="nofollow">https://github.com/goktugyil/EZSwiftExtensions</a></p>
