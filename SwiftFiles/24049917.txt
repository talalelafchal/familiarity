Passing one or more array as parameters into method in Swift
<p>I want to create a method that takes in one or more parameters but when I try the following code I get the following error message.</p> <pre><code>struct SomeStruct { static func method(arrays: AnyObject[]...) -&gt; AnyObject[] { return [] } } SomeStruct.method([1], [2]) Cannot convert the expression's type 'AnyObject[]' to type 'IntegerLiteralConvertible' </code></pre> <p>If I run the following code</p> <pre><code>SomeStruct.method(["1"], ["2"]) </code></pre> <p>I get the following error</p> <pre><code>Cannot convert the expression's type 'AnyObject[]' to type 'ExtendedGraphemeClusterLiteralConvertible' </code></pre> <p>I want to allow 0 or more array of items to be passed into this method of any element type including numbers. How can I do this?</p> <p><img src="https://i.stack.imgur.com/JX67Y.png" alt="enter image description here"></p>
<p>I was able to get it to work by using a generic instead of AnyObject.</p> <pre><code>struct SomeStruct { static func method&lt;T&gt;(arrays: T[]...) -&gt; T[] { return [] } } SomeStruct.method([1], [2]) </code></pre>
