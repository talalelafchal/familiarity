Iterate an array w/ explicit object type in Swift
<p>I have an array:</p> <pre><code>let individualScores = [75, 43, 103, 87, 12] </code></pre> <p>And I iterate like this:</p> <pre><code>for score in individualScores { } </code></pre> <p>However, is there a way to declare the object type explicitly? I think it would come in handy later w/ custom objects, or other reasons. Something like:</p> <pre><code>for Integer score in individualScores { } </code></pre>
<p>yes its possible</p> <pre><code>let individualScores:Int[] = [75, 43, 103, 87, 12] for score:Int in individualScores { } </code></pre>
<p>Yes. You can explicitly specify the type if you wish.</p> <pre><code>let individualScores = [75, 43, 103, 87, 12] for score: Int in individualScores { println(score) } </code></pre>
<p>Explicit type declarations follow an identifier declaration with a colon.</p> <pre><code>for score: Int in individualScores { // ... } let π: Double = 3.1415926535897932 var x: Int = 10 </code></pre> <p>You can read it "x is an Int". See <a href="https://developer.apple.com/library/prerelease/ios/documentation/Swift/Conceptual/Swift_Programming_Language/GuidedTour.html#//apple_ref/doc/uid/TP40014097-CH2-XID_1" rel="nofollow">A Swift Tour</a>.</p> <p>The loop variable (<code>score</code>) is explicitly and strongly typed whether you declare the type or not — it comes from the type of the array you're iterating through. Swift knows <code>individualScores</code> is an <code>Int[]</code>, short for an <code>Array&lt;Int&gt;</code> because you declared it with integer literals. See <a href="https://developer.apple.com/library/prerelease/ios/documentation/Swift/Conceptual/Swift_Programming_Language/Generics.html#//apple_ref/doc/uid/TP40014097-CH26-XID_234" rel="nofollow">Generics</a> for more about how that works.</p>
<p>When you type a variable, you do:</p> <pre><code>var score: Int </code></pre> <p>And you do the same in a loop:</p> <pre><code>for score: Int in individualScores { } </code></pre> <p>It seems to be pretty consistent in that regard.</p>
