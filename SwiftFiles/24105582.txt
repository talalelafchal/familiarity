how to animate stuff in swift (xcode 6 beta)
<p>i was wondering how do i animate things in apple's new language swift.</p> <p>in objective c i would use the following code to move an image from the top of the screen to the end :</p> <pre><code> [UIView animateWithDuration:1 delay:0 options:UIViewAnimationOptionCurveLinear animations:^{ UIImageView.center = CGPointMake(UIImageView.center.x , UIImageView.center.y + 200); } completion:^(BOOL finished) { [self move]; }]; </code></pre> <p>so the questoin is:</p> <ul> <li>how do i animate stuff in swift with the same effect that the code above(obj c)has, i would appreciate some explaination about the way you can do that aswell. </li> </ul>
