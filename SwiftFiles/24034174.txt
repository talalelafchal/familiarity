How do I concatenate strings in Swift?
<p>How to concatenate string in Swift.?</p> <p>In <code>Objective-C</code> we do like</p> <pre><code>NSString *string = @"Swift"; NSString *resultStr = [string stringByAppendingString:@" is new Programming Language"]; </code></pre> <p>or</p> <pre><code>NSString *resultStr=[NSString stringWithFormat:@"%@ is new Programming Language",string]; </code></pre> <p>But I want to do this in Swift-language.</p>
<pre><code>var language = "Swift" var resultStr = "\(language) is new Programming Language" </code></pre>
<pre><code>let the_string = "Swift" let resultString = "\(the_string) is a new Programming Language" </code></pre>
<p>This will work too:</p> <pre><code>var string = "swift" var resultStr = string + " is a new Programming Language" </code></pre>
<p>You can concatenate strings a number of ways:</p> <pre><code>let a = "Hello" let b = "World" let first = a + ", " + b let second = "\(a), \(b)" </code></pre> <p>You could also do:</p> <pre><code>var c = "Hello" c += ", World" </code></pre> <p>I'm sure there are more ways too.</p> <p><strong>Bit of description</strong></p> <p><code>let</code> creates a constant. (sort of like an <code>NSString</code>). You can't change its value once you have set it. You can still add it to other things and create new variables though.</p> <p><code>var</code> creates a variable. (sort of like <code>NSMutableString</code>) so you can change the value of it. But this has been answered several times on Stack Overflow, (see <em><a href="http://stackoverflow.com/q/24002092/61654">difference between let and var</a></em>).</p> <p><strong>Note</strong></p> <p>In reality <code>let</code> and <code>var</code> are <em>very different</em> from <code>NSString</code> and <code>NSMutableString</code> but it helps the analogy.</p>
<p>You can add a string in these ways:</p> <ul> <li><code>str += ""</code></li> <li><code>str = str + ""</code></li> <li><code>str = str + str2</code></li> <li><code>str = "" + ""</code></li> <li><code>str = "\(variable)"</code></li> <li><code>str = str + "\(variable)"</code></li> </ul> <p>I think I named them all.</p>

<p>XCode didn't accept optional strings added with a normal string. I wrote this extensions to solve that problem:</p> <pre><code>extension String { mutating func addString(str: String) { self = self + str } } </code></pre> <p>Then you can call it like:</p> <pre><code>var str1: String? var str1 = "hi" var str2 = " my name is" str1.addString(str2) println(str1) //hi my name is </code></pre> <p>However you could now also do something like this:</p> <pre><code>var str1: String? var str1 = "hi" var str2 = " my name is" str1! += str2 </code></pre>

<p>One can also use stringByAppendingFormat in Swift.</p> <pre><code>var finalString : NSString = NSString(string: "Hello") finalString = finalString.stringByAppendingFormat("%@", " World") print(finalString) //Output:- Hello World finalString = finalString.stringByAppendingFormat("%@", " Of People") print(finalString) //Output:- Hello World Of People </code></pre>
