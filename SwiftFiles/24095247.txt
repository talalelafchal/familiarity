Convert unichar to String?
<p>I'm getting a <code>unichar</code> type returned from <code>NSString</code>'s instance method <code>characterAtIndex(Int)</code> and I want to compare it with a Swift type <code>String</code>. Is there an easy way to do this?</p> <pre><code>var str = "#ffffff" var unichar = (str as NSString).characterAtIndex(0) var unicharString = // Perform magic var containsHash = unicharString == "#" // Should return `true` </code></pre> <p>Thanks</p>
<pre><code>var str:String = "#ffffff" var unichar = str[str.startIndex] var unicharString = "\(unichar)" var containsHash = unicharString == "#" </code></pre>
<p>Use UnicodeScalar to convert unichar into String or Character (element of String).</p> <pre><code>var str = "#ffffff" var unichar = (str as NSString).characterAtIndex(0) var unicharString = Character(UnicodeScalar(unichar)) var containsHash = unicharString == "#" </code></pre> <ul> <li>unichar is an alias of UInt16 (<code>typealias unichar = UInt16</code>).</li> <li>UnicodeScalar has <code>init(_ v: UInt16)</code>.</li> <li>Character (element of String) has <code>init(_ scalar: UnicodeScalar)</code>.</li> </ul> <p>Note: String also has <code>init(count: Int, repeatedValue c: UnicodeScalar)</code>, but this is not suitable for this case.</p>
