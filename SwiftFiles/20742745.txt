Navigation Controller Push View Controller
<p><strong>Question</strong></p> <p>How to navigate from one view controller to another simply using a button's touch up inside event?</p> <p><strong>More Info</strong></p> <p>What I tried in a sample project, in steps, was:</p> <ol> <li><p>Create the sample single view application.</p></li> <li><p>Add a new file -> Objective-C Class with XIB for user interface (ViewController2).</p></li> <li><p>Add a button into ViewController.xib and control click the button to ViewController.h to create the touch up inside event.</p></li> <li><p>Go to the newly made IBAction in ViewController.m and change it to this...</p> <pre><code>- (IBAction)GoToNext:(id)sender { ViewController2 *vc2 = [[ViewController2 alloc] initWithNibName:@"ViewController2" bundle:nil]; [[self navigationController] pushViewController:vc2 animated:YES]; } </code></pre></li> </ol> <p>The code runs without errors and I tested the button's functionality with NSLog. However it still doesn't navigate me to the second view controller. Any help would be appreciated.</p>
