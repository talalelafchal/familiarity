How to call designated superclass initializers in Swift
<p>In Objective-C there is this standard pattern in initializers to call the designated initializer of the superclass.</p> <p>For example:</p> <pre><code>- (instancetype)initWithFrame:(CGRect)frame { self = [super initWithFrame:frame]; if (self) { // Initialization code } return self; } </code></pre> <p>What is the standard way to do the same initialization in a Swift class?</p>
<p>Like this</p> <pre><code>init(frame: CGRect) { super.init(frame: frame) // Initialization code } </code></pre>
