How do I use whiteColor in Swift and SpriteKit
<p>I'm running Yosemite and Xcode 6. I created a new project using the "Game" option, SpriteKit and Swift.</p> <p>In the override for didMoveToView() in GameScene.swift I added the following line to try to change the back ground color:</p> <pre><code>self.backgroundColor = SKColor(whiteColor) </code></pre> <p>This compiles but won't run, stopping with the error "Use of unresolved identifier 'whiteColor'"</p> <p>The only import is:</p> <pre><code>import SpriteKit </code></pre> <p>Do I need another import to get whiteColor defined (which I assume is a constant in a header file somewhere ?</p>
<p><code>+[SKColor whiteColor]</code> is a class method, so as usual, it translate to <code>SKColor.whiteColor()</code></p> <pre><code>self.backgroundColor = SKColor.whiteColor() </code></pre>
<p>As you can see in the documentation, SKColor is just a wrapper.</p> <pre><code>#if TARGET_OS_IPHONE #define SKColor UIColor #else #define SKColor NSColor #endif </code></pre> <p>That's why we use the usual way as Bryan describes.</p>
