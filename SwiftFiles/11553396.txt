how to add an action on UITextField return key?
<p>I have a button and text textfield in my view. when i click on the textfield a keyboard appears and i can write on the textfield and i also able to dismiss the keyboard by clicking on the button by adding:</p> <pre><code>[self.inputText resignFirstResponder]; </code></pre> <p>Now I want to enable return key of keyboard. when i will press on the keyboard keyboard will disappear and something will happen. How can I do this?</p>
<p>Ensure "self" subscribes to <code>UITextFieldDelegate</code> and initialise inputText with:</p> <pre><code>self.inputText.delegate = self; </code></pre> <p>Add the following method to "self":</p> <pre><code>- (BOOL)textFieldShouldReturn:(UITextField *)textField { if (textField == self.inputText) { [textField resignFirstResponder]; return NO; } return YES; } </code></pre> <p>Or in Swift:</p> <pre><code>func textFieldShouldReturn(textField: UITextField) -&gt; Bool { if textField == inputText { textField.resignFirstResponder() return false } return true } </code></pre>
