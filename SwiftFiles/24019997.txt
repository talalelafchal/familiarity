Iterate through alphabet in Swift
<p>in Obj-C it was possible to iterate through alphabet with:</p> <pre><code>for (char x='A'; x&lt;='Z'; x++) { </code></pre> <p>In Swift this isn't possible. Any idea how I could do this?</p>
<p>In Swift, you can iterate chars on a string like this:</p> <p><strong>Swift 2</strong></p> <pre><code>for char in "abcdefghijklmnopqrstuvwxyz".characters { println(char) } </code></pre> <p><strong>Swift 1.2</strong></p> <pre><code>for char in "abcdefghijklmnopqrstuvwxyz" { println(char) } </code></pre> <p>There might be a better way though.</p>
<p>It's slightly cumbersome, but the following works:</p> <pre><code> for val in UnicodeScalar("a").value...UnicodeScalar("z").value { let x = UnicodeScalar(val); println(x) } </code></pre> <p>I suspect that the problem here is that the meaning of "a"..."z" could potentially be different for different string encodings.</p> <p>Also cumbersome, but without the extra intermediate variable:</p> <pre><code> for letter in map(UnicodeScalar("a").value...UnicodeScalar("z").value, {(val: UInt32) -&gt; UnicodeScalar in return UnicodeScalar(val); }) { println(letter) } </code></pre>
<p>Did you try this one ?</p> <pre><code>for var myChar:CChar = 65 ; myChar &lt;= 90 ; ++myChar { let x:String = String(format: "%c", myChar) println(x) } </code></pre> <p>If you already know the character code</p>
<pre><code>for i in 97...122{println(UnicodeScalar(i))} </code></pre>
