How to dismiss Keypad on iOS after UIButton is pressed using Swift
<p>I want to dismiss the Keypad on iOS after a UIButton is pressed, I am not sure what to put into the function to dismiss the keypad.</p> <pre><code> @IBAction func calculateWagesButton(sender : UIButton) { //Magic Code Here....?? } </code></pre>
<p>There's nothing swift specific here.</p> <p>Just grab the first responder and make it resign. For instance, if it's a textfield:</p> <pre><code>self.textfield.resignFirstResponder() </code></pre>
<p>1st Step :- implement <code>UITextFieldDelegate</code> like:</p> <pre><code>class yourViewController: UIViewController, UITextFieldDelegate { </code></pre> <p>2nd Step :- In viewDidLoad function wrtie following line:</p> <pre><code>self.textfield.delegate = self </code></pre> <p>3rd Step :- use following method:</p> <pre><code>func textFieldShouldReturn(textField: UITextField!) -&gt; Bool { textField.resignFirstResponder() return true; } </code></pre> <p>I hope, these code helps you . . .</p>
