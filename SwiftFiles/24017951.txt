Appearance Proxy in Swift (iOS)
<p>Has anyone tried using the appearance proxy in swift yet?</p> <p>This syntax doesn't work, has anyone figured out how to set title text attributes on controls like segmentedControl or UITabBar? I think I am close</p> <pre><code>segmentedControl.titleTextAttributesForState(UIControlState.Normal) = NSDictionary(objects: [UIFont(name: fontFamilyRegular, size: 16.0)], forKeys: [NSFontAttributeName]) </code></pre>
<p>This should do it:</p> <pre><code>segmentedControl.setTitleTextAttributes([ NSFontAttributeName: UIFont(name: "Helvetica", size: 16.0)!, NSForegroundColorAttributeName: UIColor.blueColor() ], forState: UIControlState.Normal) </code></pre>
<p>For XCode 6.1, try this:</p> <pre><code>UITabBarItem.appearance().setTitleTextAttributes(NSDictionary(object: UIFont(name: "Helvetica", size: 16.0)!, forKey: NSFontAttributeName), forState: UIControlState.Normal) </code></pre>
<p>Make sure you unwrap the the font (!)</p> <pre><code>let font = UIFont(name: "HelveticaNeue-Light", size:15.0)! UIBarButtonItem.appearance().setTitleTextAttributes([NSFontAttributeName:font,NSForegroundColorAttributeName:UIColor.redColor()], forState: UIControlState.Normal) </code></pre>
