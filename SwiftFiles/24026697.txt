Swift: ViewController Error in webView project
<p>Writing a simple app in Xcode 6 in Swift.</p> <pre><code>class htmlViewController : UIViewController { @IBOutlet var webbb: UIWebView } class ViewController: UIViewController { override func viewDidLoad() { let url = NSURL(string: "http://apple.com") let reqq = NSURLRequest(URL: url) self.webbb.loadView(reqq) super.viewDidLoad() } } </code></pre> <p>Have some error on line "self.webbb.loadView(reqq)":</p> <blockquote> <p>ViewController does not have a member named WebView.</p> </blockquote> <p>Any ideas?</p> <p>P.S. Sorry for noob question! :-)</p>
<p>You defined <code>webbb</code> in a class called <code>htmlViewController</code> but you're adding functionality to another subclass called <code>ViewController</code>.</p>
<p>The problem you are having is that the <code>ViewController</code> class has no access to the <code>webbb</code> variable. There are a few solutions for this problem.</p> <p><strong>Solution 1:</strong></p> <p>When you define the <code>ViewController</code> class make the Superclass <code>htmlViewController</code>. That would look like this:</p> <pre><code>class ViewController: htmlViewController{...} </code></pre> <p>This means that the <code>ViewController</code> class will inherit from the <code>htmlViewController</code> class. This allows you to access variables and functions from the <code>htmlViewController</code> class. You can read more about inheritance and swift in <a href="https://developer.apple.com/library/prerelease/ios/documentation/Swift/Conceptual/Swift_Programming_Language/Inheritance.html#//apple_ref/doc/uid/TP40014097-CH17-XID_251" rel="nofollow">this document</a>.</p> <hr> <p><strong>Solution 2:</strong></p> <p>Why not bring the <code>webbb</code> variable into the <code>ViewController</code> class? I would recommend doing this if you do not plan on using the <code>htmlViewController</code> class anywhere else. Here is an example</p> <pre><code>class ViewController: htmlViewController{ @IBOutlet var webbb: UIWebView //Do everything else. } </code></pre>
<p>for load a web page in a UIWebView just use this code :</p> <pre><code>@IBOutlet var web1 : UIWebView override func viewDidLoad() { super.viewDidLoad() var url = NSURL(string: "http://www.google.com") var request = NSURLRequest(URL: url) web1.loadRequest(request) } </code></pre> <p>and if you want to load a page from main bundle use this code :</p> <pre><code>@IBOutlet var web1 : UIWebView override func viewDidLoad() { super.viewDidLoad() var url = NSURL(fileURLWithPath: NSBundle.mainBundle().pathForResource("pageName", ofType: "html")) //var url = NSURL(string: "http://www.google.com") var request = NSURLRequest(URL: url) web1.loadRequest(request) } </code></pre>
<p>You also need to add a ! after UIWebView when you are defining it.</p>
