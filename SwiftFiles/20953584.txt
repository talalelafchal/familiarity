How to open &amp; view view .doc, .docx, .rtf, .ppt, .pptx, .xlsx, .xls file in iphone using UIWebview in Document directory?
<p>Can we open &amp; view .doc, .docx, .rtf, .ppt, .pptx, .xls, .xlsx file in iphone using UIWebview?</p> <p>I am using Document directory to show the file.</p> <p>.doc file is working fine here… but rest of the files' extensions are not working..</p> <p>if any one has implemented this then please help me...</p> <p>any code snippet or any web link help...</p> <p>thanks in advance...</p>
<p>You may want to look at the QuickLook Framework for iOS: <a href="https://developer.apple.com/library/ios/documentation/FileManagement/Conceptual/DocumentInteraction_TopicsForIOS/Articles/UsingtheQuickLookFramework.html">https://developer.apple.com/library/ios/documentation/FileManagement/Conceptual/DocumentInteraction_TopicsForIOS/Articles/UsingtheQuickLookFramework.html</a></p> <p>It supports the following formats:</p> <ul> <li>iWork documents</li> <li>Microsoft Office documents (Office ‘97 and newer)</li> <li>Rich Text Format (RTF) documents</li> <li>PDF files</li> <li>Images</li> <li>Text files whose uniform type identifier (UTI) conforms to the public.text type (see * Uniform Type Identifiers Reference)</li> <li>Comma-separated value (csv) files</li> </ul> <p>More specifically you may want to look at the <code>QLPreviewController</code>: <a href="https://developer.apple.com/library/ios/documentation/NetworkingInternet/Reference/QLPreviewController_Class/Reference/Reference.html#//apple_ref/occ/cl/QLPreviewController">https://developer.apple.com/library/ios/documentation/NetworkingInternet/Reference/QLPreviewController_Class/Reference/Reference.html#//apple_ref/occ/cl/QLPreviewController</a></p>
<p>Try this:</p> <pre><code> [self.m_webView loadRequest:[NSMutableURLRequest requestWithURL:[NSURL fileURLWithPath:_m_filePath]cachePolicy:NSURLCacheStorageAllowedInMemoryOnly timeoutInterval:3.0]]; </code></pre> <p>You can change timeoutInterval and cachePolicy, in sometime you have to wait more time to loading. </p>
<p>with <strong>swift 2.2</strong></p> <p><strong>For showing PDF and DOC</strong> file I have use following code snip</p> <pre><code>//Document file url var docUrl = NSURL(string: "https://www.google.co.in/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=1&amp;ved=0ahUKEwjjwPSnoKfNAhXFRo8KHf6ACGYQFggbMAA&amp; url=http%3A%2F%2Fwww.snee.com%2Fxml%2Fxslt%2Fsample.doc&amp;usg=AFQjCNGG4FxPqcT8RXiIRHcLTu0yYDErdQ&amp;sig2=ejeAlBgIZG5B6W-tS1VrQA&amp;bvm=bv.124272578,d.c2I&amp;cad=rja") let req = NSURLRequest(URL: docUrl!) webView.delegate = self //here is the sole part webView.scalesPageToFit = true webView.contentMode = .ScaleAspectFit webView.loadRequest(req) </code></pre> <p><strong>Note</strong>: make sure on backed side (for api) we have to define content-type as document in headers.</p>
