How to insert a sublayer in swift?
<p>I am creating a view and adding a gradient layer to it.</p> <p>I have this:</p> <pre><code>import UIKit import QuartzCore let rect : CGRect = CGRectMake(0,0,320,100) var vista : UIView = init(frame: rect) let gradient : CAGradientLayer = CAGradientLayer() gradient.frame = vista.bounds let cor1 = UIColor.blackColor().CGColor let cor2 = UIColor.whiteColor().CGColor let arrayColors = [cor1, cor2] gradient.colors = arrayColors </code></pre> <p>now I have to do this</p> <pre><code>[view.layer insertSublayer:gradient atIndex:0]; </code></pre> <p>How do I do that in swift?</p>
<p>You can use the following snippet</p> <pre><code>view.layer.insertSublayer(yourLayer, atIndex: yourIndex) </code></pre>
<p>Have you tried this?</p> <pre><code>view.layer.insertSublayer(gradient, atIndex:0) </code></pre> <p>Note that whenever you're editing Swift code, you can command-click on a symbol from the frameworks to see a Swift-ified version of that API's header file. Also, the documentation <a href="https://developer.apple.com/library/prerelease/ios/documentation/GraphicsImaging/Reference/CALayer_class/#//apple_ref/occ/instm/CALayer/insertSublayer%3aatIndex%3a" rel="nofollow">online</a> and in Xcode shows Swift syntax for all the APIs it covers.</p>
