Swift Playground: Indexing a List within a class gives an error
<p>I tried the following in Swift Playground </p> <pre><code>class C { init(test: Integer) { let simpleList:String[] = ["A","B","C"] simpleList[test] println(simpleList[test]) } } </code></pre> <p>I get an error <strong>"Could not find an overload for subscript that accepts the supplied arguments"</strong>. This pops up in multiple places when I try to index a list. Kindly help</p>
<p>There are usually two causes for this error:</p> <p>In your case, you need to change <code>test</code> to the <code>Int</code> type like this: <code>(test: Int)</code>. </p> <p>Another case that throws a similar error is due to <code>simpleList</code> being of type <code>Any</code> or <code>AnyObject</code>.</p> <p>In cases like those you need to cast it to type <code>String[]</code> or another Swift array type before indexing</p>
