Could not find an overload for '^' that accepts the supplied arguments
<pre><code>func squareArea(side:Double) -&gt; Double { return side ^ 2 } </code></pre> <p>I get the error:</p> <pre><code>could not find an overload for '^' that accepts the supplied arguments </code></pre> <p>I've also tried</p> <pre><code>func squareArea(side:Double) -&gt; Double { return Double(side ^ 2) } </code></pre> <p>and</p> <pre><code>func squareArea(side:Double) -&gt; Double { return side ^ Double(2) } </code></pre> <p>but</p> <pre><code>func squareArea(side:Double) -&gt; Double { return side * side } </code></pre> <p>works fine. What is the correct syntax?</p>
<p>^ Is the XOR operator not the "power" operator. <a href="https://developer.apple.com/library/prerelease/ios/documentation/Swift/Conceptual/Swift_Programming_Language/AdvancedOperators.html" rel="nofollow">See here and search "XOR".</a></p> <blockquote> <p>The bitwise XOR operator, or “exclusive OR operator” (^), compares the bits of two numbers. The operator returns a new number whose bits are set to 1 where the input bits are different and are set to 0 where the input bits are the same:</p> </blockquote> <pre><code>let firstBits: UInt8 = 0b00010100 let otherBits: UInt8 = 0b00000101 let outputBits = firstBits ^ otherBits </code></pre>
<p>There's a function for that:</p> <pre><code>func squareArea(side:Double) -&gt; Double { return pow(side, 2) } </code></pre>
