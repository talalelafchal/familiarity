Anonymous closure argument not contained in a closure
<p>Why this code doesn't work?</p> <pre><code>func function (param1 : Int, param2 : Int) -&gt; Int { return $0 + $1 } </code></pre> <p>Error: Anonymous closure argument not contained in a closure</p>
<p>It seems you can only access parameters by number inside anonymous closures, not functions.</p> <p>For example:</p> <pre><code>var sevenMultiplyedByThree: Int = { return $0 * 3 }(7) </code></pre> <p>Also, this is just for anonymous parameters, so the following code will NOT work:</p> <pre><code>var sevenMultiplyedByThree: Int = { (namedParameter : Int) -&gt; Int in return $0 * 3 }(7) </code></pre>
<p>I got it, accessing parameters by their index is used when you don't have the parameters of the closure named:</p> <pre><code>var result = {$0 + 10}(5) result </code></pre> <p>result now is 15</p> <p>in contrast of</p> <pre><code>var result2 = { (param: Int) -&gt; Int in return param + 10 }(5) result2 </code></pre> <p>it's not possible to use <strong>$0</strong> instead of <strong>param</strong> because param is a <strong>named</strong> parameter.</p>
