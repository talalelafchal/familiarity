UITableView, Separator color where to set?
<p>I have added a <code>UITableView</code> in IB and set the "delegate" and "datasource" and all is working well. What I wanted to do next was change the separator color, but the only way I could find to do this was to add the method to one of the delegate callbacks, is there a better place I should put this?</p> <p>I don't have this at the moment but I was thinking that maybe I need to add an "iVar" from my controller that I can link to the <code>UITableView</code> in IB and then set separator color in the <code>viewDidload</code>?</p> <pre><code>-(CGFloat)tableView:(UITableView *)tableView heightForRowAtIndexPath:(NSIndexPath *)indexPath { [tableView setSeparatorColor:[UIColor blackColor]]; return 65; } </code></pre>
<pre><code>- (void)viewDidLoad { [self.tableView setSeparatorColor:[UIColor myColor]]; } </code></pre> <p>I hope that helps - you'll need the <code>self.</code> to access it, remember.</p>
<p>Now you should be able to do it directly in the IB.</p> <p>Not sure though, if this was available when the question was posted originally.</p> <p><img src="https://i.stack.imgur.com/mfk7u.png" alt="enter image description here"></p>
<p><strong>Swift</strong> version:</p> <pre><code>override func viewDidLoad() { super.viewDidLoad() // Assign your color to this property, for example here we assign the red color. self.tableView.separatorColor = UIColor.redColor() } </code></pre>
<p>If you just want to set the same color to every separator and it is opaque you can use:</p> <pre><code> self.tableView.separatorColor = UIColor.redColor() </code></pre> <p>If you want to use different colors for the separators or clear the separator color or use a color with alpha.</p> <p>BE CAREFUL: You have to know that there is a backgroundView in the separator that has a default color.</p> <p>To change it you can use this functions:</p> <pre><code> func tableView(tableView: UITableView, willDisplayHeaderView view: UIView, forSection section: Int) { if(view.isKindOfClass(UITableViewHeaderFooterView)){ var headerView = view as! UITableViewHeaderFooterView; headerView.backgroundView?.backgroundColor = myColor //Other colors you can change here // headerView.backgroundColor = myColor // headerView.contentView.backgroundColor = myColor } } func tableView(tableView: UITableView, willDisplayFooterView view: UIView, forSection section: Int) { if(view.isKindOfClass(UITableViewHeaderFooterView)){ var footerView = view as! UITableViewHeaderFooterView; footerView.backgroundView?.backgroundColor = myColor //Other colors you can change here //footerView.backgroundColor = myColor //footerView.contentView.backgroundColor = myColor } } </code></pre> <p>Hope it helps!</p>
<p>Try <strong><em>+ (instancetype)appearance</em></strong> of UITableView:</p> <p><em>Objective-C:</em></p> <pre><code>[[UITableView appearance] setSeparatorColor:[UIColor blackColor]]; // set your desired colour in place of "[UIColor blackColor]" </code></pre> <p><em>Swift 3.0:</em></p> <pre><code>UITableView.appearance().separatorColor = UIColor.black // set your desired colour in place of "UIColor.black" </code></pre> <p><strong>Note:</strong> Change will reflect to all tables used in application.</p>
