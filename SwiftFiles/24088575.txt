Instanciate FBLoginView class in swift
<p>I try to implement Facebook SDK in an IOS Swift project, but I can't include the SDK. Moreover, I don't know how to translate this Objectiv-c snippet to Swift :</p> <pre><code>[FBLoginView class]; </code></pre> <p>Someone can help me ?</p>
<p>The easiest way I found is to add a new "Objective-C File" into your project.</p> <p>It should ask you "<em>Would you like to configure an Objective-C bridging header?</em>" Choose <strong>Yes</strong>.</p> <p>Two files will be added to your project, the Objective-C file and "<strong>(project name)-Bridging-Header.h</strong>"</p> <p>Delete the empty Objective-C file you added and open <strong>(project name)-Bridging-Header.h</strong></p> <p>Add your import for the Facebook SDK into the file (using the Objective-C style import), example:</p> <pre><code>// // Use this file to import your target's public headers that you would like to expose to Swift. // #import &lt;FacebookSDK/FacebookSDK.h&gt; </code></pre> <p>Save the header file and you can now access Facebook SDK in your swift code. <strong>You do not</strong> need to add "import FacebookSDK" into your swift files. Example:</p> <pre><code>import UIKit class LoginViewController: UIViewController { var fbl: FBLoginView = FBLoginView() // etc... </code></pre>
<p>What are you trying to do with the class?</p> <p>In swift you can reference the class with <code>.self</code> like so:</p> <pre><code>FBLoginView.self </code></pre>
<p>Believe with the latest updates from Facebook, the imports for the Header File are:</p> <pre><code>#import &lt;FBSDKCoreKit/FBSDKCoreKit.h&gt; #import &lt;FBSDKLoginKit/FBSDKLoginKit.h&gt; </code></pre>
