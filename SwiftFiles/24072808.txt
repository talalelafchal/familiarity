<p>At first, I'm very very bad at Obj-C (don't like the syntax), but now with Swift I love iPhone developing :D</p> <p>Here is my problem, I try to work with a Obj-C Library (xmppframework), but when I try to initialize the XMPPRoster the required Method isn't available...</p> <h3>Objective-C usage:</h3> <pre><code>xmppRosterStorage = [[XMPPRosterCoreDataStorage alloc] initWithInMemoryStore]; xmppRoster = [[XMPPRoster alloc] initWithRosterStorage:xmppRosterStorage]; </code></pre> <p>But within my Swift Class the Method "initWithRosterStorage" isn't available :(</p> <h3>Swift:</h3> <pre><code>var jabberRosterStorage: XMPPCoreDataStorage = XMPPCoreDataStorage() var jabberRoster:XMPPRoster = XMPPRoster(....) </code></pre> <p>XMPPRoster.m:</p> <pre><code>- (id)initWithRosterStorage:(id &lt;XMPPRosterStorage&gt;)storage { return [self initWithRosterStorage:storage dispatchQueue:NULL]; } </code></pre> <p>Maybe Swift has problem with (id) because in Swift I think it's now AnyObject</p>
<p>Try initializing it like this in Swift:</p> <pre><code>var jabberRosterStorage: XMPPCoreDataStorage, XMPPRosterStorage = XMPPCoreDataStorage() var jabberRoster: XMPPRoster = XMPPRoster(rosterStorage: jabberRosterStorage) </code></pre> <p>Swift will convert the <code>[[Class alloc] initWithVar:variable]</code> to <code>Class(var: variable)</code>.</p> <p><a href="https://developer.apple.com/library/prerelease/ios/documentation/Swift/Conceptual/BuildingCocoaApps/InteractingWithObjective-CAPIs.html#//apple_ref/doc/uid/TP40014216-CH4-XID_26" rel="nofollow">Reference</a></p>
