converting timestamp to nsdate format
<p>I want to convert my timestamp value <code>1308031456</code> to <code>NSDate</code> format (which yields the value <code>Tue, 14 Jun 2011 06:04:16 GMT</code> in online web conversion). How would I convert between the two programmatically?</p>
<p>You can use the use the method <code>dateWithTimeIntervalSince1970:</code> to convert the timestamp you've which is the epoch time.</p> <pre><code>NSDate * myDate = [NSDate dateWithTimeIntervalSince1970:1308031456]; </code></pre>
<p><strong>OBJ - C:</strong> Use <a href="http://developer.apple.com/library/mac/documentation/Cocoa/Reference/Foundation/Classes/NSDate_Class/Reference/Reference.html#//apple_ref/occ/clm/NSDate/dateWithTimeIntervalSince1970:">dateWithTimeIntervalSince1970:</a>,</p> <pre><code>NSDate *date = [NSDate dateWithTimeIntervalSince1970:timeStamp]; </code></pre> <p><strong>SWIFT:</strong> Use <a href="https://developer.apple.com/library/mac/documentation/Cocoa/Reference/Foundation/Classes/NSDate_Class/#//apple_ref/occ/instm/NSDate/initWithTimeIntervalSince1970:">init(timeIntervalSince1970:)</a></p> <pre><code>let date = NSDate(timeIntervalSince1970:timeStamp) </code></pre>
