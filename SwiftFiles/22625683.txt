Show scrollbar in UITableView before the table is dragged down. In a Way, the UIScrollBar should always be visible
<p>In a I am using several <strong>UITableView</strong> in my app. Some of them contain several rows, while some have two three only. What I need is: if the table contains several rows, then the <strong>scrollbar should be visible before the user even drags the table down</strong>. Or in another way, is it possible to make scrollbar visible always??</p>
<p>You can flash them but there isn't an option to make them always visible:</p> <pre><code>[ScrollView flashScrollIndicators]; </code></pre> <blockquote> <p>UITableView is a subclass of UIScrollView,So you can use it directly with it </p> </blockquote> <pre><code> - (void)viewDidAppear:(BOOL)animated { [super viewDidAppear:animated]; [self.tableView performSelector:@selector(flashScrollIndicators) withObject:nil afterDelay:0]; } </code></pre> <p>Check out for more info : <a href="http://stackoverflow.com/questions/1888647/uiscrollview-showing-the-scroll-bar">UIScrollView - showing the scroll bar</a></p> <p><strong>Swift:</strong></p> <pre><code>override func viewDidAppear(animated: Bool) { super.viewDidAppear(animated) self.tableView.flashScrollIndicators() } </code></pre>
<p><strong>Swift 3.0</strong></p> <p><strong>1) Timer</strong></p> <pre><code>var timerForShowScrollIndicator: Timer? </code></pre> <p><strong>2) Methods</strong></p> <pre><code>/// Show always scroll indicator in table view func showScrollIndicatorsInContacts() { UIView.animate(withDuration: 0.001) { self.tableView.flashScrollIndicators() } } /// Start timer for always show scroll indicator in table view func startTimerForShowScrollIndicator() { self.timerForShowScrollIndicator = Timer.scheduledTimer(timeInterval: 0.3, target: self, selector: #selector(self.showScrollIndicatorsInContacts), userInfo: nil, repeats: true) } /// Stop timer for always show scroll indicator in table view func stopTimerForShowScrollIndicator() { self.timerForShowScrollIndicator?.invalidate() self.timerForShowScrollIndicator = nil } </code></pre> <p><strong>3) Use</strong></p> <p><strong>startTimerForShowScrollIndicator</strong> in <strong>viewDidAppear</strong></p> <p><strong>stopTimerForShowScrollIndicator</strong> in <strong>viewDidDisappear</strong></p>
