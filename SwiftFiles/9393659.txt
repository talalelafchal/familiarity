UILabel Truncation
<p>I have text like this: "My text is blabla blabla, lala lala ".</p> <p>I would like to have the text in my <code>UILabel</code> like this: "My text is ...lala". </p> <p>How can I configure my <code>UILabel</code> to display the text to have the ellipsis in the middle?</p>
<p>The word you are looking for is "ellipsis" ;)</p> <p>Set the following properties:</p> <pre><code>label.adjustsFontSizeToFitWidth = NO; label.lineBreakMode = UILineBreakModeMiddleTruncation; </code></pre> <p>You can also set these properties in interface builder.</p> <p>Example stolen from here: <a href="http://stackoverflow.com/questions/7725795/getting-uilabel-to-produce-an-ellipsis-rather-then-shrinking-the-font">Getting UILabel to produce an ellipsis rather then shrinking the font</a></p> <p><strong>UPDATE:</strong></p> <p>This was deprecated in iOS 6. The current solution would be the slightly modified:</p> <pre><code>label.adjustsFontSizeToFitWidth = NO; label.lineBreakMode = NSLineBreakByTruncatingMiddle </code></pre>
<p><strong>@DerekTomes</strong> answer in Swift 2.x:</p> <pre><code>label.adjustsFontSizeToFitWidth = false label.lineBreakMode = .ByTruncatingMiddle </code></pre>
