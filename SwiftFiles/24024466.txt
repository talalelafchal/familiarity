How do I make a class conform to a delegate in swift?
<p>I need to make a class conform to a delegate in Swift. How would I do so?</p>
<pre><code>class YourClass: SuperClassIfAny, FirstProtocol, SecondProtocol { } </code></pre> <p>Note, though, that some protocols <strong>require</strong> you to implement delegate methods. For instance, <code>UITableViewDataSource</code> <strong>requires</strong> you to implement </p> <pre><code>func tableView(tableView: UITableView!, numberOfRowsInSection section: Int) -&gt; Int </code></pre> <p>and </p> <pre><code>func tableView(tableView: UITableView!, cellForRowAtIndexPath indexPath: NSIndexPath!) -&gt; UITableViewCell! </code></pre> <p>If those are not implemented by the class conforming to the protocol, Xcode will give you a compile error (always check the protocol declaration, Cmd + Click will show you what methods you <strong>must</strong> implement).</p>
<p>Xcode 6 beta 7 changed the protocol slightly for UITableViewDataSource to match the following syntax on the two required implementations:</p> <p>6b7 : <a href="https://developer.apple.com/library/prerelease/ios/documentation/UIKit/Reference/UITableViewDataSource_Protocol/index.html" rel="nofollow">UITableViewDataSource</a></p> <pre><code>func tableView(tableView: UITableView, numberOfRowsInSection section: Int) -&gt; Int func tableView(tableView: UITableView, cellForRowAtIndexPath indexPath: NSIndexPath) -&gt; UITableViewCell! </code></pre> <p>compared to 6b6</p> <pre><code>func tableView(tableView: UITableView!, numberOfRowsInSection section: Int) -&gt; Int func tableView(tableView: UITableView!, cellForRowAtIndexPath indexPath: NSIndexPath!) -&gt; UITableViewCell </code></pre> <p>Of key difference is that UITableView, NSIndexPath and UITableViewCell are no longer '<a href="http://stackoverflow.com/questions/24018327/what-does-an-exclamation-mark-mean-in-the-swift-language">Implicitly Unwrapped Optionals</a>' </p>
