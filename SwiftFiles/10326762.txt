Objective-C Simplest way to create comma separated string from an array of objects
<p>So I have a nsmutablearray with a bunch of objects in it. I want to create a comma separated string of the id value of each object.</p>
<p>Use the <code>NSArray</code> instance method <code>componentsJoinedByString:</code>.</p> <p><code>- (NSString *)componentsJoinedByString:(NSString *)separator</code>.</p> <p><strong>Example:</strong></p> <p><code>NSString *joinedComponents = [array componentsJoinedByString:@","];</code></p>
<p>If you're searching for the same solution in Swift, you can use this:</p> <pre><code>var array:Array&lt;String&gt; = ["string1", "string2", "string3"] var commaSeperatedString = ", ".join(array) // Results in string1, string2, string3 </code></pre> <p>To make sure your array doesn't contains nil values, you can use a filter:</p> <pre><code>array = array.filter { (stringValue) -&gt; Bool in return stringValue != nil &amp;&amp; stringValue != "" } </code></pre>
<p><strong>Create String from Array:</strong></p> <pre><code>-(NSString *)convertToCommaSeparatedFromArray:(NSArray*)array{ return [array componentsJoinedByString:@","]; } </code></pre> <p><strong>Create Array from String:</strong></p> <pre><code>-(NSArray *)convertToArrayFromCommaSeparated:(NSString*)string{ return [string componentsSeparatedByString:@","]; } </code></pre>
<p>Swift :)</p> <pre><code> var commaSeparatedString = arrayOfEntities.joinWithSeparator(",") </code></pre>
