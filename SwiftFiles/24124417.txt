Swift init Array with capacity
<p>How do I initialize an Array in swift with a specific capacity?</p> <p>I've tried:</p> <pre><code>var grid = Array &lt;Square&gt; () grid.reserveCapacity(16) </code></pre> <p>but get the error</p> <pre><code>expected declaration </code></pre>
<p>How about:</p> <pre><code>class Square { } var grid = Array&lt;Square&gt;(count: 16, repeatedValue: Square()); </code></pre> <p>Though this will call the constructor for each square.</p> <p>If you made the array have optional Square instances you could use:</p> <pre><code>var grid2 = Array&lt;Square?&gt;(count: 16, repeatedValue: nil); </code></pre> <hr> <p>EDIT: <strong>With Swift3 this initializer signature has changed to the following</strong>:</p> <pre><code>var grid3 = Array&lt;Square&gt;(repeating: Square(), count: 16) </code></pre> <p>or</p> <pre><code>var grid4 = [Square](repeating: Square(), count: 16) </code></pre> <p>Of course, both also work with <code>Square?</code> and <code>nil</code>.</p>
