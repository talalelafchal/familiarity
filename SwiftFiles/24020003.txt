Using swift func vars in other functions
<p>How can I access a variable in a function which is initialized within another function? Is it even possible?</p> <p>e.g..:</p> <pre><code>@IBAction func myButtonPressed(sender : AnyObject) { var i = 1 } </code></pre> <p>how can i access i to increment it if I press another button and give this value to an IBOutlet?</p> <p>So something like:</p> <pre><code>@IBOutlet var myLabel : UILabel = nil @IBAction func mySecondButtonPressed(sender : AnyObject) { i = i + 1 myLabel.text = String(i) } </code></pre>
<p>You'll need to declare i as an instance variable.</p> <pre><code>var i = 0 @IBAction func myButtonPressed(sender : AnyObject) { i = 1 } @IBAction func mySecondButtonPressed(sender : AnyObject) { i = i + 1 myLabel.text = String(i) } </code></pre>
