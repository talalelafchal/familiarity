How do I access Swift's REPL?
<p>I would like to know how to do this from the CLI so that I can test out some codes without launching Xcode. I saw someone did this but I couldn't find a way to do this yet.</p>
<p>After installing the Xcode6 beta, do</p> <pre><code>$ sudo xcode-select -s /Applications/Xcode6-Beta.app/Contents/Developer/ $ xcrun swift </code></pre> <p>is a Swift REPL</p> <p><strong>EDITED:</strong></p> <p>For XCode 6, use:</p> <pre><code>xcrun swift -sdk /Applications/Xcode.app/Contents/Developer/Platforms/iPhoneSimulator.platform/Developer/SDKs/iPhoneSimulator.sdk </code></pre>
<p>To extend on @vinicius answer, you can also use it without changing your default Xcode by using:</p> <pre><code>DEVELOPER_DIR=/Applications/Xcode6-Beta.app/Contents/Developer/ xcrun swift </code></pre> <p>The easiest way to do that will be to edit your .bashrc and append:</p> <pre><code>alias swift='DEVELOPER_DIR=/Applications/Xcode6-Beta.app/Contents/Developer/ xcrun swift' </code></pre> <p>At that point you'll be able to run it just by typing <code>swift</code></p> <p>If you use a csh variant you'll need:</p> <pre><code>env DEVELOPER_DIR=/Applications/Xcode6-Beta.app/Contents/Developer/ xcrun swift </code></pre> <p>and create the alias with</p> <pre><code>alias swift 'env DEVELOPER_DIR=/Applications/Xcode6-Beta.app/Contents/Developer/ xcrun swift' </code></pre>
