How to get all values of an enum in swift
<p>Is there a way to get all values of an enum in an array?</p> <p>Let's say I have the following code:</p> <pre><code>enum Suit { case Spades, Hearts, Diamonds, Clubs } </code></pre> <p>Is there a method to get the following array?</p> <pre><code>[Spades, Hearts, Diamonds, Clubs] </code></pre>
<p>I don't know if there is a method to do that. It's a really good question, hope someone will find another way much generic that mine. Anyways, I've done something that do the trick:</p> <p>Here the enum:</p> <pre><code>// SuitCount is the last one, so the total of elements (used) is SuitCount-1 enum Suit: Int { case Spades, Hearts, Diamonds, Clubs, SuitCount } </code></pre> <p>The function that return the values:</p> <pre><code>func getValueFromSuitAtIndex(#indexOfElement: Int) -&gt; String { var value = "" switch indexOfElement { case 0: value = "Spades" case 1: value = "Hearts" case 2: value = "Diamonds" case 3: value = "Clubs" default: value = "" } return value } </code></pre> <p>And in another function, where ever you want:</p> <pre><code>var suitElements = String[]() for index in 0...Suit.SuitCount.toRaw()-1 { suitElements.append(self.getValueFromSuitAtIndex(indexOfElement: index)) } // suitElements printed is: [Spades, Hearts, Diamonds, Clubs] println(suitElements) </code></pre> <p>Not really sure it's what you want, but hope that helps a bit.</p> <p><strong>EDIT 1:</strong></p> <p>An other solution, better: <a href="http://stackoverflow.com/a/24137319/2061450">http://stackoverflow.com/a/24137319/2061450</a></p>
