String formatting of a Double
<p>I can't seem to find a way to format a Double limiting the precision to just a couple of positions after the decimal point when presenting that in a label. Maybe I'm missing something major, but I couldn't find anything in the official documentation.</p> <p>Thanks in advance for the help!</p>
<p>Unsure if there is a pure Swift-way but you can always use NSStrings <code>+stringWithFormat:</code>:</p> <pre><code>let str = NSString(format: "%.2f", 1.23) </code></pre>
<p>There's no way to do this using <code>String</code> interpolation.</p> <p>You can find a good discussion about this on <a href="https://devforums.apple.com/message/973494">this Apple Dev Forum post</a>.</p> <p>You can use <code>NSNumberFormatter</code> or <code>NSDateFormatter</code> to format the output.</p> <p>Alternatively, to use <code>printf</code>-like syntax, you can still use <code>NSString</code>:</p> <pre><code>var str = NSString(format: "Hello, world %.2f", 42.12312312313) </code></pre>
