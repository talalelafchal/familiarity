UIAlertController EXC_BAD_ACCESS error - Swift
<p>I am trying to use a UIAlertController in Swift, with iOS 7, and I keep getting the following error when the alert should appear: EXC_BAD_ACCESS (code = 1, address = 0x10)</p> <p>Here is the code for the alert.</p> <pre><code>var alert:UIAlertController = UIAlertController(title: "Ooops", message: "Please Fill In Everything", preferredStyle: UIAlertControllerStyle.Alert) alert.addAction(UIAlertAction(title: "OK", style: UIAlertActionStyle.Default, handler: nil)) self.presentViewController(alert, animated: true, completion: nil) </code></pre>
<p><a href="https://developer.apple.com/library/prerelease/ios/documentation/UIKit/Reference/UIAlertController_class/" rel="nofollow"><code>UIAlertController</code></a> is available only since <strong>iOS 8.0</strong>. You should still use <code>UIAlertView</code> instead</p>
