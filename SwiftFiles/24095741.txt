swift : Enum constant with type and value
<p>I know, enumeration constant should be like this in swift</p> <pre><code>enum CompassPoint { case North case South case East case West } </code></pre> <p>But how can I assign value to first element, like Objective-C code as below</p> <pre><code>enum ShareButtonID : NSInteger { ShareButtonIDFB = 100, ShareButtonIDTwitter, ShareButtonIDGoogleplus }ShareButtonID; </code></pre>
<p>You need to give the enum a type and then set values, in the example below <code>North</code> is set as <code>100</code>, the rest will be <code>101</code>, <code>102</code> etc, just like in <code>C</code> and <code>Objective-C</code>.</p> <pre><code>enum CompassPoint: Int { case North = 100, South, East, West } let rawNorth = CompassPoint.North.rawValue // =&gt; 100 let rawSouth = CompassPoint.South.rawValue // =&gt; 101 // etc. </code></pre> <p><strong>Update</strong>: Replace <code>toRaw()</code> with <code>rawValue</code>.</p>
