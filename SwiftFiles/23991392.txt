Split string by particular character ,iOS
<p>I have a string structured in this way:</p> <p><code>"Description#Data#IMG"</code></p> <p>What's the best method to obtain three distinct strings through the position of sharps?</p>
<p>I would do something like this:</p> <pre><code>NSString *string = @"Description#Data#IMG"; NSArray *items = [string componentsSeparatedByString:@"#"]; </code></pre>
<pre><code>NSArray *components=[@"Description#Data#IMG" componentsSeparatedByString:@"#"]; </code></pre>
<pre><code>NSString *str=@"Description#Data#IMG"; //is your str NSArray *items = [str componentsSeparatedByString:@"#"]; //take the one array for split the string NSString *str1=[items objectAtIndex:0]; //shows Description NSString *str2=[items objectAtIndex:1]; //Shows Data NSString *str3=[items objectAtIndex:2]; // shows IMG Finally NSLog(@"your 3 stirs ==%@ %@ %@", str1, str2, str3); </code></pre> <p><strong>Swift</strong></p> <pre><code> //is your str var str: String = "Description#Data#IMG" var items: [AnyObject] = str.componentsSeparatedByString("#") //take the one array for split the string var str1: String = items.objectAtIndex(0) //shows Description var str2: String = items.objectAtIndex(1) //Shows Data var str3: String = items.objectAtIndex(2) // shows IMG </code></pre> <p><strong>option-2</strong></p> <pre><code>let items = str.characters.split("#") var str1: String = String(items.first!) var str1: String = String(items.last!) </code></pre>
