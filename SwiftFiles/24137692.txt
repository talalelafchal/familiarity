How to get the hour of the day with Swift?
<p>How would I get the hour of the day in Swift. I have tried <code>NSCalendar</code> and <code>NSDateComponents</code>, but I'm afraid I'm just starting with Swift.</p>
<p><strong>Swift 3:</strong> </p> <pre><code> let date = Date() let calendar = Calendar.current() let hour = calendar.component(.hour, from: date) </code></pre> <p><strong>Swift 2:</strong></p> <pre><code>let currentHour = NSCalendar.currentCalendar().component(.Hour, fromDate: NSDate()) </code></pre> <hr> <p>This could be enough :</p> <pre><code>let currentDate = NSDate() // You can input the custom as well let calendar = NSCalendar.currentCalendar() let components = calendar.components(.CalendarUnitHour | .CalendarUnitMinute, fromDate: currentDate) let currentHour = components.hour // You can play around with the ""components"" </code></pre>
<p>If you want the current hour in a String, this is as short and readable I could think of.</p> <pre><code>let formatter = NSDateFormatter() formatter.dateFormat = "HH" let timeString = formatter.stringFromDate(NSDate()) </code></pre>
<p>Finally I was able to find the easiest solution after struggling for a time</p> <pre><code>let dateComponents = NSCalendar.currentCalendar().components(NSCalendarUnit.HourCalendarUnit, fromDate: NSDate()) let nHourOfDay = dateComponents.hour </code></pre>
<p><strong>Swift 3.0</strong></p> <pre><code>let hour = Calendar.current.component(.hour, from: Date()) </code></pre> <p><strong>Swift 2.0</strong></p> <pre><code>let hour = NSCalendar.currentCalendar().component(.Hour, fromDate: NSDate()) </code></pre> <p><strong>Swift 1.0</strong></p> <pre><code>let hour = NSCalendar.currentCalendar().component(.CalendarUnitHour, fromDate: NSDate()) </code></pre>
<p>For Swift 2.0: </p> <pre><code>let hour = NSCalendar.currentCalendar().component(NSCalendarUnit.Hour, fromDate: NSDate()) </code></pre>
<p>You can get the integer value of the current hour in one step like this:</p> <pre><code>let currentHour = NSCalendar.currentCalendar().components(.Hour, fromDate: NSDate()).hour </code></pre>
