Swift Error: Variable used within its own initial value
<p>When I'm initializing an instance of an entity I'm getting the error <code>Variable used within its own initial value</code>.</p> <p>Here is the code throwing the error:</p> <pre><code>class func buildWordDefinition (word:String, language:Language, root:TBXMLElement) -&gt; WordDefinition { let word = WordDefinition(word: word, language: language) </code></pre> <p>The error points at the <code>word</code> variable. </p> <p>Here is the WordDefinition class:</p> <pre><code>class WordDefinition { let word: String let language: Language init(word: String, language:Language) { self.word = word self.language = language } } </code></pre> <p>What does this error mean ?</p>
<p>You are declaring a constant named <code>word</code>, and trying to use the argument with the same name to initialize it. The compiler tries to use the just declared constant to assign its own initial value, instead of using the argument.</p>
<p>You have a function parameter called <code>word</code> in scope and you're trying to create a constant with the same name. Name your constant something other than <code>word</code>.</p>
<p>You are redefining a constant <code>word</code> which has the same name as a parameter within your function</p> <pre><code>class func buildWordDefinition (word:String, language:Language, root:TBXMLElement) -&gt; WordDefinition { // same name as the parameter here let word = WordDefinition(word: word, language: language) </code></pre>
