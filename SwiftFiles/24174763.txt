swift unrecognized selector for target action pattern
<p>I am trying to use a gesture recognizer in swift but I keep getting an unrecognized selector crash.</p> <pre><code>init(coder aDecoder: NSCoder!) { super.init(coder: aDecoder) self.addGestureRecognizer(tapRecognizer) tapRecognizer.addTarget(self, action:"respondToTap:") self.userInteractionEnabled = true } func respondToTap(#recognizer: UITapGestureRecognizer!)-&gt; Void { switch recognizer.state { case UIGestureRecognizerState.Began: showTouchDown() case UIGestureRecognizerState.Ended: showTouchUp() default: return } } </code></pre> <p>How do a use a method signature in swift for target-action stuff?</p>
<p>interesting. It was using the # in my parameter name. </p>
<p>By defining your function as <code>func respondToTap(#recognizer: UITapGestureRecognizer!)</code>, you're giving the first parameter an <a href="https://developer.apple.com/library/prerelease/ios/documentation/swift/conceptual/swift_programming_language/Functions.html#//apple_ref/doc/uid/TP40014097-CH10-XID_214">external name</a>. So your method is actually called <code>respondToTaprecognizer:</code>. You don't want the parameter to have an external name, it should be <code>func respondToTap(recognizer: UITapGestureRecognizer!)</code>.</p>
