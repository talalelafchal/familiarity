In Swift, what is a DictionaryLiteralConvertible?
<p>If you place the following straightforward snippet of code into a Playground:</p> <pre><code>import Foundation let humanReadableNames = [ "test": NSLocalizedString("foo") ] </code></pre> <p>you'll get the compiler error:</p> <pre><code>Cannot confer the expression's type 'Dictionary' to type 'DictionaryLiteralConvertible' </code></pre> <p>The Swift function <code>NSLocalizedString</code> is defined to return a <code>String</code>, so I don't see why the type of this variable isn't simply <code>Dictionary&lt;String, String&gt;</code>.</p>
<p>It looks like you need to add the comment parameter to NSLocalizedString().</p> <pre><code>let humanReadableNames = [ "test": NSLocalizedString("foo", comment:"bar") ] </code></pre>
