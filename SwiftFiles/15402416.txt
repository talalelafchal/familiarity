How to get an array of JSON objects from NSData object
<p>So I'm using an HTTP GET method which returns an array of JSON objects, which are stored in NSData. The array looks like this:</p> <pre><code>[{"created_at":"2013-03-09T04:55:21Z","data_type":"image","id":5354,"latitude":37.785834,"longitude":-122.406417,"name":"tempObject","privacy":"public","radius":1000.0,"updated_at":"2013-03-09T04:55:21Z","user_id":101},{"created_at":"2013-03-10T20:57:08Z","data_type":"image","id":5364,"latitude":37.785834,"longitude":-122.406417,"name":"tempObject","privacy":"public","radius":1000.0,"updated_at":"2013-03-10T20:57:08Z","user_id":101}] </code></pre> <p>How would I go about extracting these JSON objects and iterate through them from the NSData?</p>
<p>If you're using iOS 5.0 and up, you can do this:</p> <p>Objective-C</p> <pre><code>NSError *error = nil; NSArray *jsonArray = [NSJSONSerialization JSONObjectWithData:myNSData options:kNilOptions error:&amp;error]; if (error != nil) { NSLog(@"Error parsing JSON."); } else { NSLog(@"Array: %@", jsonArray); } </code></pre> <p>Swift 2.1</p> <pre><code>do { let jsonArray = try NSJSONSerialization.JSONObjectWithData(myNSData, options:[]) print("Array: \(jsonArray)") } catch { print("Error: \(error)") } </code></pre>
