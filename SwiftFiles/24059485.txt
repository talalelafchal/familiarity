How to create bitmask in Swift

<p>From the documentation: Apple Inc. 'The Swift Programming Language'. iBooks. <a href="https://itun.es/nl/jEUH0.l">https://itun.es/nl/jEUH0.l</a></p> <pre><code>let shiftBits: UInt8 = 4 // 00000100 in binary shiftBits &lt;&lt; 1 // 00001000 shiftBits &lt;&lt; 2 // 00010000 shiftBits &lt;&lt; 5 // 10000000' </code></pre>
