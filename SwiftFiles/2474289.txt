How can I change UIButton title color?
<p>I create a button programmatically..........</p> <pre><code>button = [UIButton buttonWithType:UIButtonTypeRoundedRect]; [button addTarget:self action:@selector(aMethod:) forControlEvents:UIControlEventTouchDown]; [button setTitle:@"Show View" forState:UIControlStateNormal]; button.frame = CGRectMake(80.0, 210.0, 160.0, 40.0); [view addSubview:button]; </code></pre> <p>how can I change title color?</p>
<p>You can use <code>-[UIButton setTitleColor:forState:]</code> to do this.</p> <p>Example:</p> <p><strong>Objective-C</strong></p> <pre><code>[buttonName setTitleColor:[UIColor blackColor] forState:UIControlStateNormal]; </code></pre> <p><strong>Swift</strong></p> <pre><code>buttonName.setTitleColor(UIColor.blackColor(), forState: .Normal) </code></pre> <p>Thanks to <a href="http://stackoverflow.com/users/87601/richardchildan">richardchildan</a> </p>
<p>You created the UIButton is added the ViewController, The following instance method to change UIFont, tintColor and TextColor of the UIButton </p> <p><strong>Objective-C</strong></p> <pre><code> buttonName.titleLabel.font = [UIFont fontWithName:@"LuzSans-Book" size:15]; buttonName.tintColor = [UIColor purpleColor]; [buttonName setTitleColor:[UIColor purpleColor] forState:UIControlStateNormal]; </code></pre> <p><strong>Swift</strong></p> <pre><code>buttonName.titleLabel.font = UIFont(name: "LuzSans-Book", size: 15) buttonName.tintColor = UIColor.purpleColor() buttonName.setTitleColor(UIColor.purpleColor(), forState: .Normal) </code></pre> <p><strong>Swift3</strong></p> <pre><code> buttonName.titleLabel?.font = UIFont(name: "LuzSans-Book", size: 15) buttonName.tintColor = UIColor.purple buttonName.setTitleColor(UIColor.purple, for: .normal) </code></pre>
<p>If you are using Swift, this will do the same:</p> <p><code>buttonName.setTitleColor(UIColor.blackColor(), forState: .Normal)</code></p> <p>Hope that helps!</p>
