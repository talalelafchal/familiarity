Getting spreadsheets from Google-Drive using Perl module Net::Google::Drive::Simple
<p>in my script I'm using perl module <a href="https://metacpan.org/pod/Net::Google::Drive::Simple" rel="nofollow"><code>Net::Google::Drive::Simple</code></a>. I can see/download all my imported/shared files on my Google Drive, but I can't see/download any spreadsheets which I created/are shared with me.</p> <p>Am I using bad module for this or are there any special methods for handling spreadsheets?</p> <p>Thank you in advance.</p>
<p>Only documents like PDF or png can be downloaded directly. Google Drive Documents like spreadsheets or (text) documents need to be exported into one of the available formats. Check for "exportLinks" on a file given.</p> <p><a href="https://metacpan.org/pod/Net::Google::Drive::Simple#gd-download-item-local_filename" rel="nofollow">Source</a></p>
