How to sum the values in if condition?
<p>My output has the many values. But sum the particular values. My answer from the script like this</p> <pre><code># $ans 3.801 5.381 16.26 3.801 18.3 3.80 5.151 5.636 6.732 . . . my $sum; if($ans &lt;= 8) { $sum += $ans; print "$ans\n\n\n"; } print "Total value below (8) : $sum"; </code></pre> <p>How to sum the answer below 8 outside of the if condition? How can i do it? </p>
<p>Just use grep to filter out numbers you don't want from your list, and then use <a href="http://perldoc.perl.org/List/Util.html"><code>List::Util qw(sum)</code></a></p> <pre><code>use strict; use warnings; use List::Util qw(sum); my $sum = sum grep {$_ &lt; 8} &lt;DATA&gt;; print $sum; __DATA__ 3.801 5.381 16.26 3.801 18.3 3.80 5.151 5.636 6.732 </code></pre> <p>Outputs:</p> <pre><code>34.302 </code></pre>
<p>Another way that doesn't need a module.</p> <pre><code>use strict; use warnings; my $sum; $_ &lt; 8 and $sum += $_ while &lt;DATA&gt;; print "Total value below (8): $sum\n"; __DATA__ 3.801 5.381 16.26 3.801 18.3 3.80 5.151 5.636 6.732 </code></pre> <p><strong>output</strong></p> <pre><code>Total value below (8): 34.302 </code></pre>
