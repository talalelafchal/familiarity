Perl checkbox check based on database value
<p>I have a set of checkboxes like these but was wondering how to approach checking them be default based on the values in the database.</p> <pre><code>&lt;input type="checkbox" name = "others_post" id = "others_post" value="1"&gt; Other's Posts &lt;br/&gt; &lt;input type="checkbox" name = "clients_post" id = "clients_post" value="1"&gt; Cilent's Post &lt;br/&gt; &lt;input type="checkbox" name = "assigned_tasks" id = "assigned_tasks" value="1"&gt; Task Assigned </code></pre> <p>This Perl script holds all the database values:</p> <pre><code>my $prefs = USystems::UserPrefs-&gt;new($user-&gt;userid); </code></pre> <p>I found something similar on a PHP thread which is exactly what I want but with Perl <a href="http://stackoverflow.com/questions/16239663/php-checkbox-set-to-check-based-on-database-value">PHP checkbox set to check based on database value</a></p> <p>I'm not very fimiliar with Perl so I dont know if there was syntax like the PHP example that you can insert into the checkboxes?</p>
<p>I think what you're looking for is something like this:</p> <pre><code>&lt;input type="checkbox" name = "others_post" id = "others_post" value="1" &lt;% if ($prefs-&gt;{OTHERS_POST} == 1){print 'checked'} %&gt;&gt; Other's Posts &lt;br/&gt; &lt;input type="checkbox" name = "clients_post" id = "clients_post" value="1" &lt;% if ($prefs-&gt;{CLIENTS_POST} == 1){print 'checked'} %&gt;&gt; Cilent's Post &lt;br/&gt; &lt;input type="checkbox" name = "assigned_tasks" id = "assigned_tasks" value="1" &lt;% if ($prefs-&gt;{ASSIGNED_TASKS} == 1){print 'checked'} %&gt;&gt; Task Assigned </code></pre> <p>Assuming that your <code>$prefs</code> has all the information you needed stored in it. Just put in an Perl if statement that checks to see if the database value is 1. If it is than print <code>checked</code> which will give the checkbox the <code>checked</code> attribute and check it.</p>
