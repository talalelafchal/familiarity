Compare string values inside a hash in Perl
<p>I want to sort this hash in alphabetical order .This is my hash structure</p> <pre><code>my $hash = { 'names' =&gt; [ { 'state' =&gt; 'I', 'names' =&gt; 'INTTEST', }, { 'state' =&gt; 'I', 'names' =&gt; 'TEST', }, { 'state' =&gt; 'D', 'names' =&gt; 'GREATTEST', }, { 'state' =&gt; 'I', 'names' =&gt; 'Stest', }, { 'state' =&gt; 'I', 'names' =&gt; 'Atest', }, { 'state' =&gt; 'D', 'names' =&gt; 'SPtest', }, ] }; </code></pre> <p>What i have tried is </p> <pre><code>my @sorted_data = (sort { lc($b-&gt;{names}) cmp lc($a-&gt;{names});} $hash-&gt;{names}); </code></pre> <p>I want to return the same structure to the user.</p>
<p>You have to dereference <code>$hash-&gt;{names}</code> array,</p> <pre><code>use feature 'fc'; my @sorted_data = sort{ fc($b-&gt;{names}) cmp fc($a-&gt;{names}) } @{$hash-&gt;{names}}; </code></pre> <p><a href="http://perldoc.perl.org/functions/fc.html" rel="nofollow"><code>fc</code></a> checks if two strings are equal regardless of case</p> <p>If you want to sort in place, just assign result back to originating array,</p> <pre><code>@{$hash-&gt;{names}} = sort{ fc($b-&gt;{names}) cmp fc($a-&gt;{names}) } @{$hash-&gt;{names}}; </code></pre>
