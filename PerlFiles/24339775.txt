Extracting text between HTML tags using perl
<p>I have HTML tags in a document as follows:</p> <pre><code>&gt;&lt;H2 align="justify" &gt;&lt;FONT size="+2" color="#008AD9"&gt;&lt;B&gt;ACCESS_NUMBER&lt;FONT size="+2" color="#008AD9"&gt;&lt;B&gt; &lt;/H2 &gt; </code></pre> <p>I want to extract only ACCESS_NUMBER from the above HTML text.</p> <p>How can I do this? I want to make sure only the text between all <code>&lt;H2&gt;</code> tags is extracted. Any help would be appreciated.</p>
<p>Use <a href="https://metacpan.org/pod/Mojo%3a%3aDOM" rel="nofollow"><strong>Mojo::DOM</strong></a></p> <pre><code>#!/usr/bin/perl use strict; use warnings; use Mojo::DOM; my $HTML = &lt;&lt;"EOF"; &lt;html&gt; &lt;head&gt; &lt;title&gt;Test&lt;/title&gt; &lt;/head&gt; &lt;body&gt; &lt;h2&gt; &lt;font&gt;&lt;b&gt;ACCESS_NUMBER&lt;/b&gt;&lt;/font&gt; &lt;/h2&gt; &lt;/body&gt; &lt;/html&gt; EOF my $dom = Mojo::DOM-&gt;new( $HTML ); print $dom-&gt;find('h2 font b')-&gt;text; </code></pre> <p>For a 8 minute video tutorial on <code>Mojo::DOM</code> and <code>Mojo::UserAgent</code> check out <a href="http://mojocasts.com/e5" rel="nofollow"><code>Mojocast Episode 5</code></a></p>
<p>Based on what's given above, this will work, but something tells me you have more complicated HTML and/or you actually want \d+.</p> <pre><code>#!/usr/bin/perl use strict; use warnings; while(&lt;DATA&gt;){ print "$1\n" if /&gt;(\w+)&lt;/; } __DATA__ &lt;H2 align="justify" &lt;FONT size="+2" color="#008AD9"&gt;&lt;B&gt;ACCESS_NUMBER&lt;FONT size="+2" color="#008AD9"&gt;&lt;B&gt;S &lt;/H2 </code></pre>
<h1>For each line remove html tags like this:</h1> <pre><code>$l=~s/&lt;.+?&gt;/ /g; # Replace each tag with a space so you don't get run-on words. </code></pre> <p>What you're left with is only text with no html tags. </p> <p>I use software that uses tags (not html) which I have to remove, so I do this a lot. </p>
