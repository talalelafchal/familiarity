Perl adding array with another array
<p>I cam across the code below online where it's trying to add two array. Can anyone explain what it is calculating to get 14?</p> <pre><code>my @a = (1,2,5)+(8,9); print "@a"; output: 14 </code></pre>
<p>Output is 14 as <code>$a[0]</code> is 14 => <code>5+9</code></p> <p><code>+</code> operator imposes scalar context on both lists so last elements are taken and added,</p> <pre><code># in scalar context $x is assigned with last element my $x = (1,2,5); print "\$x is $x\n"; </code></pre> <p>outputs <code>$x is 5</code></p> <p><code>warnings</code> pragma would also complain, giving you a hint that something fishy is going on,</p> <pre><code>Useless use of a constant (8) in void context </code></pre>
<p>Starting with:</p> <pre><code>my @a = (1,2,5)+(8,9); </code></pre> <p>When using a list in a <a href="http://perldoc.perl.org/functions/scalar.html" rel="nofollow"><code>scalar</code></a> context, the last element is returned. Consult <a href="http://perldoc.perl.org/perlfaq4.html#What-is-the-difference-between-a-list-and-an-array%3f" rel="nofollow"><code>What is the difference between a list and an array?</code></a> for details.</p> <p>Therefore the above two lists reduce to:</p> <pre><code>my @a = 5 + 9; </code></pre> <p>Which mathematically equals:</p> <pre><code>my @a = (14); </code></pre>
