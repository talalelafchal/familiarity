How to call an onclick function from a radio button when it is changed dynamically?
<p>I'm working a CGI file and it gets the below parameter from query string and uses it as shown below.</p> <pre><code>$rettype = p-&gt;param('rettype'); if($rettype eq "A") { $achecked="checked"; $rchecked=""; } if($rettype eq "R") { $achecked=""; $rchecked="checked"; } </code></pre> <p>This is used in later portion of the file that chooses which radio button needs to be selected.</p> <pre><code>&lt;INPUT TYPE="radio" name="gctyp" VALUE="a" $achecked onClick="showTextArea()";&gt;Add &lt;INPUT TYPE="radio" name="gctyp" VALUE="r" $rchecked onClick="hideTextArea()";&gt;Remove </code></pre> <p>Now, I want the onclick JS function present in the radio button to be called based on which option was checked. It happens dynamically and the function isn't getting called.</p> <p>Thanks for your help</p>
<p>Seems like you are executing some Perl code and printing direct HTML to the output. That's known as hardcoded HTML and is not a good practice. Look into templating modules like <a href="http://search.cpan.org/perldoc?Template%3a%3aToolkit" rel="nofollow">Template::Toolkit</a> for better results and cleaner code. Also, for your own good, I hope you are using <code>use strict</code> and <code>use warnings</code> at the top of your Perl code.</p> <p>Also, your HTML tags are written in uppercase. That's deprecated, use lowercase tags and parameters, and add <code>/</code> before closing tags that don't have a closing tag, just like <code>input</code> case.</p> <p>That being said, this is an example of how to solve what you mentioned:</p> <pre><code>my $javascript_call; $rettype = p-&gt;param('rettype'); if ($rettype eq "A") { $achecked = "checked"; $rchecked = ""; $javascript_call = 'showTextArea()'; } if ($rettype eq "R") { $achecked = ""; $rchecked = "checked"; $javascript_call = 'hideTextArea()'; } print &lt;&lt;END; &lt;input type="radio" name="gctyp" value="a" $achecked onclick="showTextArea()" /&gt;Add &lt;input type="radio" name="gctyp" value="r" $rchecked onclick="hideTextArea()" /&gt;Remove &lt;script type="text/javascript"&gt; $javascript_call &lt;/script&gt; END </code></pre> <p>I'm just adding an extra variable containing a call that will be executed after printing the radio buttons and using the same code structure you presented for that.</p> <p>HTH</p>
