How to sort multidimensional array
<p>How would I go about sorting the following arrays in the following by the value in TIME?</p> <pre><code>$VAR1 = [ { 'TIME' =&gt; '45', 'PLAYER' =&gt; '', 'TYPE' =&gt; '6' }, { 'INFO' =&gt; 'Dissent', 'TEAMFLAG' =&gt; '1', 'PLID' =&gt; '8570', 'TIME' =&gt; '85', 'PLAYER' =&gt; 'Player1', 'TYPE' =&gt; '4' }, { 'TEAMFLAG' =&gt; '1', 'PLID' =&gt; '8570', 'TIME' =&gt; '35', 'PLAYER' =&gt; 'Player1', 'TYPE' =&gt; '1' }, { 'TEAMFLAG' =&gt; '0', 'PLID' =&gt; '145399', 'TIME' =&gt; '60', 'PLAYER' =&gt; 'Player3', 'TYPE' =&gt; '1' }, { 'TEAMFLAG' =&gt; '0', 'PLID' =&gt; '145248', 'TIME' =&gt; '12', 'PLAYER' =&gt; 'Player1', 'TYPE' =&gt; '1' }, { 'TIME' =&gt; '0', 'PLAYER' =&gt; '', 'TYPE' =&gt; '10' } ] </code></pre> <p>The above is a datadump of <code>$data-&gt;{EVENTS}-&gt;{EVENT}</code> which I need sorting, so have tried the following but it doesn't sort:</p> <pre><code>my @sorted = sort { $b-&gt;{TIME} &lt;=&gt; $a-&gt;{TIME} } $data-&gt;{EVENTS}-&gt;{EVENT}; </code></pre>
<p>Hash and array elements are scalars. In this case, <code>$data-&gt;{EVENTS}-&gt;{EVENT}</code> is more speceifically a reference to an array. You want <code>@{ $data-&gt;{EVENTS}-&gt;{EVENT} }</code>.</p>
