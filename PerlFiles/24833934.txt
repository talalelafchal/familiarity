Perl REGEX repetive capture groups when post processing output file
<p>I am dealing with a string of numbers in scientific format. for example </p> <pre><code>24 6.924E+06 8.316E-01 1.052E-01 1.622E+01 1.311E+01 0.000E+00 6.059E-06 (snip.. extends for a bit) </code></pre> <p>Now I want to write a regex for perl which allows me to capture the ith value in the list. So my current set up is the folloiwng </p> <pre><code>$_ =~ ^\s+\d+\s+(\d+[.]\d+E[+]\d+); my $temp = $1; </code></pre> <p>Which will get me the first number. I want to be able to capture the 7th or the 50th if I wanted without having to write a really long regex expression. </p> <p>is there a concise way of doing this? </p> <p>Thanks in advance.</p>
<p>use <a href="http://perldoc.perl.org/functions/split.html"><code>split</code></a></p> <pre><code>my @cols = split ' ', $_; my $seventh = $cols[6]; my $fiftieth = $cols[49]; </code></pre>
<p>An example for the third number:</p> <pre><code>^\s+\d+(?:\s+(\d+\.\d+E[-+]?\d+)){3} </code></pre> <p>When you repeat a capture group, the content is overwritten with the last match.</p> <p>NB: in this case, it is more clean to use <code>[0-9]</code> instead of <code>\d</code> since you only need arabic digits.</p>
<p>split is the best option for this case.<br></p> <pre><code>my @val = split ' ', $_; my $val7 = $val[6]; </code></pre>
