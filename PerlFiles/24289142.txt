<p>There are modules available in Perl for you to convert XML data to a Perl hash.</p> <p>Perl is extremely XML-friendly given the number of modules avaible in CPAN for that. </p> <p>In this case I would use the <a href="http://search.cpan.org/~braceta/XML-Hash-0.95/lib/XML/Hash.pm" rel="nofollow">XML::Hash</a> module.</p> <p>Given that you now have two hashes with key value pairs.</p> <p>You can iterate over the old hash keys, searching for the values in the new hash and print the results as needed.</p>
