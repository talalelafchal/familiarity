Plot graph in an existing excel sheet using perl
<p>I have an existing excel(.xls) file with data. I need to read the data from the file and plot chart/graph based on the same data on the same sheet. Is there any methods in Spreadsheet::ParseExcel module to plot graph. </p>
<p>You cannot use <a href="https://metacpan.org/pod/Spreadsheet%3a%3aParseExcel" rel="nofollow"><code>Spreadsheet::ParseExcel</code></a> as it can just read excel files.</p> <p>I would suggest you to read the excel data in and then try either of the following:</p> <ol> <li><p>Create another xls file and use the <a href="https://metacpan.org/pod/Spreadsheet%3a%3aWriteExcel%3a%3aChart" rel="nofollow"><code>Spreadsheet::WriteExcel::Chart</code></a> library.</p></li> <li><p>Create a xlsx file and use the <a href="https://metacpan.org/pod/Excel%3a%3aWriter%3a%3aXLSX%3a%3aChart" rel="nofollow"><code>Excel::Writer::XLSX::Chart</code></a> library.</p></li> <li><p>Another close option would be to read the excel in with <a href="https://metacpan.org/pod/Spreadsheet%3a%3aParseExcel%3a%3aSaveParser" rel="nofollow"><code>Spreadsheet::ParseExcel::SaveParser</code></a> and then add the chart and save it, but with this module all original charts are lost.</p></li> <li><p>If you are on a Windows machine you may trying to use <a href="https://metacpan.org/pod/Win32%3a%3aOLE" rel="nofollow"><code>Win32::OLE</code></a>.</p></li> </ol> <p>This alternate SO post may be helpful: <a href="https://stackoverflow.com/questions/1368960/how-can-i-modify-an-existing-excel-workbook-with-perl">How can I modify an existing Excel workbook with Perl?</a></p>
