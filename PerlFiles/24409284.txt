How can I get the path of a module dynamically?
<p>If I have something like this: </p> <p>In some script.pl: </p> <pre><code>#code here my $result = $class-&gt;foo(); # extra code here </code></pre> <p>In the module of the class: </p> <pre><code>package Test::A::Foo; use parent Test::A; sub foo { # code here } </code></pre> <p>How can I get the exact file path where <code>Test::A::Foo</code> resides when I am executing code inside the module? It is not the working directory as I am running the <code>script.pl</code> and that calls the <code>$class-&gt;foo</code>, so how can I get the actual path?</p>
<p>The built-in hash <code>%INC</code> contains the resolved paths to module files that have been loaded using <code>require</code> or <code>do</code></p> <p>The location of the <code>pm</code> file for <code>Test::A::Foo</code> is in <code>$INC{'Test/A/Foo.pm'}</code></p> <p>Something like</p> <pre><code>$INC{ (__PACKAGE__ =~ s|::|/|gr) . '.pm' } </code></pre> <p>should do what you need</p>
