Regex for &quot;-day every&quot; OR &quot;every -day time&quot; convert - with NOT if we give &quot;every-day&quot; it should not convert in perl
<p>How to make regex for <code>-day every</code> OR <code>every -day time</code> convert <code>-</code> with <code>NOT</code> if we give <code>every-day</code> it should not convert in Perl.</p> <p>For example: if we give <code>-day every</code> it should convert it to <code>NOT day every</code> .</p> <p>But we give <code>every-day</code> it should not convert it. </p> <p>I have used the regex as mentioned <code>s/(?&lt;!\w)-/NOT /g;</code> and it works fine but i have an another case where we don't need to convert dash (-) if (-) belongs under double quotes "".</p> <p>Like an example: keyword = "every - day" -time light. </p> <p>It should convert "every - day" NOT time light.</p>
<p>Regex to replace any dash not following a word character: <code>s/(?&lt;!\w)-/NOT /g;</code></p> <p>Now updated the regex to to the same, except not within double quotes:</p> <pre><code>use strict; use warnings; while (&lt;DATA&gt;) { s{(".*?")|(?&lt;!\w)-}{$1 // 'NOT '}esg; print; } __DATA__ -day every every -day time every-day "every - day" -time light. </code></pre> <p>Outputs:</p> <pre><code>NOT day every every NOT day time every-day "every - day" NOT time light. </code></pre>
