<p>The <code>qw</code> operator creates a list of strings. It quotes words, so it's "qw".</p> <pre><code>my @array = qw( x y z ); </code></pre> <p>is exactly the same as</p> <pre><code>my @array = ( 'x', 'y', 'z' ); </code></pre> <p>Your line:</p> <pre><code>my @array = qw($a,$b,$c); </code></pre> <p>is saying</p> <pre><code>my @array = ( '$a,$b,$c' ); </code></pre> <p>What you want is to drop the <code>qw</code>.</p> <pre><code>my @array = ( $a, $b, $c ); </code></pre>
