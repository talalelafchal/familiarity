Remove micro sign unicode character
<p>I'm trying to strip the micro (μ) unicode character from a string using perl regexs. Take the string</p> <pre><code>$string = "This is a micro μ and some more μμμ"; </code></pre> <p>Using a brute force approach to remove all 'more specialised' unicode characters does the job, i.e., </p> <pre><code>$string =~ s/[\x80-\xFF]+//g; </code></pre> <p>But the following that singles out the micro character does not work for me</p> <pre><code>$string =~ s/\xB5+//g; </code></pre> <p>Pretty sure 00B5 is the unicode for the micro sign. Any ideas where I'm going wrong?</p>
<p>This may not be the micro sign, check out the similar Greek small letter mu, as <a href="http://stackoverflow.com/questions/24570643/regex-remove-micro-sign-unicode-character#comment38060697_24570643">tobyink has suggested in his comment</a>.</p> <pre><code>#!/usr/bin/perl use strict; use warnings; use utf8; my $string = "This is a micro μ and some more μμμ"; $string =~ s/\x{03BC}//g; print $string; </code></pre> <p>Output: <code>This is a micro and some more</code> </p> <p><a href="http://ideone.com/wEXG2B" rel="nofollow"><strong>Demo</strong></a></p> <p>References:</p> <ul> <li><a href="http://www.fileformat.info/info/unicode/char/03BC/index.htm" rel="nofollow">Unicode Character 'GREEK SMALL LETTER MU' (U+03BC)</a></li> <li><a href="http://www.fileformat.info/info/unicode/char/00B5/index.htm" rel="nofollow">Unicode Character 'MICRO SIGN' (U+00B5)</a> </li> </ul>
