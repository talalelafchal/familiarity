Capture turbospeed while running a program
<p>I wanted to create an automation program to capture the turbospeed of the CPU while a program is running.</p> <p>My automation script should be using perl language.</p> <p>I did try to use:</p> <pre><code>system("script mytest.txt"); system("./turbostat"); </code></pre> <p>When I execute this program, only script line works but not turbostat.</p> <p>Can someone suggest me any ways through which I can capture the turbospeed?</p>
<p>Have a look at <a href="https://metacpan.org/pod/Sys%3a%3aInfo%3a%3aDevice%3a%3aCPU" rel="nofollow">Sys::Info::Device::CPU</a> module.</p> <p><strong><em>SYNOPSIS</em></strong></p> <pre><code> use Sys::Info; use Sys::Info::Constants qw( :device_cpu ); my $info = Sys::Info-&gt;new; my $cpu = $info-&gt;device( CPU =&gt; %options ); printf "CPU: %s\n", scalar($cpu-&gt;identify) || 'N/A'; printf "CPU speed is %s MHz\n", $cpu-&gt;speed || 'N/A'; printf "There are %d CPUs\n" , $cpu-&gt;count || 1; printf "CPU load: %s\n" , $cpu-&gt;load || 0; </code></pre>
