<p>So I think you want something like <a href="http://jsfiddle.net/C99TF/1/" rel="nofollow">this</a> fiddle? If so then your code just needs to be updated as follows:</p> <pre><code>my %data = ( 1 =&gt; "http://www.google.com" , 2 =&gt; "http://www.yahoo.com" ); &lt;a href="#" id="popover" data-title="Websites"&gt; Click me! &lt;/a&gt; &lt;div id="popoverTemplate" class="collapse"&gt; foreach my $website (values %data) { &lt;a href="$data-&gt;{$website}" target="_blank"&gt;$data-&gt;{$website}&lt;/a&gt; } &lt;/div&gt; &lt;script type="text/javascript"&gt; $(document).ready(function () { $("#popover").popover({ html : true, content: function() { return $("#popoverTemplate").html(); } }); }); &lt;/script&gt; </code></pre> <p>This basically uses a link as the source of the popover and then an HTML template containing your list of sites as the content.</p>
