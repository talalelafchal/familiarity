What is this perl object and how do I iterate through it?
<p>I have a perl object that was returned to me whose data I can't seem to extract. If I run Data::Dumper->Dump on it as:</p> <pre><code>Data::Dumper-&gt;Dump($message_body) </code></pre> <p>I get:</p> <pre><code>$VAR1 = 'SBM Message '; $VAR2 = '--SBD.Boundary.605592468 '; $VAR3 = 'Content-Type: text/plain;charset=US-ASCII '; $VAR4 = 'Content-Disposition: inline </code></pre> <p>If I execute the line:</p> <pre><code>print $message_body; </code></pre> <p>I get:</p> <pre><code>ARRAY(0x9145668) </code></pre> <p>I would think this is an array. However, trying to iterate through it there only seems to be a single element. How do I extract each of the elements from this? By the way this, is basically the body of a mail message extracted using the MIME::Parser package. It was created using the following:</p> <pre><code>my $parser = new MIME::Parser; my $entity = $parser-&gt;parse($in_fh); # Where $in_fh points to a mail message $message_body = $entity-&gt;body; </code></pre>
<p><code>Data::Dumper</code> is only a poor man's choice to see the content. To see all the gory internal details use <code>Devel::Peek</code> instead.</p> <pre><code>use Devel::Peek; Dump $message_body; </code></pre>
<p>Try below <code>foreach</code> loop.</p> <pre><code>foreach my $item (@{$message_body}) { print $item."\n"; } </code></pre> <p><code>$message_body</code> is an ARRAY reference. Hence you need to dereference it and then iterate through each element using the <code>foreach</code> loop.</p> <p><strong>Read:</strong> </p> <p><a href="http://perlmeme.org/howtos/using_perl/dereferencing.html" rel="nofollow">http://perlmeme.org/howtos/using_perl/dereferencing.html</a> and <a href="http://www.thegeekstuff.com/2010/06/perl-array-reference-examples/" rel="nofollow">http://www.thegeekstuff.com/2010/06/perl-array-reference-examples/</a></p>
