How to parse complicated hash ref in perl?
<p>I used <a href="https://metacpan.org/pod/JSON%3a%3aParse" rel="nofollow"><code>JSON::Parse</code></a> to decode a json file. When I dumped it to a file it looks as printed below. I need to extract the 'url' from each of the repositories. How can I go about doing that? I have tried to access the elements, but cannot seem to get to them. Any suggestions?</p> <p>How I decoded the .json file:</p> <pre><code>my $json_hash_ref = {}; $json_hash_ref= json_file_to_perl ($REPO_JSON_FILE) or confess "No repos found"; </code></pre> <p>This is the data dump:</p> <pre><code>$VAR1 = { 'repositories' =&gt; [ { 'size' =&gt; 2970, 'watchers' =&gt; 49, 'created_at' =&gt; '2012-03-20T05:04:42Z', 'url' =&gt; 'http://github.cerner.com/aeon-core/aeon-topology', 'open_issues' =&gt; 0, 'followers' =&gt; 49, 'owner' =&gt; 'aeon-core', 'has_downloads' =&gt; 1, 'has_issues' =&gt; 0, 'language' =&gt; 'Java', 'pushed' =&gt; '2014-06-17T18:32:37Z', 'private' =&gt; ${\$VAR1-&gt;{'repositories'}[0]{'has_issues'}}, 'name' =&gt; 'aeon-topology', 'score' =&gt; '1', 'has_wiki' =&gt; ${\$VAR1-&gt;{'repositories'}[0]{'has_issues'}}, 'pushed_at' =&gt; '2014-06-17T18:32:37Z', 'description' =&gt; '', 'username' =&gt; 'aeon-core', 'created' =&gt; '2012-03-20T05:04:42Z', 'homepage' =&gt; '', 'forks' =&gt; 59, 'fork' =&gt; ${\$VAR1-&gt;{'repositories'}[0]{'has_issues'}}, 'type' =&gt; 'repo' }, { ..... },... </code></pre>
