Perl program for palindrome in paragraph
<p>I want to write a perl program for finding the number of palindromes in a paragraph. But here I have to do it using an array. Please help.</p>
<p>Take a look at <a href="http://perldoc.perl.org/perlretut.html" rel="nofollow"><code>perlretut - Recursive patterns</code></a>.</p> <p>It includes an example of testing for palindromes that you can no doubt adapt to your purposes, whatever they are exactly:</p> <pre><code>use strict; use warnings; my $pp = qr/^(\W* (?: (\w) (?1) \g{-1} | \w? ) \W*)$/ix; for $s ( "saippuakauppias", "A man, a plan, a canal: Panama!" ){ print "'$s' is a palindrome\n" if $s =~ $pp; } </code></pre> <p>Outputs:</p> <pre><code>'saippuakauppias' is a palindrome 'A man, a plan, a canal: Panama!' is a palindrome </code></pre>
