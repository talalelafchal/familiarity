Perl script to download a file from LAN
<p>path is similar to this <code>\\pc1\folder1\folder2\file.bin</code> Above is the shared folder on the network and I have to save it to a path similar to <code>c:\Users\me\Desktop\file_copy.bin</code></p> <p>I am new to Perl and thus clear explanation of how it is happening will be appreciable.</p>
<p>You can try the <a href="https://metacpan.org/pod/Win32::FileOp" rel="nofollow">Win32::FileOp</a> module, below is how you open the file, you can use the CopyFile option for downloading.</p> <pre><code>use Win32::FileOp qw/ Map /; Map 'X:' =&gt; ' \\\\machine\\share', { user =&gt; 'foo', passwd =&gt; 'bar' }; open FILE, 'x:/path/file.ext' or die "..."; </code></pre> <p>Also check out <a href="http://perldoc.perl.org/File/Fetch.html" rel="nofollow">File::Fetch</a></p>
