How to match a table tag in Perl regex?
<p>I tried</p> <p><code>/(^&lt;table)(.*?)($&gt;)/</code></p> <p>It should match everything that is between <code>&lt;&gt;</code> for the table tag, but it does not</p>
<p>As mentioned in the comments on <a href="http://stackoverflow.com/questions/24427540/html-content-regular-expression-perl/" title="HTML content regular expression - perl">this question</a>, it's not really practical to parse HTML with regular expressions.</p> <p>Here's an example using <a href="http://search.cpan.org/dist/Mojolicious/lib/Mojo/DOM.pm" rel="nofollow" title="get it from CPAN">Mojo::DOM</a>, inspired by <a href="http://stackoverflow.com/a/4598384/3746946" title="HTML parsing in perl">this StackOverflow answer</a>:</p> <pre><code>#!/usr/bin/env perl use strict ; use warnings ; use Mojo::DOM ; my $html = &lt;&lt;EOHTML; &lt;!DOCTYPE html&gt; &lt;html&gt; &lt;head&gt; &lt;title&gt;Sample HTML with a table&lt;/title&gt; &lt;/head&gt; &lt;body&gt; &lt;table border&gt; &lt;tr&gt; &lt;td&gt;a&lt;/td&gt; &lt;td&gt;b&lt;/td&gt; &lt;td&gt;c&lt;/td&gt; &lt;/tr&gt; &lt;tr&gt; &lt;td&gt;1&lt;/td&gt; &lt;td&gt;2&lt;/td&gt; &lt;td&gt;3&lt;/td&gt; &lt;/tr&gt; &lt;/table&gt; &lt;/body&gt; &lt;/html&gt; EOHTML my $dom = Mojo::DOM-&gt;new ; $dom-&gt;parse( $html ) ; for my $div ( $dom-&gt;find( 'td' )-&gt;each ) { print $div-&gt;all_text . "\n" ; } </code></pre> <p>The output is:</p> <pre><code>a b c 1 2 3 </code></pre>
