Printing values of an array from an array of array references
<p>How can I print the values of an array. I have tried several ways but I am unable to get the required values out of the arrays:</p> <p>@array; Dumper output is as below :</p> <pre><code>$VAR1 = [ 'a', 'b', 'c' ]; $VAR1 = [ 'd', 'e', 'f' ]; $VAR1 = [ 'g', 'h', 'i' ]; $VAR1 = [ 'j', 'k', 'l' ]; for my $value (@array) { my $ip = $value-&gt;[0]; DEBUG("DEBUG '$ip\n'"); } </code></pre> <p>I am getting output as below, which means <code>foreach</code> instance I am only getting the first value.</p> <pre><code> a d g j </code></pre> <p>I have tried several approaches :</p> <p>First option :</p> <pre><code>my $size = @array; for ($n=0; $n &lt; $size; $n++) { my $value=$array[$n]; DEBUG( "DEBUG: Element is as $value" ); } </code></pre> <p>Second Option :</p> <pre><code> for my $value (@array) { my $ip = $value-&gt;[$_]; DEBUG("DEBUG Element is '$ip\n'"); } </code></pre> <p>What is the best way to do this?</p>
<p>It is obvious that you have list of arrays. You only loop over top list and print first (0th) value in your first example. Barring any automatic dumpers, you need to loop over both levels.</p> <pre><code>for my $value (@array) { for my $ip (@$value) { DEBUG("DEBUG '$ip\n'"); } } </code></pre>
<p>You want to <a href="http://perldoc.perl.org/perlreftut.html" rel="nofollow">dereference</a> here so you need to do something like:</p> <pre><code>my @array_of_arrays = ([qw/a b c/], [qw/d e f/ ], [qw/i j k/]) for my $anon_array (@array_of_arrays) { say for @{$anon_array} } </code></pre> <p>Or using your variable names:</p> <pre><code>use strict; use warnings; my @array = ([qw/a b c/], [qw/d e f/], [qw/i j k/]); for my $ip (@array) { print join "", @{$ip} , "\n"; # or "say" } </code></pre> <p>Since there are anonymous arrays involved I have focused on dereferencing (using PPB style!) instead of nested loops, but <code>print for</code> is a loop in disguise really.</p> <p>Cheers.</p>
