Modify the URL with the variable
<p>I want to add a variable to the URL as www.example.com/index.cgi?username=demo</p> <p>I have a form in index.html which has action=login.cgi</p> <p>In login.cgi</p> <pre><code>if (($cgi-&gt;param('username') eq 'demo') &amp;&amp; ($cgi-&gt;param('password') eq 'demo')) { my cookie= CGI::Cookie-&gt;new(-name=&gt;'username', -value=&gt;$cgi-&gt;param('username')); print $cgi-&gt;redirect(-uri =&gt; 'index.cgi', -cookie =&gt; $cookie); } else { print $cgi-&gt;redirect(-uri =&gt; 'index.html'); } </code></pre> <p>In index.cgi</p> <pre><code>my $username = $cookies{username}-&gt;value; </code></pre> <p>So I want to modify the URL for index.cgi such that it adds the username on it as well from the variable $username </p> <pre><code>www.example.com/index.cgi?username=demo </code></pre>
<p>You can combine strings using the dot operator or double quotes.</p> <p>Using the dot operator: <code>my $url = 'www.example.com/index.cgi?username=' . $username;</code>.</p> <p>Using double quotes: <code>my $url = "www.example.com/index.cgi?username=$username";</code>.</p> <p>You can combine multiple strings: </p> <pre><code>my $base_url = 'www.example.com/index.cgi'; my $key = 'username'; my $value = $username; my $url = "$base_url?$key=$value"; </code></pre> <p>Do not forget to escape special characters (such as &amp; or %) in your strings:</p> <pre><code>use URI::Escape; $username = uri_escape $username; </code></pre> <p>If you have a hash of keys and values, you can use this:</p> <pre><code>use URI::Escape; my %h = (username =&gt; 'demo', password =&gt; 'demo', topic =&gt; 'perl'); my $url = 'www.example.com/index.cgi'; $url .= '?' . join '&amp;', map {uri_escape "$_=$h{$_}"} keys %h; </code></pre>
