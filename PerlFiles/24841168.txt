how to match other languages words with regex

The only possibility I found is xregexp, but it seems that only would work to detect character classes and not individual words. ¿Is really impossible to make it work in JavaScript? </p> <p>The Python 3 Documentation states that this language can handle unicode characters, but in this case it seems that it's necessary to write characters with the unicode code... With which languaje would it be possible to replace words in the simplest way as I wrote in the code? Python, Java, Perl ...?</p>
<p>This should do it:</p> <p><code>'Ενώ Gallant δεν λειτουργεί στην προσπάθεια να χτίσει πια μηχανές αποκωδικοποίησης, οι άλλοι.'.replace( /(προσπάθεια)/g, 'effort' )</code></p> <p><strong>Edit</strong></p> <p>I think this does exactly what you want:</p> <pre><code>String.prototype.translate = function translate( greek, english ) { return this.replace( new RegExp( '(' + greek + ')' ), english ); } var translatedString = 'Ενώ Gallant δεν λειτουργεί στην προσπάθεια να χτίσει πια μηχανές αποκωδικοποίησης, οι άλλοι.'.translate( 'προσπάθεια', 'effort' ); console.log( translatedString ); </code></pre>
<p>Perl has exceptional unicode handling. E.g. the following code:</p> <pre><code>use 5.016; use warnings; use utf8; use open qw(:std :utf8); my $str= "Ενώ Gallant δεν λειτουργεί στην προσπάθεια να χτίσει πια μηχανές αποκωδικοποίησης, οι άλλοι."; $str =~ s/\bπροσπάθεια\b/effort/g; say $str; </code></pre> <p>prints</p> <pre><code>Ενώ Gallant δεν λειτουργεί στην effort να χτίσει πια μηχανές αποκωδικοποίησης, οι άλλοι. </code></pre>
