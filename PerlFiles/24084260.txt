Perl - Iterate through hash of array values
<p>I have a data structure which I want to iterate through, then push to another array for temporary storage. </p> <p>Each view in the hash has an array of fieldsets.</p> <pre><code>1 = { fieldset =&gt; ('package', 'payment'), }, 2 =&gt; { fieldset =&gt; ('address, 'review'), } 3 =&gt; { fieldset =&gt; ('confirm'), } </code></pre> <p>etc. </p> <p>I want to grab all these values, and comma separate them into an another array, so I can see which steps the customer has left. </p> <p>if I try </p> <pre><code>@array = $value-&gt;{fieldsets} </code></pre> <p>it only grabs the first item. How do I grab all of them?</p> <p>Let me know if I haven't explained it in enough depth.</p>
<p>Hash values contain scalars.</p> <pre><code>1 = { fieldset =&gt; ('package', 'payment'), }, </code></pre> <p>flattens to:</p> <pre><code>1 = { fieldset =&gt; 'package', payment =&gt; undef, }, </code></pre> <p>You want:</p> <pre><code>1 = { fieldset =&gt; ['package', 'payment'], }, </code></pre> <p>To store an arrayref scalar, and access by dereferencing the contents of the fieldsets key:</p> <pre><code>@array = @{$value-&gt;{fieldset}} </code></pre>
