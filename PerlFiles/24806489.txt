Access perl attribute in associative array
<p>Is there an easier way to access the value of the 'type' attribute without looping through the whole object to find it?</p> <pre><code>[ { type =&gt; "voipPassword", vals =&gt; ["data"] }, { type =&gt; "sn", vals =&gt; ["data"] }, { type =&gt; "voipExtension", vals =&gt; [data] }, { type =&gt; "cn", vals =&gt; ["data"] }, { type =&gt; "telephoneNumber", vals =&gt; [data] }, { type =&gt; "objectClass", vals =&gt; ["data"] }, { type =&gt; "phoneMAC", vals =&gt; ["data"] }, ] </code></pre>
<p>You can access <code>type</code> directly like this example:</p> <pre><code>#!/usr/bin/perl use strict; use warnings; my $ref = [ { type =&gt; "voipPassword", vals =&gt; ["data"] }, { type =&gt; "sn", vals =&gt; ["data"] }, { type =&gt; "voipExtension", vals =&gt; ["data"] }, { type =&gt; "cn", vals =&gt; ["data"] }, { type =&gt; "telephoneNumber", vals =&gt; ["data"] }, { type =&gt; "objectClass", vals =&gt; ["data"] }, { type =&gt; "phoneMAC", vals =&gt; ["data"] }, ]; print $ref-&gt;[0]-&gt;{'type'} . "\n"; print $ref-&gt;[1]{'type'} . "\n"; </code></pre> <p>Output:</p> <pre><code>voipPassword sn </code></pre> <p>See <a href="http://perldoc.perl.org/perlreftut.html" rel="nofollow">perlreftut</a> for more details.</p>
