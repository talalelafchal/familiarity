In Perl, how to split a string in particular three parts?
<p>If I have </p> <p>$string= 0000111111110101;</p> <p>How can I split it into two part as </p> <p>$s1= 0000; and $s2= 11111111; $s3= 0101;</p> <p>namely, I want the first 4-bit, the last 4-bit, and the remained middle bits. What if the three wanted parts' lengths are parameters that indicated by user?</p> <p>Is there any method in split to do it this way? If I use regex to get it done, what's it will be like?</p>
<p>Just use a regular expression or just <a href="http://perldoc.perl.org/functions/substr.html" rel="nofollow"><code>substr</code></a>:</p> <pre><code>my $string = '0000111111110101'; # Regular expression method if ($string =~ /(.{4})(.*)(.{4})/s) { print "$1, $2, $3\n"; } # Substr method my $pre = substr $string, 0, 4, ''; my $post = substr $string, -4, 4, ''; print "$pre, $string, $post"; </code></pre> <p>Outputs:</p> <pre><code>0000, 11111111, 0101 0000, 11111111, 0101 </code></pre>
<p>Use grouping and get it from the matched groups using index</p> <pre><code>^([^1]*)(1+)(.*)$ </code></pre> <p><a href="http://regex101.com/r/pV8qU8/7" rel="nofollow">Online demo</a></p> <p>3 Matched Groups: </p> <pre><code>1 Group: 0000 2 Group: 11111111 3 Group: 0101 </code></pre> <hr> <p>Try this one as well</p> <pre><code>^([^1]*)([^0]*)(.*)$ </code></pre> <p><a href="http://regex101.com/r/pV8qU8/8" rel="nofollow">Online demo</a></p> <hr> <p>Make it more precise</p> <pre><code>^([^1]0*)([^0]1*)(\d+)$ </code></pre> <p><a href="http://regex101.com/r/pV8qU8/9" rel="nofollow">Online demo</a></p>
<pre><code>use feature qw/say/; my ($s1, $s2, $s3) = unpack "A4A8A4", '0000111111110101'; say for $s1, $s2, $s3; </code></pre>
