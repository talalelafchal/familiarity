<p>I am in the initial stage of my college mini-project and am stuck.</p> <p>Can anybody please let me know the basic and advanced concepts and ways of "Extracting Data from an HTML page Using Perl" along with the code?</p> <p>If not yjrm please show me the path to go through the resources related to the concept so that I can learn on my own.</p>
<p>Depending on what you need to do exactly you may want to look at <a href="https://metacpan.org/pod/HTML::TreeBuilder" rel="nofollow">HTML::TreeBuilder</a> (and the extension <a href="https://metacpan.org/pod/HTML::TreeBuilder::XPath" rel="nofollow">HTML::TreeBuilder::XPath</a> to get XPath goodness) or, if you need to interact with websites, <a href="https://metacpan.org/pod/WWW::Mechanize" rel="nofollow">WWW::Mechanize</a>.</p> <p>An other fairly popular tool is <a href="https://metacpan.org/pod/Mojo::DOM" rel="nofollow">Mojo::DOM</a>.</p>
<p>This should get you started.</p> <pre><code>#!/usr/bin/perl use strict; use warnings; use autodie; use LWP::Simple; #For getting a websites HTML also see LWP::UserAgent use HTML::Tree; #Use a parser to parse HTML, read the docs on CPAN #Use LWP get a page's contents #We'll use the url to this question http://stackoverflow.com/questions/24858906/data-extraction-from-html-using-perl my $url = "http://stackoverflow.com/questions/24858906/data-extraction-from-html-using-perl"; #All the html will be in content my $content = get($url); my $p = HTML::Tree-&gt;new(); #parse the string in $content. You can also parse_from_file or parse_from_url #Though for learning sake you should get used to LWP $p-&gt;parse($content); #Check HTML::Element documentation for the data manipulation part my $post = $p-&gt;find_by_attribute('class', 'post-text'); #Should print your question out. print $post-&gt;as_text(); </code></pre> <p>Now review the documentation for:</p> <p><a href="https://metacpan.org/pod/LWP::Simple" rel="nofollow">LWP::Simple</a></p> <p><a href="https://metacpan.org/pod/LWP::UserAgent" rel="nofollow">LWP::UserAgent</a></p> <p><a href="https://metacpan.org/pod/HTML::Tree" rel="nofollow">HTML::Tree</a></p> <p><a href="https://metacpan.org/pod/HTML::Element" rel="nofollow">HTML::Element</a> </p>
