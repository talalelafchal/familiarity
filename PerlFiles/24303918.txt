How to encrypt and decrypt a specific column in a file by using Perl?
<p>I have log file like below,</p> <pre><code>NAME ID LOCATION aa 12 in bb 13 freak cc 14 test </code></pre> <p>I want to encrypt and as well as decrypt the field "LOCATION". How to do that by using any Encryption module in Perl ?</p> <p>Since I'm new to Perl, help me .</p>
<p>Use <a href="https://metacpan.org/pod/Crypt%3a%3aCBC" rel="nofollow"><strong>Crypt::CBC</strong></a></p> <p>A sample script</p> <pre><code>#!/usr/bin/perl use strict; use Crypt::CBC; unless (scalar @ARGV == 3) { die "Usage: $0 encrypt|decrypt|en|de \$mysecretkey \$file_to_dencrypt"; } my $type = shift @ARGV; my $key = shift @ARGV; my $file = shift @ARGV; die "The first ARGV should be one of de, en, encrypt, decrypt" if ($type !~ /^(en|de)(crypt)?$/); die "the file $file is not existence" unless (-f $file); my $DEBUG = 1; print "type is $type, key is $key, file is $file\n" if $DEBUG; my $cipher = Crypt::CBC-&gt;new( -key =&gt; $key, -cipher =&gt; 'Blowfish' ); local $/; open(FH, $file) or die $!; flock(FH, 2); my $data = &lt;FH&gt;; close(FH); my ($save_data, $save_file); if ($type =~ /^en(crypt)?$/) { $save_data = $cipher-&gt;encrypt($data); $save_file = $file . '.encrypt'; } else { $save_data = $cipher-&gt;decrypt($data); $save_file = $file . '.decrypt'; } open(FH, '&gt;', $save_file) or die $!; print FH $save_data; close(FH); if (-e $save_file) { print "$type file $file to $save_file OK\n"; } else { print "failed without reason\n"; } </code></pre>
