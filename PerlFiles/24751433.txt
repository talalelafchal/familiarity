File creatiion date in ddmmyy format in perl
<p>I want to know how can i get file creation date in DDMMYYYY format. I have tried this code but it does not meet my epxectations..</p> <pre><code>$creationtime=ctime(stat($filen)-&gt;ctime); print "File was created on $creationtime\n"; </code></pre> <p>Output is not in DDMMYYY format. It is printing time also. I just want date only and that too in DDMMYYYY format.</p>
<p><code>ctime</code> returns an epoch value. To get an alternate format, you must convert it.</p> <pre><code>use strict; use warnings; use File::stat use Time::Piece; my $creationtime = localtime( stat($filename)-&gt;ctime )-&gt;strftime("%d%m%Y"); </code></pre>
<p>Adapted from <a href="http://stackoverflow.com/questions/1333463/how-to-change-ctime-to-normal-string-representation">this post</a>:</p> <pre><code>use POSIX qw (strftime); use File::stat; $creationtime = stat($filen)-&gt;ctime; # in Unix epoch representation print "File was created on ", strftime ('%d%m%Y', localtime $creationtime), "\n"; </code></pre> <p>or</p> <pre><code>$creationtime = strftime ('%d%m%Y', localtime stat($filen)-&gt;ctime); # in DDMMYYYY representation print "File was created on $creationtime\n"; </code></pre>
