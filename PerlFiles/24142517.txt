perl command line one liner + delete word after match the line in file
<p>Please advise how to remove the first word after the line <strong>export PASSORD=</strong> </p> <p>by perl one liner code ( on linux machines )</p> <p>For example, I have this line in my file</p> <pre><code> export PASSORD=HG&amp;@VFQ1 </code></pre> <p>the expected results after we run the perl syntax should be without the password</p> <pre><code> export PASSORD= </code></pre> <p>perl must match exactly the line --> “export PASSORD=” otherwise password will not be deleted from the file.</p> <p>examples: The password must not deleted in the following cases:</p> <pre><code> export PASSORD =HG&amp;@VFQ1 exportPASSORD=HG&amp;@VFQ1 export_PASSORD=HG&amp;@VFQ1 #export_PASSORD=HG&amp;@VFQ1 </code></pre> <p>I created the basic perl one liner syntax to delete any word in file but couldn’t successfully add the ability to delete a word only after matching the line --> <strong>export PASSORD=</strong></p> <pre><code> perl -i -pe 's/\QANY_STRING//' file </code></pre>
<p>It will delete everything after <code>export PASSORD=</code></p> <pre><code>perl -i -pe 's/^export PASSORD=\K.*//' file </code></pre> <p>For <code>\K</code> positive look behind, at Perl v5.10 or higher is required.</p> <p>For older versions,</p> <pre><code>perl -i -pe 's/(?&lt;=export PASSORD=).*//' file -or- perl -i -pe 's/(export PASSORD=).*/$1/' file -or- perl -i -pe 's/export PASSORD=.*/export PASSORD=/' file </code></pre>
<p>Just use substitution. <code>^</code> matches a line beginning.</p> <pre><code>perl -i~ -pe 's/^(export PASSORD=).*/$1/' </code></pre>
