Perl Program to search for a string over a set of files over SSH
<p>I have a perl script which can be used to ssh into a remote server using <a href="https://metacpan.org/pod/Net%3a%3aSSH2" rel="nofollow"><code>Net::SSH2</code></a>, i need to search for a particular string over the files in a given directory in the remote system and print the given files in which the string occurs . Any ideas /sample codes on how i can go about this ? Thanks </p>
<p>Solution proposed and accepted in the comments:</p> <pre><code>ssh &lt;user&gt;@&lt;host&gt; grep -d recurse -l &lt;string&gt; &lt;directories&gt; </code></pre>
