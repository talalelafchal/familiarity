Perl no warnings 'experimental' on old perl
<p>I have a piece of code which uses a lot of experimental (<code>when</code>, <code>smartmatch</code>, <code>given</code>) features, it's not my code, and I don't want to see a lot of warnings about experimental features. So I added <code>no warnings 'experimental';</code> to this code. But <code>experimental</code> category was added only in perl 5.18 so my code dies in older perls: </p> <blockquote> <p>Unknown warnings category 'experimental'</p> </blockquote> <p>How can I disable this warnings and do not break my code in older perls?</p>
<p>Try installing and using the <a href="https://metacpan.org/pod/experimental" rel="nofollow noreferrer">experimental</a> module. On older perls it will do nothing, and on newer perls it will disable the warnings.</p> <pre><code>use experimental qw(smartmatch switch); </code></pre>
<p>Try <a href="https://metacpan.org/pod/if">if</a>:</p> <pre><code>no if ($] &gt;= 5.018), 'warnings' =&gt; 'experimental'; </code></pre>
