extract numerals from a line in file using perl
<p>Suppose I have the following lines in a file:</p> <pre><code>set gate report abc 28 -ext set gate report xyz 29 -ext set gate report hello 35 -ext </code></pre> <p>I want to extract the numerals 28, 29 and so on from the lines and put them into a variable. Supposing this file has 100 similar lines, how can I extract the numerals? Can I use split in some manner for this?</p>
<p>You can use a <a href="http://perldoc.perl.org/perlre.html" rel="nofollow">regular expression</a> to capture the numbers and store them in an array:</p> <pre><code>use warnings; use strict; my @nums; while (&lt;DATA&gt;) { push @nums, $1 if /(\d+)/; } use Data::Dumper; print Dumper(\@nums); __DATA__ set gate report abc 28 -ext set gate report xyz 29 -ext set gate report hello 35 -ext </code></pre> <p>Output:</p> <pre><code>$VAR1 = [ '28', '29', '35' ]; </code></pre> <p>I think regex is a more appropriate tool than <code>split</code> in this case.</p> <p><a href="http://perldoc.perl.org/perlintro.html#Files-and-I/O" rel="nofollow">Basics of reading a file.</a></p>
