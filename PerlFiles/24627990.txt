Perl - Html write to PDF
<p>I'm trying to create PDF from HTML using <a href="https://metacpan.org/pod/PDF%3a%3aFromHTML" rel="nofollow"><code>PDF::FromHTML</code></a> package. It works fine if I use load_file(html file). How do I change if I want to write text instead the html file. For an example:</p> <pre><code>use PDF::FromHTML; my $pdf = PDF::FromHTML-&gt;new( encoding =&gt; 'utf-8' ); $pdf-&gt;load_file('source.html'); $pdf-&gt;convert; $pdf-&gt;write_file('target.pdf'); </code></pre> <p>it take the html from source.html and create the PDF. How to do if I want a text like this:</p> <pre><code>use PDF::FromHTML; my $pdf = PDF::FromHTML-&gt;new( encoding =&gt; 'utf-8' ); $pdf-&gt;load_file('&lt;h1&gt;Hello World&lt;/h1&gt;'); $pdf-&gt;convert; $pdf-&gt;write_file('target.pdf'); </code></pre> <p>Any references or advice is highly appreciated.</p>
<p>As <a href="https://metacpan.org/pod/PDF%3a%3aFromHTML" rel="nofollow"><code>PDF::FromHTML</code></a> suggests, use scalar reference to your html</p> <pre><code>use strict; use warnings; use autodie; use PDF::FromHTML; my $data = do {local $/; &lt;DATA&gt;}; my $pdf = PDF::FromHTML-&gt;new( encoding =&gt; 'utf-8' ); $pdf-&gt;load_file(\$data); $pdf-&gt;convert; $pdf-&gt;write_file('target.pdf'); # Start Fresh each time BEGIN { unlink 'target.pdf'; } __DATA__ &lt;html&gt; &lt;body&gt; &lt;h1&gt;Hello world&lt;/h1&gt; &lt;p&gt;From data block.&lt;/p&gt; &lt;/body&gt; &lt;/html&gt; </code></pre>
