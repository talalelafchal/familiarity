Regex in ARGV. Perl
<p>If i put variable instead of regexp in code, program didn't find pattern, why?</p> <p><strong>Not working:</strong></p> <pre><code>my $regex = $ARGV[0]; print $file "$1\n" if $_ =~ /(\b\$regex\b)/; </code></pre> <p><strong>Works:</strong></p> <pre><code>print $file "$1\n" if $_ =~ /(\b\w{3}\b)/; </code></pre> <p>Also if i input as argument to program smth like W{3,2}, the 2 counts as the second argument. How to avoid that?</p>
<p>This is a shell question. Enclose your parameter in quotes:</p> <pre><code>perl yourscript.pl '\w{3}' </code></pre> <p>Then remove the escaping in your code so the <code>$regex</code> variable can be interpolated:</p> <pre><code>my $regex = $ARGV[0]; print $file "$1\n" if $_ =~ /(\b$regex\b)/; # Removed \ before $regex </code></pre>
