How to get rid of this white background color on all variables in Padre Perl IDE
<p>Padre is always my favourite IDE for Perl. I downloaded from here: <a href="http://code.google.com/p/dwimperl/downloads/detail?name=dwimperl-5.14.2.1-v7-32bit.exe&amp;can=2&amp;q=" rel="nofollow noreferrer">http://code.google.com/p/dwimperl/downloads/detail?name=dwimperl-5.14.2.1-v7-32bit.exe&amp;can=2&amp;q=</a></p> <p>I have one small problem. There is white block (like background color) around all variables, as shown in the attached image. This is on a XP machine. Most of other machines are not like this. How can I get rid of it?</p> <p><img src="https://i.stack.imgur.com/KkCL7.jpg" alt="enter image description here"> <a href="http://tinypic.com/r/316x1ug/8" rel="nofollow noreferrer">http://tinypic.com/r/316x1ug/8</a></p>
<p><a href="http://padre.perlide.org/trac/wiki/Features/SyntaxHighlighting" rel="nofollow">Wiki: Padre syntax highlighting</a></p> <p>Follow this link. Go to the menu options, look for a backgroundcolor: entry. Then remove it and make it the correct color you want.</p> <p>Otherwise create your own YML file for highlighting.</p> <p>I would be more specific IF I could actually install padre, but this should get you in the right direction.</p>
