How can you modify a DBIx::Class::Row in place?
<p>Using DBIx::Class::ResultSet is it possible to make changes to the results in memory as attempted below?</p> <pre><code>my @results = $self-&gt;search(...); for my $result (@results) { my $row = {$result-&gt;get_columns}; $row-&gt;{fieldname} = 'something else'; } return @results; </code></pre>
<p>You may use something like this:</p> <pre><code>my @rows = $self-&gt;search( {...}, {result_class =&gt; 'DBIx::Class::ResultClass::HashRefInflator'} ); for my $row (@rows) { $row-&gt;{fieldname} = 'something else'; } return @rows; </code></pre>
<p>A column accessor is generated by DBIx::Class for every column you define. It can be used both as a reader</p> <pre><code>my $value = $row-&gt;columnname; </code></pre> <p>and as a writer</p> <pre><code>$row-&gt;columnname('new value'); </code></pre>
