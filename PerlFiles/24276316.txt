frequency of letters at position in string
<p>I want to count the frequency of 4 letters at every position across strings. The letters are A, T, G, C</p> <pre><code>TGAGGTAGTAGTTTGTGCTGTTAT TAGTAGTTTGTGCTGTTA TGAGGTAGTAGTTTGTAC TGAGAACTGAATTCCATAGG </code></pre> <p>desired output:</p> <pre><code> Pos1 Pos2 Pos3 and so on. A 0 1 T 4 0 C 0 0 G 0 3 </code></pre> <p>So far I have used an R package called Biostrings, which works, but I wonder if perl would do this?</p>
<p>For the record, for</p> <pre><code>x = "TGAGGTAGTAGTTTGTGCTGTTAT TAGTAGTTTGTGCTGTTA TGAGGTAGTAGTTTGTAC TGAGAACTGAATTCCATAGG" </code></pre> <p>a <a href="http://bioconductor.org/packages/release/bioc/html/Biostrings.html">Biostrings</a> solution is</p> <pre><code>library(Biostrings) consensusMatrix(DNAStringSet(strsplit(x, "\n")[[1]])) </code></pre> <p>which will be fast for millions of sequences.</p>
