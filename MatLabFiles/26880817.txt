<p>There is no definite answer to your question, it is too broad and mainly belongs to statistics. Before doing any computation you should answer some questions:</p> <ul> <li>is there a specific distribution type which the data follow?</li> <li>is there any theoretical justification to select a distribution type and discard others?</li> <li>do I need parametric or non-parametric distribution?</li> <li>if no specific distribution type can be selected than what set of distributions should I investigate?</li> <li>how to compare the distributions, <a href="http://en.wikipedia.org/wiki/Goodness_of_fit" rel="nofollow noreferrer">goodness-of-fit measures</a>?</li> <li>what fitting method should I use, e.g. <a href="http://en.wikipedia.org/wiki/Maximum_likelihood" rel="nofollow noreferrer">max-likelihood</a>, <a href="http://en.wikipedia.org/wiki/Method_of_moments_(statistics)" rel="nofollow noreferrer">method of moments</a>, <a href="http://en.wikipedia.org/wiki/Bayesian_inference" rel="nofollow noreferrer">Bayesian</a>, etc.?</li> <li>how to treat uncertainties?</li> <li>how and for what want I use the results?</li> <li>etc.</li> </ul> <p>Without answering these question it is meaningless to talk about fitting distribution to data. I give you an example how to do the fit in Matlab using maximum-likelihood method, just for illustration, but I would strongly discourage you to use it without considering the above points.</p> <p>Since I have no additional background information in respect of the nature of the data, normal and <a href="http://en.wikipedia.org/wiki/Kernel_density_estimation" rel="nofollow noreferrer">kernel</a> distributions are fitted to illustrate 1 parametric and 1 non-parametric distribution.</p> <pre><code>cdfplot(y) hold on xx = -20:40; %normal distribution pd_norm = fitdist(y', 'normal'); F_norm = normcdf(xx, pd_norm.mu, pd_norm.sigma); plot(xx, F_norm, 'r') %kernel distribution pd_kernel1 = fitdist(y', 'kernel', 'Kernel', 'normal', 'Width', 6); F_kernel1 = cdf(pd_kernel1, xx); plot(xx, F_kernel1, 'g') %kernel distribution pd_kernel2 = fitdist(y', 'kernel', 'Kernel', 'normal', 'Width', 2); F_kernel2 = cdf(pd_kernel2, xx); plot(xx, F_kernel2, 'black') legend('ecdf', 'normal', 'kernel1', 'kernel2', 'Location', 'NorthWest') </code></pre> <p><img src="https://i.stack.imgur.com/vXX8k.png" alt="enter image description here"></p>
