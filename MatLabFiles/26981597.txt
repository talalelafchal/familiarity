Distinguish 'inf' and '-inf'
<p>How can I separate <code>inf</code> and <code>-inf</code> from each other?</p> <p>I have matrix that contains <code>inf</code> and <code>-inf</code> and I want to replace them with <code>1e6</code> and <code>-1e6</code> respectively.</p> <p>for example,</p> <pre><code>a = [1 2 3 0 3 4]./[1 1 1 0 1 0]; b = log2([0 2 1 2 1 2]); c = cat(1,a,b); </code></pre> <p>which is,</p> <pre><code>c = [ 1 2 3 NaN 3 Inf; -Inf 1 0 1 0 1]; </code></pre> <p>I want,</p> <pre><code>newc = [ 1 2 3 NaN 3 1e6; -1e6 1 0 1 0 1]; </code></pre> <p>I couldn't find a function that separates them.</p> <p>Thanks.</p>
<p>You can detect <code>inf</code> using <code>isinf()</code>, and you can detect sign using <code>sign()</code>. Combine the two:</p> <pre><code>newc = c; inf_filter = isinf(newc); newc(inf_filter) = 1e6 * sign(newc(inf_filter)); </code></pre>
