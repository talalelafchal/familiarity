Elegant way of stripping the 1st layer of cell array in MATLAB?
<p>I have a <code>1x2</code> cell array a such that</p> <ul> <li><code>a{1, 1}</code> is a <code>5x1</code> int array containing <code>[1 2 3 4 5]</code></li> <li><code>a{1, 2}</code> is a <code>5x1</code> cell array containing <code>'aa', 'bb', 'cc', 'dd', 'ee'</code></li> </ul> <p>What is the most elegant way of stripping the first layer, producing a <code>5x2</code> cell array as follows?</p> <pre><code>1 'aa' 2 'bb' 3 'cc' 4 'dd' 5 'ee' </code></pre>
<p>You can try this code:</p> <pre><code>a{1, 1} = [1,2,3,4,5]; a{1, 2} = {'aa','bb','cc','dd','ee'}; temp = num2cell(a{1}); b = {temp{:};a{2}{:}}.' b = [1] 'aa' [2] 'bb' [3] 'cc' [4] 'dd' [5] 'ee' </code></pre>
<p>How about:</p> <pre><code>% original cell a = cell(1,2); a{1} = [1 2 3 4 5]; a{2} = {'aa', 'bb', 'cc', 'dd', 'ee'}; % flattened aa = reshape([num2cell(a{1}) a{2}], [], 2) </code></pre>
<p>I figured one solution out, but am not sure about its "elegantness".</p> <pre><code>a = cell(1, 2); a{1, 1} = [1, 2, 3, 4, 5]; a{1, 2} = {'aa','bb','cc','dd','ee'}; result = [num2cell(a{1, 1})' a{1, 2}'] result = [1] 'aa' [2] 'bb' [3] 'cc' [4] 'dd' [5] 'ee' </code></pre>
