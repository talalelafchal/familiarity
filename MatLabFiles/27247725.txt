<p>You need to specify a guess, <em>x0</em> point</p> <pre><code>fun = @sin; % function x0 = 3; % initial point x = fzero(fun,x0) </code></pre>
<p>There are several mistakes in your code. For a start, <code>fzero</code> is for finding numerical roots of a non-linear equation, it is not for symbolic computations (check the <a href="http://uk.mathworks.com/help/matlab/ref/fzero.html" rel="nofollow">documentation</a>), so get rid of <code>syms x</code>. The correct way to call <code>fzero</code> in your case is a as follows:</p> <pre><code>A = 5; B = 6; C = 10; eq = @(x) A*x^2 + B*x + C*(asinh(x)); x0 = 0; % or whatever starting point you want to specify x = fzero(eq,x0) </code></pre>
