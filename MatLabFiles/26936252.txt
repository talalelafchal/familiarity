how can I load file.txt in matlab?
<p>I have a folder contained several files in it loc1.txt loc2.txt .... loc10.txt I want to use them in matlab this is my code :</p> <pre><code>for i=1:10 myFile =['E:\dis\locs\loc' i '.txt']; b= importdata(myFile); </code></pre> <p>but it does not work and output is like this : <code>'E:\dis\locs\loc .txt'</code> is there any body help me here? </p>
<p>You need to convert <code>i</code> into characters.</p> <pre><code>myFile =['E:\dis\locs\loc' num2str(i) '.txt']; </code></pre>
<p>Nemesis' answer using <a href="http://www.mathworks.com/help/matlab/ref/num2str.html" rel="nofollow"><code>num2str</code></a> is correct. Another possibility is to use <a href="http://www.mathworks.com/help/matlab/ref/sprintf.html" rel="nofollow"><code>sprintf</code></a>:</p> <pre><code>myFile = sprintf('E:\dis\locs\loc%d.txt', i); </code></pre> <p>The interface is less intuitive if you have never seen it before but it is also very convenient when you have zero-padded numbers, like <code>loc0001.txt</code>, <code>loc0002.txt</code>, etc. In this case just replace <code>%d</code> by <code>%04d</code> like this:</p> <pre><code>myFile = sprintf('E:\dis\locs\loc%04d.txt', i); </code></pre>
