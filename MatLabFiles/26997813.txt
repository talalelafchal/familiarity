<p>You can access MATLAB from Visual Studio and Eclipse through the MATLAB engine interface. As long as you MATLAB installed you just need to set the right paths in your project settings and link in 3 library files. </p> <p>Here is a link to the description and setup.<br> <a href="http://www.mathworks.com/help/matlab/matlab_external/introducing-matlab-engine.html" rel="nofollow">http://www.mathworks.com/help/matlab/matlab_external/introducing-matlab-engine.html</a> <a href="http://www.mathworks.com/matlabcentral/answers/100603-how-can-i-compile-a-matlab-engine-application-using-microsoft-visual-studio-9-0-or-10-0" rel="nofollow">http://www.mathworks.com/matlabcentral/answers/100603-how-can-i-compile-a-matlab-engine-application-using-microsoft-visual-studio-9-0-or-10-0</a></p> <p>Once you link Visual Studio / Eclipse to MATLAB you can do call MATLAB functions, plots, scripts, and pass variables back and forth (see below):</p> <p>engEvalString(ep, "cd c:/MATLAB_Demos"); // Change to active directory<br> engEvalString(ep, "img = imread('image_2.jpg');"); // Read in image to process engEvalString(ep, "Igray = rgb2gray(img);"); // Convert to grayscale engEvalString(ep, "figure; imshow(Igray);"); // Display image to verify engEvalString(ep, "Thresh_Fn(1)"); // Call MATLAB scripts/functions and pass arguments engPutVariable(ep, "output", output);</p> <p>Igray = engGetVariable(ep, "Igray"); // Pointer to MATLAB variable const uint8_t* Ip = static_cast(mxGetData(Igray)); // Pointer in C variable</p> <p>Hope that helps,</p> <p>Andy</p> <p>Image Processing Applications - MathWorks</p>
