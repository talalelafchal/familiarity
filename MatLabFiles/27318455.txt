How to define a new function from a matrix function?
<p>It is a very simple problem, though I do not know how to deal with it. </p> <p>I defined a function <code>f(x,y)</code> which will return a, say, 2*2 matrix. I want to define a new function <code>g</code> whose value is the (1,1) element of <code>f(x,y)</code>. I naively tried <code>g=@(x,y)(f(x,y))(1,1)</code>, which of course failed . Please help me!</p>
<p>You can use :)</p> <pre><code>g = @(x,y)([1 0] * f(x,y) * [1 0]') </code></pre>
<p>You can use the function <a href="http://uk.mathworks.com/help/matlab/ref/getfield.html" rel="nofollow"><code>getfield</code></a></p> <p>Let's define <code>f(x,y)</code> to return a 2x2 matrix</p> <pre><code>f = @(x,y) [1*x 2*y;3*x 4*y].^2 ; </code></pre> <p>Then let's define a function <code>g11(x,y)</code> which return the element {1,1} of <code>f(x,y)</code>. (and an extra function <code>g21</code> which return the {2,1} element)</p> <pre><code>g11 = @(x,y) getfield( f(x,y) , {1,1} ) ; g21 = @(x,y) getfield( f(x,y) , {2,1} ) ; </code></pre> <p>And now:</p> <pre><code>&gt;&gt; f(3,4) ans = 9 64 81 256 &gt;&gt; g11(3,4) ans = 9 &gt;&gt; g21(3,4) ans = 81 </code></pre> <hr> <p><em>For more fancy usage of field assignment without temporary variable, read all the nice answers in <a href="http://stackoverflow.com/questions/3627107/how-can-i-index-a-matlab-array-returned-by-a-function-without-first-assigning-it">this question</a></em></p>
