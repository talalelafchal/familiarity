Replace values in a submatrix in Julia
<p>Is there an efficient way of coding other than what I did below while converting matlab code to Julia? Especially when the <code>parent_matrix</code> is of the size <code>2000X2000</code> and <code>inner_matrix</code> is of size <code>800X1</code>?</p> <p>Matlab: </p> <pre><code>parent_matrix(inner_matrix(:),inner_matrix(:)) = replace_matrix; </code></pre> <p>Julia:</p> <pre><code>ind_inner_vec=reshape(inner_matrix,size(inner_matrix)[1].*size(inner_matrix)[2],1) z=1 for x in ind_inner_vec for y in ind_inner_vec parent_matrix[y,x]=replace_matrix[z] z=z+1 end end </code></pre>
<p>If I understand correctly</p> <pre><code>parent_matrix[vec(inner_matrix),vec(inner_matrix)] = replace_matrix </code></pre> <p>will do what you want. Note that the <code>vec</code> is only needed because you said <code>inner_matrix</code> is a column matrix - if it was actually a vector, it wouldn't be needed.</p>
