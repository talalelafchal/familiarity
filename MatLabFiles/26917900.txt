rounding to n significant digits
<p>I'm trying to write a code in matlab that will round number to certain (as I ask) significant digits.</p> <p>I'm not sure how to do it. any suggestions? </p>
<p>To round to <code>d</code> <strong>significant digits</strong>:</p> <pre><code>&gt;&gt; d = 3; %// number of digits &gt;&gt; x = 5.237234; %// example number &gt;&gt; D = 10^(d-ceil(log10(x))); &gt;&gt; y = round(x*D)/D y = 5.2400 </code></pre> <p>To round to <code>d</code> <strong>decimal digits</strong>:</p> <pre><code>&gt;&gt; d = 3; %// number of digits &gt;&gt; x = 5.237234; %// example number &gt;&gt; D = 10^d; &gt;&gt; y = round(x*D)/D y = 5.2370 </code></pre> <p><strong>EDIT</strong></p> <p>Actually it's easier: the <a href="http://es.mathworks.com/help/matlab/ref/round.html" rel="nofollow"><code>round</code></a> function supports these options:</p> <pre><code>&gt;&gt; d = 3; &gt;&gt; x = 5.237234; &gt;&gt; y = round(x, d, 'significant') y = 5.2400 &gt;&gt; d = 3; &gt;&gt; x = 5.237234; &gt;&gt; y = round(x, d) %// or y = round(x, d, 'decimals') y = 5.2370 </code></pre>
