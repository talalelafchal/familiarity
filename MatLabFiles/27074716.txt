How can I decompose an image into 8 x 8 pixel blocks?
<pre><code> I=imread('X.jpg'); [a b]=size(I); figure;imshow(I); j=rgb2ycbcr(I); figure; imshow(j); ca=mat2cell(j,8*ones(1,size(j,1)/8),8*ones(1,size(j,2)/8),3); p=1; for c=1:size(ca,1) for r=1:size(ca,2) subplot(8,8,p); imshow(ca{c,r}); p=p+1; end end </code></pre> <p>i get the following error: Index exceeds number of subplots. Any thoughts?</p>
<p>The reason why is because you have more image blocks to show than you have subplot spaces when plotting. Change your <code>for</code> loop code to this instead:</p> <pre><code>p = 1; figure; for c=1:size(ca,1) for r=1:size(ca,2) subplot(size(ca,1),size(ca,2),p); %// Change imshow(ca{c,r}); p=p+1; end end </code></pre> <p>This way, you will have as many subplot spaces to plot as you have pixel blocks to show.</p>
