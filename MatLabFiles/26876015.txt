Using matlab FCM to cluster my own data
<p>I am trying to use fcm (fuzzy C-means clustering) matlab tool, but I don't know how to put my own data. I am trying to cluster nodes based on distance from the center. So my data are x and y coordinates. I am basically trying to compare it with k-means this is how I did the k-means:</p> <pre><code>X=[x_users,y_users]; nc=20; idx = kmeans(X,nc); </code></pre> <p>I need to know how to do the same thing with fcm, am sorry if my question is too naive.</p> <p>Thanks, </p>
<pre><code>fcm(X,nc); </code></pre> <p>will do it. 'marker', '*','color','r'); </code></pre> <p><img src="https://i.stack.imgur.com/hU7YR.png" alt="enter image description here"></p> <p>Using <code>kmeans</code> the answer would be like the following plot:</p> <pre><code>idx = kmeans(data,nc); data1 = data(idx==1,:); data2 = data(idx==2,:); figure;plot(data1(:,1),data1(:,2),'x'); hold on;plot(data2(:,1),data2(:,2),'or'); </code></pre> <p><img src="https://i.stack.imgur.com/YCOOX.png" alt="enter image description here"></p>
