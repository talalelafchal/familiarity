How to calculate MATLAB linspace function by hand
<p>There is a function named linspace in matlab and splits interval in given range. For example:</p> <pre><code>&gt;&gt; x = linspace(-10,5, 10) x = -10.0000 -8.3333 -6.6667 -5.0000 -3.3333 -1.6667 0 1.6667 3.3333 5.0000 </code></pre> <p>How can I find x(4) by doing calculations by hand ? </p>
<p>This seems to work -</p> <pre><code>x = linspace(-10,5, 10) start = -10; stop = 5; num_elements = 10; index = 4; out = start + (index-1)*(stop - start)./(num_elements-1) </code></pre> <p>Output -</p> <pre><code>x = -10.0000 -8.3333 -6.6667 -5.0000 -3.3333 -1.6667 0 1.6667 ... out = -5 </code></pre> <hr> <p>Thus, <code>(stop - start)./(num_elements-1)</code> would be the <code>stepsize</code>. </p> <p>So, if you want the complete array, do this -</p> <pre><code>complete_array = start : (stop - start)./(num_elements-1) :stop </code></pre> <p>But, be careful of the floating point precision issues if you are comparing these results against the <code>linspace</code> results - <a href="http://stackoverflow.com/questions/26292695/what-is-the-advantage-of-linspace-over-the-colon-operator"><code>What is the advantage of linspace over the colon “:” operator?</code></a>.</p>
