Is is possible to suppress 'ans' in MATLAB?
<p>I have a function, but I do not want the <code>ans = ...</code> in the end. Is it possible to suppress it?</p> <p>For instance:</p> <pre><code>function c = Example (b) c = b + 2; end </code></pre> <p>On the command window:</p> <pre><code>&gt;&gt;Example (8) &gt;&gt;ans = 10 </code></pre>
<p>Put a semicolon at the end of the line where the function is called. Here's the bit of documentation that covers this.</p> <p>From the <a href="http://www.mathworks.com/help/matlab/matlab_prog/symbol-reference.html#bsgigzp-46" rel="nofollow">symbol reference</a>:</p> <blockquote> <p><strong>Output Suppression</strong></p> <p>When placed at the end of a command, the semicolon tells MATLAB not to display any output from that command. In this example, MATLAB does not display the resulting 100-by-100 matrix:</p> <pre><code>A = ones(100, 100); </code></pre> </blockquote> <p>This pertains to display. AFAIK, there's no way to prevent MATLAB from still assigning to <code>ans</code> (i.e. <code>rand;</code> still assigns the result to <code>ans</code>).</p>
<p>Further to @chappjc's answer, it <em>is</em> possible to suppress the creation of <code>ans</code> - but it may not be all that desirable. The trick is to assign the outputs only if <code>nargout</code> is non-zero. For example:</p> <pre><code>function varargout = Example(b) c = b + 2; if nargout &gt; 0 varargout = {c}; end end </code></pre>
