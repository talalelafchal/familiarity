How to combine the data of two cell arrays?
<p>I have two cell arrays <code>cellA</code> and <code>cellB</code> each having 1060 elements.The size of each cell within cell array <code>cellA</code> is <code>870*1</code> and the same for <code>cellB</code> is <code>16*1</code>. I want to combine these two into a third cell array <code>cellC</code>. Not combine the cells themselves but the data of each, such that after combining their size becomes <code>886*1</code>.</p> <p>Ive tried these commands but they combine the cells and not the data within each</p> <pre><code>cellC = [cellA cellB]; cellC = [{cellA} {cellB}]; </code></pre>
<p>Convert each cell array to a matrix by concatenating cells horizontally; then concatenate those matrices vertically; and convert back to a cell array of columns:</p> <pre><code>cellC = mat2cell([[cellA{:}]; [cellB{:}]], numel(cellA{1})+numel(cellB{1}), [1 1]); </code></pre> <p>Example:</p> <pre><code>&gt;&gt; cellA = {(1:4).', (11:14).'}; &gt;&gt; cellB = {(101:103).', (111:113).'}; &gt;&gt; cellC = mat2cell([[cellA{:}]; [cellB{:}]], numel(cellA{1})+numel(cellB{1}), [1 1]) cellC = [7x1 double] [7x1 double] &gt;&gt; cellC{:} ans = 1 2 3 4 101 102 103 ans = 11 12 13 14 111 112 113 </code></pre>
<p>Try this:</p> <pre><code>cellC=cellfun(@(x,y) [x y], cellA, cellB, 'UniformOutput', false); </code></pre>
<p>I just do something simple like:</p> <pre><code>cellC = {cellA{:} cellB{:}}; </code></pre> <p>This will result in a cell array that is 886 long. From your question, I'm not sure if you wanted a regular array (which is what the other answers above will give) or a cell array. If you're Ok with getting a cell array, this is a pretty simple way of doing it.</p>
