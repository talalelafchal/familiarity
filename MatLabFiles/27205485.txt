Superscript in figure caption in bar graphs
<p>I have this code and I want to add Superscript to each bar graph.</p> <pre><code>y=[0.3435 0.36 0.6491 0.6908]; bar(y); Labels={'A_{1}','A_{2}','A_{3}','A_{4}'}; set(gca, 'XTick', 1:4, 'XTickLabel', Labels); set(gca,'fontsize',10); ylabel('Size','FontSize',10) axis([0 5 0 1]) </code></pre> <p>but the subscripts are not applied in the result:(</p> <p><img src="https://i.stack.imgur.com/Z32vM.jpg" alt="enter image description here"></p>
<p>To get subscripts you can use the LaTeX interpreter. But I don't think the <code>'xticklabel'</code> property supports TeX or LaTeX. So: remove your <code>xticklabel</code>s and create <code>text</code> objects, which does support TeX. To create all text objects at once, you can pass arrays as inputs to the <a href="http://es.mathworks.com/help/matlab/ref/text.html" rel="nofollow noreferrer"><code>text</code></a> function:</p> <pre><code>y=[0.3435 0.36 0.6491 0.6908]; bar(y); Labels={'A_{1}','A_{2}','A_{3}','A_{4}'}; set(gca, 'XTick', 1:4, 'XTickLabel', []); %// no ticklabels set(gca,'fontsize',10); ylabel('Size','FontSize',10) axis([0 5 0 1]) text(1:numel(y), -.05*ones(1,numel(y)), Labels, 'interpreter', 'TeX') %// desired text. Interpreted in TeX. Adjust position changing -0.05 if needed </code></pre> <p><img src="https://i.stack.imgur.com/7XgOB.png" alt="enter image description here"></p> <p>If you want superscripts instead of subscripts, use</p> <pre><code>Labels={'A^{1}','A^{2}','A^{3}','A^{4}'}; </code></pre>
