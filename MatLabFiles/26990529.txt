How to write a inline function which will accept two arguments in MATLAB
<p>I want to write an inline function which will accept two arguments in which one argument is a vector.</p> <pre><code>&gt;&gt;nCk = @(n,k)(nchoosek(n,k)); &gt;&gt;nCk(3,1:2) Error using nchoosek (line 29) The second input has to be a non-negative integer. </code></pre> <p>How can I make the second argument accepts a vector.</p>
<p>While its probably not what you want, I think this is one situation where I would use a for loop, as nchoosek only accepts an integer for its <code>k</code> value:</p> <pre><code>nCk = @(n,k)(nchoosek(n,k)); n = 3; for k = 1:2 disp(nCk(n,k)); end </code></pre> <p>Though if you do it this way, then the inline statement is likely redundant, so it could be reduced to:</p> <pre><code>n = 3; for k = 1:2 disp(nchoosek(n,k)); end </code></pre>
<p>From <a href="http://fr.mathworks.com/help/matlab/ref/nchoosek.html" rel="nofollow">here</a> ,you can see that you can have as vector the first argument and not the second.</p> <p>So ,you can use:</p> <pre><code>nCk = @(k,n)(nchoosek(k,n)); </code></pre> <p>If that helps you of course</p>
<p>As mentioned, <code>nchoosek</code> only allows integer inputs for the second argument. If you do want to make an inline function, you can fold the loop into a call to <code>arrayfun</code>, however:</p> <pre><code>nCk = @(n,kVec)arrayfun(@(k)nchoosek(n,k),kVec); </code></pre> <p>And use like this:</p> <pre><code>nCk(5,0:5) ans = 1 5 10 10 5 1 </code></pre>
