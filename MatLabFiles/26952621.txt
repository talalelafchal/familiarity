Convert array to string in MATLAB?
<p>I have a vector <code>a = [1 2 3 4 5]</code> how would I make it so b = '12345'?</p> <p>I have tried <code>b = num2str(a)</code> but it outputs <code>1 2 3 4 5</code>.</p>
<p>You can specify the format in <a href="http://es.mathworks.com/help/matlab/ref/num2str.html" rel="nofollow"><code>num2str</code></a>, much as you would in C's function <code>sprintf</code>:</p> <pre><code>b = num2str(a,'%i'); </code></pre> <p>Or use <a href="http://es.mathworks.com/help/matlab/ref/sprintf.html" rel="nofollow"><code>sprintf</code></a>:</p> <pre><code>b = sprintf('%i',a); </code></pre> <p>If <code>a</code> contains only single-digit numbers, you can also convert to <a href="http://es.mathworks.com/help/matlab/ref/char.html" rel="nofollow">char</a> directly:</p> <pre><code>b = char(a+'0'); </code></pre>
<p>You need to convert yor vector to a single number first (assuming <strong>all</strong> elements are in range 0..9):</p> <pre><code>a = 1:5; num = ( 10.^((numel(a)-1):-1:0) ) * a'; %' b = num2str( num ) </code></pre> <p>You can try this code <a href="http://ideone.com/s3uSoR" rel="nofollow">here</a>.</p>
