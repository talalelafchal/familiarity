plotting a text file with 4 columns in matlab
<p>I want to plot a text file with 4 columns that first column in longitude,second in latitude, third is depth and forth is amount of displacement in each point.(it's related to a fualt)</p> <p>-114.903874 41.207504 1.446784 2.323745</p> <p>I want a plot to show the amount of displacement in each point (like images that we plot with imagesc),unfortunately "imagesc" command doesn't work for it.</p> <p>how can I plot it? Thanks for your attention</p>
<p>You can plot each point using <code>plot3(longitude,latitude,depth)</code>. You can color each point according to the displacement in a <code>for</code> loop. The easiest way to do this is create a colormap, e.g. using <code>jet</code> and chosing the color according to the displacement.</p> <pre><code>figure; hold on; cmap = jet(256); dispRange = [min(displacement),max(displacement)]; for k=1:size(longitude,2) c = cmap(1+round(size(cmap,1)*(displacement(k)-dispRange(1))/dispRange(2)),:); plot3(longitude(k),latitude(k),depth(k),'o', ... 'MarkerEdgeColor',c,'MarkerFaceColor',c); end </code></pre>
<p>A simple way would be to use <a href="http://uk.mathworks.com/help/matlab/ref/scatter3.html" rel="nofollow">scatter3</a> and assign your displacements to be the colours. Note that you have to supply a size for this to work - I'm using <code>[]</code> (empty matrix) which will set it to default. If your four sets of values are four vectors of the same size, then it's just something like:</p> <pre><code>scatter3(lat,lon,depth,[],displacement, 'filled') </code></pre> <p>Values in <code>displacement</code> will be linearly mapped to the current colormap. <code>'filled'</code> gives you filled markers rather than open ones (default marker is a circle but can be changed).</p>
