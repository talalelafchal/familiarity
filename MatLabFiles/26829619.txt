How to use udp() in MATLAB?
<p>I tried to use <strong><code>udp()</code></strong> in MATLAB with this code:</p> <pre><code> u1 = udp('192.168.6.207', 8878, 'LocalPort', 8840); u2 = udp('192.168.6.207', 8840, 'LocalPort', 8878); fopen( u1 ); fopen( u2 ); a =[ char( 240 ) ]; fprintf( u1, a ); recStr = double( fscanf( u2 ) ); pause( .1 ); fclose( u1 ); delete( u1 ); clear u1; fclose( u2 ); delete( u2 ); clear u2; </code></pre> <p>But I get <strong><code>63</code></strong> instead of <strong><code>240</code></strong>.</p> <p><strong>How can I fix this?</strong></p> <p>Any idea about my problem?</p> <p>Thanks Hamid</p>
<p>You are using <code>fprintf</code> and <code>fscanf</code> wrong.<br> You should do something like this:</p> <pre><code>a = 240; fprintf( u1,'%d', a ); recStr =fscanf(u2,'%d'); </code></pre> <p>Or, if you want to transfer only numerical values, you should use <code>fwrite</code> and <code>fread</code>:</p> <pre><code>a = 240; fwrite(u1,a); recStr =fread(u2,1); </code></pre>
