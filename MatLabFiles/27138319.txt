Reading string and numbers from text file in MATLAB
<p>My text file format is something like this:</p> <p>X Y</p> <p>1.2 3.2</p> <p>1.0 4</p> <p>0.9 10</p> <p>I want to grab only the numbers in two columns. So far I have tried load, textread, it gives error with first line of the txt file. How can I ignore the first string line and grab only the numbers in file?</p> <p>Thanks!</p>
<p>Another option is to use <code>textread</code>, which has the option <code>'headerlines'</code>. You'd call it like</p> <pre><code>[A,B]=textread('textfile,txt','%f %f','headerlines',1); %this ignores the first line </code></pre>
<p>Try the <a href="http://www.mathworks.com/help/matlab/ref/importdata.html" rel="nofollow">importdata</a> function.</p> <pre><code>A = importdata ('textfile.txt', ' ', 1); data = A.data </code></pre> <p>The parameters: 'textfile' is the name of the file that contains your data, ' ' is the delimiter, 1 tells the function that there is one header line</p> <p>With <code>A.colheaders{1}</code> and <code>A.colheaders{2}</code> you even can access the names of you columns.</p>
