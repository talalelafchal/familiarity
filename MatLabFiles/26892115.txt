Conditional elements deleting from a matrix
<p>I have matrix, 40x20. I want to delete all the elements less than lets say <code>a</code>.</p> <p>I know how to do it for a column or row using logical indexing:</p> <pre><code>idxToKeep = yourMat(1, :) &lt;= a; newMat = yourMat(:, idxToKeep); </code></pre> <p>But what if the <code>idxToKeep</code> I get is also a matrix. I can imagine how to do it using loops, but I don't want to use loop.</p> <p>Thanks for your time</p>
<p>Is this what you need? </p> <pre><code>idxToKeep=yourMat&lt;=a; values=yourMat(idxToKeep); </code></pre> <p>Off course you can do it in one line if you wish:</p> <pre><code>values=yourMat(yourMat&lt;=a); </code></pre> <p>And if you want the output in a matrix format:</p> <pre><code>newMat=yourMat; newMat(newMat&gt;a)=NaN; %or 0 </code></pre>
