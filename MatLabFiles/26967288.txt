How to find the index of dependent column in sparse matrix
<p>I have a sparse matrix such as</p> <pre><code>A=sparse(zeros(10,20))% 10 rows by 20 columns </code></pre> <p>Now, for each iteration, I will put random value for each column</p> <pre><code>for i=1:20 %% insert value - it is done end </code></pre> <p>My task is that find the index of dependent column and replate it by zeros values for all element in the dependent column. For example, the full matrix A can assume as</p> <pre><code>full(A)= [ 1 1 1 0 0 1 1 1 1 ] </code></pre> <p>We see that the second column is a dependent row with first column. So my expected result such as</p> <pre><code> full(A)= [ 1 0 1 0 0 1 1 0 1 ] </code></pre> <p>Could you help me resolve this problem?</p>
<p>I don't know why you want discuss this in the sparse context, but apart from simply checking each dependency you could do it simultaneously. I suggest something like</p> <pre><code>%% create some data... A = double(rand(3,3) &gt; 0.5) %% create linearDependencyMatrix with % important % linearDependencyMatrix(i,j) == 1 % part % iff % % column i and column j are linearly dependent % AA = A'*A; % normCols = diag(AA); % normMat = normCols*normCols'; % linearDependencyMatrix = abs(AA.^2-normMat) &lt; 10*eps % %% set linearly dependent columns to 0 nonzeroCols = find(sum(abs(A)) ~= 0); numCols = size(A,2); for i = nonzeroCols for j = i+1:numCols if linearDependencyMatrix(i,j) A(:,j) = 0; end end end A </code></pre> <p><strong>Example Output</strong></p> <pre><code>A = 1 0 0 1 0 0 1 1 1 linearDependencyMatrix = 1 0 0 0 1 1 0 1 1 A = 1 0 0 1 0 0 1 1 0 </code></pre>
