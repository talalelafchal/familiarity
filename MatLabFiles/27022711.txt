Generate numbers randomly from a set?
<p>In MATALB, I have a set of <code>P</code> numbers. I would like to generate a random array of size <code>N</code> from this set.</p> <p>For the sake of example, let say I have the set <code>{1, 4}</code>. Let say I would like to generate an array of size <code>5</code> (e.g., <code>[1 1 4 1 4]</code>). </p> <p>What I did is this: I generated the following array using <code>randi</code>. </p> <pre><code>N = 5; v = randi([1 4],[1 N]); </code></pre> <p>The problem is that I got a random array which contains values in <code>1:4</code> and not in <code>{1, 4}</code>. I can simply do this but I need a better way. </p> <pre><code>for i = 1:length(v) if v(i) ~= 1 || v(i) ~= 4 v(i) = 1; % or v(i) = 4 end end </code></pre> <p>I think I am missing a simple hint here. Thanks for your time.</p>
<p>You should use <code>datasample</code>,</p> <p><code>y = datasample(data,k)</code> returns <code>k</code> observations sampled uniformly at random, with replacement, from the data in <code>data</code>. </p> <pre><code>a = [1,4]; datasample(a,5) </code></pre> <p>Depending on the usage, you might consider using,</p> <pre><code>datasample(unique(a),5) </code></pre>
<p>If you don't have the Statistics Toolbox (which contains the <code>datasample</code> function), you can use <a href="http://es.mathworks.com/help/matlab/ref/randi.html" rel="nofollow"><code>randi</code></a>:</p> <pre><code>N = 5; %// desired number of samples data = [1 4]; %// data values sample = data(randi(numel(data),1,N)); </code></pre> <p>And if you use a very old version of Matlab that doesn't have <code>randi</code>, you can employ <a href="http://es.mathworks.com/help/matlab/ref/rand.html" rel="nofollow"><code>rand</code></a>:</p> <pre><code>sample = data(ceil(numel(data)*rand(1,N))); </code></pre>
