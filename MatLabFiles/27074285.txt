<p>If all you are wanting to do is plot that function, then something along the lines of this should suffice:</p> <pre><code>s = -10:.5:10; G = 1 .* (exp(1)-0.5.*s) ./ (s+1) .* (s+5).^2; plot(s,G) </code></pre> <p><img src="https://i.stack.imgur.com/smHTY.png" alt="Plotted Graph"></p>
<p>You are mixing things very badly. You can't make <code>s</code> syms and then make transfer function using it. Also, you do not <code>plot</code> a transfer function. You should use Bode on it to show the frequency and the magnitude of the transfer function for that. Just forget about syms all together when using Matlab control systems.</p> <pre><code>s=tf('s'); sys=exp(-0.5*s)/((s+1)*(s+5)^2) sys = 1 exp(-0.5*s) * ------------------------ s^3 + 11 s^2 + 35 s + 25 Continuous-time transfer function. sysf= feedback (sys,1); step(sysf) </code></pre> <p><img src="https://i.stack.imgur.com/CNCiU.png" alt="Mathematica graphics"></p>
<p>You can use <code>ezplot</code> too if you want to plot <code>G(s)</code> where:</p> <pre><code>G(s)=exp(-0.5*s)/((s+1)*((s+5)^2)); </code></pre> <p>You can use <code>ezplot</code> as follows:</p> <pre><code>ezplot('exp(-0.5*s)/((s+1)*((s+5)^2))'); </code></pre> <p>to generate the following image:</p> <p><img src="https://i.stack.imgur.com/5Hmst.jpg" alt="enter image description here"></p>
