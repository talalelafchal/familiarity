Elegant way of finding all integer element indices in a MATLAB array?
<p>I have a double array containing a mixture of integers and doubles. I wish to get all the indices of the integer elements. My way (see below) is quite ugly.</p> <pre><code>idx_list = []; for idx = 1:numel(A) if isinteger(A(idx)) idx_list = [idx_list idx]; end end </code></pre> <p>I believe there is a one liner / a more elegant way.</p>
<p>Something like this would find <em>perfect</em> integers -</p> <pre><code>idx_list = find( floor(A) == A ) </code></pre> <p>Changing <strong><a href="http://in.mathworks.com/help/matlab/ref/floor.html" rel="nofollow"><code>floor</code></a></strong> to <strong><a href="http://in.mathworks.com/help/matlab/ref/ceil.html" rel="nofollow"><code>ceil</code></a></strong> or <strong><a href="http://in.mathworks.com/help/matlab/ref/round.html" rel="nofollow"><code>round</code></a></strong> or <strong><a href="http://in.mathworks.com/help/matlab/ref/fix.html" rel="nofollow"><code>fix</code></a></strong> should do it too.</p> <p>If you want to take care of floating point precision issues, you can introduce <strong><a href="http://in.mathworks.com/help/matlab/ref/eps.html" rel="nofollow"><code>eps</code></a></strong> here -</p> <pre><code>idx_list = find(abs(round(A) - A)&lt;eps(round(A))) </code></pre>
