<p>Because your function (<code>h_a_b</code>) takes a vector as input and gives scalar as output it represents a surface, thus <code>plot</code> cannot be used to visualize it (that is only for 2D, scalar-scalar plots).</p> <p>Are you looking for something like this?:</p> <pre><code>f = @ (x) 5*sin(x); g = @ (x) 10*cos(x); h_a_b = @ (a,b) sqrt(a.^2 + b.^2); z = @(a,b) sqrt(h_a_b(f(a),g(b))); [A, B] = meshgrid(0:0.1:8, 0:0.1:9); Z = z(A,B); surfc(A,B,Z) xlabel('a') ylabel('b') figure contourf(A,B,Z) xlabel('a') ylabel('b') </code></pre> <p><img src="https://i.stack.imgur.com/dFzzh.png" alt="enter image description here"> <img src="https://i.stack.imgur.com/NfiDZ.png" alt="enter image description here"></p> <hr> <p><strong>Second option</strong>, considering <code>z</code> as scalar-scalar function and using your <code>plotfunc</code> function:</p> <pre><code>f = @ (x) 5*sin(x); g = @ (x) 10*cos(x); h_a_b = @ (a,b) sqrt(a.^2 + b.^2); z = @(x) sqrt(h_a_b(f(x),g(x))); points = 0:0.1:8; plotfunc(z,points) </code></pre> <p><img src="https://i.stack.imgur.com/vlLbt.png" alt="enter image description here"></p> <p>Which is one slice of the above surface.</p>
