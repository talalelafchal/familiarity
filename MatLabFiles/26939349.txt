How to draw a cone with given height and radius
<p>I have to draw a cone with a given height and radius at a specific co-ordinate.</p> <p>MATLAB has the function <code>cylinder(r)</code>, but it draws only a unit cylinder whereas I need it to be of a specific height.</p> <p>None of the other links specify how to draw a cone of height 'h'.</p>
<p>Matlab's <a href="http://es.mathworks.com/help/matlab/ref/cylinder.html" rel="nofollow noreferrer"><code>cylinder</code></a> can be used to draw cones by specifying a radius that decreases to 0. And height is just a <code>z</code> scaling factor. (Radius could also be realized as an <code>x</code>,<code>y</code> scaling factor, but the <code>cylinder</code> function permits directly specifying the radius values so that's not needed).</p> <pre><code>R = 1; %// radius H = 3; %// height N = 100; %// number of points to define the circumference [x, y, z] = cylinder([0 R], N); mesh(x, y, H*z) </code></pre> <p><img src="https://i.stack.imgur.com/90OH4.png" alt="enter image description here"></p>
<p>A cylinder at <code>[x y z]</code> with height <code>h</code>, radius <code>r</code>, </p> <pre><code>x = 10; y = 20; z = 5; h = 10; r = 1; [X,Y,Z] = cylinder(r,30); X = X + x; Y = Y + y; Z = Z*h + z; surf(X,Y,Z) </code></pre> <p><img src="https://i.stack.imgur.com/qR4ec.png" alt="enter image description here"></p>
