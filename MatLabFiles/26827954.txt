click coordinates, axes and imshow
<p>I'm having some problems with MATLAB GUIDE. </p> <p>I have an axes object in a GUI (axes1) which is displaying an image ( imshow(I) ). I need to store the coordinates of a user click on the image.</p> <p>I tried </p> <pre><code>[x,y] = ginput(1). </code></pre> <p>The problem with this function is that it returns the coordinates of the axes, not the coordinates of the image I. </p>
<p>The coordinates of the axes and the image are aligned:</p> <p>For example:</p> <pre><code>figure(); imshow('peppers.png'); [x,y]=ginput(1) </code></pre> <p>If you're bothered about out of bounds conditions, you could check and remove them after calling <code>ginput</code>:</p> <pre><code>if x&gt; size(im,2) x = size(im,2); end if y&gt; size(im,1) y = size(im,1); end </code></pre>
