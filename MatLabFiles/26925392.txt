How to insert infinity symbol to X axis of Matlab Bar graph?
<p>How can I add the infinity symbol to X axis of Matlab Bar graph?</p> <p>Naturally it is possible to insert the infinity symbol i.e. <code>'\infty'</code> for xlabel, as seen in the last line of the inserted code.<br> But, I want to add the infinity sign in the x axis bar not in the x axis label.<br> How can I do that? For the sake of detailed clarification, the following script is added bellow:</p> <pre><code>data=[1 2 3; 1 3 4; 3 1 2]; bar(data) set(gca,'YLim',[0 3]) set(gca,'YTick',[0:0.5:3]) set(gca, 'YTickLabel',num2str(get(gca,'YTick')','%02.1f%%')) set(gca,'Xtick',1:3,'XTickLabel',{'\infty' ; '20 dB'; '15 dB'}) xlabel('\infty dB') % x-axis label </code></pre> <p><img src="https://i.stack.imgur.com/OWSrY.jpg" alt="Image showing problem"></p>
<p>How about this solution, using <a href="http://www.mathworks.com/matlabcentral/fileexchange/23841-format-tick-labels--extended-" rel="nofollow noreferrer"><code>format_tick</code></a> function from File Exchange?:</p> <pre><code>data=[1 2 3; 1 3 4; 3 1 2]; bar(data) set(gca,'YLim',[0 3]) set(gca,'YTick',[0:0.5:3]) set(gca, 'YTickLabel',num2str(get(gca,'YTick')','%02.1f%%')) set(gca,'Xtick',1:3) format_ticks(gca, {'$\infty$' ; '20 dB'; '15 dB'}) </code></pre> <p><img src="https://i.stack.imgur.com/Pyhhs.png" alt="enter image description here"></p> <p>I left out the <code>xlabel</code> because it interfers with the <code>Xtick</code>, but probably that can be easily moved to lower position.</p> <p><strong>EDIT:</strong> To fix the overlap of <code>Xtick</code> and <code>xlabel</code> add this to the end of the code:</p> <pre><code>xlabh = get(gca,'XLabel'); set(xlabh,'Position',get(xlabh,'Position') - [0 .1 0]) </code></pre> <p><img src="https://i.stack.imgur.com/7ubxL.png" alt="enter image description here"></p>
