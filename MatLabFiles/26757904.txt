Check if a point is inside a polytope
<p>I am looking for a function in R (or matlab) or an algorithm that checks if a point belongs to a <a href="http://en.wikipedia.org/wiki/Polytope" rel="nofollow">polytope</a> which is define by its "triangular" faces.</p> <p>So far, I have found functions for 2D and 3D (i.e. for polygons and polyhedrons) in <a href="http://cran.r-project.org/web/packages/sp/index.html" rel="nofollow">sp</a> and <a href="http://cran.r-project.org/web/packages/ptinpoly/ptinpoly.pdf" rel="nofollow">ptinpoly</a> packages.</p>
<p>Try the <a href="http://people.ee.ethz.ch/~mpt/3/" rel="nofollow">MPT</a> toolbox for Matlab: it supports different ways of creating polyhedron and has a function to check if a point is inside the polyhedron ( function <code>contains</code> returns <code>1</code> if a point is inside).</p> <p>Short description is available <a href="http://control.ee.ethz.ch/~mpt/3/Geometry/OperationsWithPolyhedra#SetContainment" rel="nofollow">here</a> and <a href="http://infoscience.epfl.ch/record/186265/files/mpt3_ecc2013_final.pdf" rel="nofollow">here</a>.</p>
