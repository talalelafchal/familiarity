Retrieve first value from cell array of cells
<p>I have a cell array of cell arrays...</p> <pre><code> data = {1x16} {1x16} {1x16} </code></pre> <p>and I am trying to retrieve the first value from each of the cells.. However when I do </p> <pre><code> data{:}(1) </code></pre> <p>I end up with an error saying </p> <pre><code> Bad cell reference operation. </code></pre> <p>First of all is there a good reason as to why I shouldn't be allowed to do this? And secondly is there a way around this?</p> <p>Cheers! Ben</p>
<p>For a <code>1D</code> cell array -</p> <pre><code>first_vals = arrayfun(@(n) data{n}(1),1:numel(data)) </code></pre> <p>This should work for a general case -</p> <pre><code>first_vals = reshape(arrayfun(@(n) data{n}(1),1:numel(data)),size(data)) </code></pre>
<p>I would prefer doing it using a simple for loop :</p> <p>For instance, this would work :</p> <pre><code>data = [{[1:16]}; {[17:32]}; {[33:48]}]; b= [] for i=1:length(data) b = [b data{i,1}(1)]; end </code></pre>
