How can a function have multiple return values in Julia (vs. MATLAB)?
<p>In MATLAB, the following code returns <code>m</code> and <code>s</code>:</p> <pre><code>function [m,s] = stat(x) n = length(x); m = sum(x)/n; s = sqrt(sum((x-m).^2/n)); end </code></pre> <p>If I run the commands</p> <pre><code>values = [12.7, 45.4, 98.9, 26.6, 53.1]; [ave,stdev] = stat(values) </code></pre> <p>I get the following results: </p> <pre><code>ave = 47.3400 stdev = 29.4124 </code></pre> <p>How would I define my <code>stat</code> function in Julia?</p>
<blockquote> <p>How would I define my <code>stat</code> function in Julia?</p> </blockquote> <pre><code>function stat(x) n = length(x) m = sum(x)/n s = sqrt(sum((x-m).^2/n)) return m, s end </code></pre> <p>For more details, see the section entitled <a href="http://julia.readthedocs.org/en/latest/manual/functions/#multiple-return-values">Multiple Return Values</a> in the Julia documentation:</p> <blockquote> <p>In Julia, one returns a tuple of values to simulate returning multiple values.
