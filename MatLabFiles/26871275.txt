Save output to another directory when output was made with cell2csv
<p>I am using the FileExchange code <a href="http://www.mathworks.com/matlabcentral/fileexchange/7601-cell2csv" rel="nofollow">cell2csv</a> to save a cell in my code. I'd like the output file, the csv file, to be saved to a different directory. How can that be done?</p> <p>Here's my current code for saving the cell:</p> <pre><code>cell2csv([fname(13:16), '-', u_id{k}, '_24hrBlkAvg.csv'], site_data, '\t'); % cell2csv(filename,cellArray,delimiter) </code></pre>
<p>you can use <code>fullfile</code> to construct your path in a system-independent way</p> <pre><code>%// do not put everything in one line, the code should be readable to human: filename = [fname(13:16), '-', u_id{k}, '_24hrBlkAvg.csv'] filepath = fullfile(beetroot_folder, daughter_folder, subsubfolder, filename) cell2csv(filepath, site_data, '\t'); </code></pre> <p>if you need to save in a subfoder, use <code>'.'</code> for your current folder, or for saving in a sister folder use <code>'..'</code> for parent folder:</p> <pre><code>filepath = fullfile('.', filename) </code></pre>
