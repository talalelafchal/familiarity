I also want the both axes of each subplot to share the same range so that I can easily compare the graphs. Does anyone know how to do this? </p> <pre><code>figure() hold on subplot (1,2,1); % xlim([0 400]); % ylim([0 25]); graph_made = [num_calls_made]; plot (graph_made); title('Number of calls made') xlabel('ID Number of caller'); ylabel('Number of calls'); set(gca, 'XTick', [0:100:400]); set(gca, 'YTick', [0:5:20]); subplot (1,2,2); graph_rec = [num_calls_received]; plot (graph_rec); title('Number of calls received') xlabel('ID Number of caller'); ylabel('Number of calls'); set(gca, 'XTick', [0:100:400]); set(gca, 'YTick', [0:5:20]); hold off </code></pre>
<p><code>XTick</code> and <code>YTick</code> only change where the labels on axes go, not the limits of the axes. To change those, you have to use <code>axis</code> (or <code>xlim</code> and <code>ylim</code>):</p> <pre><code>axis([0 400 0 20]) %// [xmin xmax ymin ymax] </code></pre>
<p>If you want the axes limits to stay linked as a user interactively zooms or pans, you can also use the <code>linkaxes</code> command...</p> <pre><code>subplot(1,2,1) % your plotting code here... ax = gca; %get the handle to the current axis subplot(1,2,2) % your plotting code here... ax(end+1) = gca; %get the handle to the current axis linkaxes(ax); %this will link both the x and y axes. </code></pre>
