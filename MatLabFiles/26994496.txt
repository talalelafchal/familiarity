Change figures into ppt slides
<p>I need to present MATLAB plots and figures in the form of ppt slides.</p> <p>I tried out using the Publish option in MATLAB,but it is showing all the figures in a single slide. I am looking out for other options so that each figure will be placed in an individual slide. </p> <p>Is there any other option or how to write the program in MATLAB?</p> <p>Can you guide me</p>
<p>Break your code into <a href="http://www.mathworks.com/help/matlab/matlab_prog/run-sections-of-programs.html" rel="nofollow">sections</a>, each section of your published code will get its own slide. This really should be mentioned in the documentation but I don't see it.</p> <p>Compare the publishing outputs from the following:</p> <pre><code>% One Slide x = 1:10; y = 1:10; h.f1 = figure(); plot(x,y); h.f2 = figure(); plot(x,y); </code></pre> <p>vs</p> <pre><code>%% % 3 Slides x = 1:10; y = 1:10; %% h.f1 = figure(); plot(x,y); %% h.f2 = figure(); plot(x,y); </code></pre>
<p>You could save the figures as .eps files and simply include those in your ppt. For example:</p> <pre><code>print(h.f1,'-depsc2','NameOfFigure1') print(h.f2,'-depsc2','NameOfFigure2') </code></pre> <p>Alternatively you can look at: <a href="http://www.mathworks.com/matlabcentral/answers/99150-is-there-an-example-of-using-matlab-to-create-powerpoint-slides" rel="nofollow">http://www.mathworks.com/matlabcentral/answers/99150-is-there-an-example-of-using-matlab-to-create-powerpoint-slides</a></p>
