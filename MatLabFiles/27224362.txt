Reading numeric data from a text file to one column vector in Matlab
<p>I have a text file <code>a.text</code> of the form </p> <pre><code> 2 3 4 1 5 6 </code></pre> <p>Now, I want to read this text file using Matlab and make a column vector <code>b</code> of the form :</p> <pre><code>b = 2 3 4 1 5 6 </code></pre> <p>The original text file contains <code>100*100</code> numbers. Could anyone help? </p>
<p>Use <a href="http://in.mathworks.com/help/matlab/ref/textread.html" rel="nofollow">textread</a></p> <pre><code>b = textread('a.txt','%d'); </code></pre>
