Make plot to start at 0
<p>How to make lines start at 0, when plotting a matrix using the <code>plot</code> function?</p> <p>To be more precise, this is what I want to solve: <img src="https://i.stack.imgur.com/0olPU.png" alt="enter image description here"></p>
<p>See the <a href="http://www.mathworks.it/help/matlab/ref/plot.html" rel="nofollow">help</a> for the <code>plot</code> command: When using the plot command with a single argument, Matlab implicitly plots the argument versus its index, so <code>plot([3,5,9])</code> is equivalent to <code>plot([1,2,3], [3,5,9])</code>. More in general, <code>plot(y)</code> corresponds to <code>plot(1:length(y), y)</code> for a vector and <code>plot(M)</code> corresponds to <code>plot(1:size(M, 1), M)</code> for a matrix.</p> <p>To plot starting from zero, you should thus do <code>plot(0:length(y) - 1, y)</code> for vectors, or <code>plot(0:size(M, 1) - 1, M)</code> for matrices, as in your case.</p>
<p>Maybe with "xlim" will be enought.</p> <p>It this case you can just write:</p> <pre><code>xlim([0,32]); </code></pre> <p>In a more generic case you might need to use:</p> <pre><code>a1=min(x); a2=max(x); xlim([a1,a2]) </code></pre>
