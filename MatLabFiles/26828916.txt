Matlab:How to find the indices of rows without any zero in a matrix?
<p>How to find the indices of rows without any zero in a matrix?</p> <p>Example:</p> <pre><code>A = [ 14 0 6 9 8 17 85 14 1 3 0 99 0 0 0 0 0 0 29 4 5 8 7 46 0 0 0 0 0 0 17 0 5 0 0 49 ] </code></pre> <p>the desired result :</p> <pre><code>V =[4] </code></pre>
<p>Since Adiel did not post an answer, I'll make their comment a CW: the command</p> <pre><code>V = find(all(A,2)) </code></pre> <p>does the job, because <code>all(A,2)</code> processes every row, returning 1 if there are any nonzero entries. Then <code>find</code> returns the indices of nonzero entries, which are the desired row numbers.</p> <p>Similarly, <code>V = find(all(A,1))</code> works column-wise. </p>
