How to plot the coordinates of the nonzero elements in a matrix?
<p>I have a 418x284 matrix filled with 0s and 1s and want to plot a graph where the points are all the location of one's and the x and y coordinates are (0 to 284, 0 to -418).<br> How can I go about doing this? Thank you for all your help!</p>
<p>You can also use <a href="http://www.mathworks.co.uk/help/matlab/ref/spy.html" rel="nofollow noreferrer"><code>spy()</code></a> but you need to adjust the tick labels.</p> <p>Suppose you have the following matrix:</p> <pre><code>A = rand(418,284)&gt;.7; </code></pre> <p>Then:</p> <pre><code>spy(A) </code></pre> <p><img src="https://i.stack.imgur.com/B4uqq.png" alt="enter image description here"></p> <p>Adjusting the labels:</p> <pre><code>yticks = get(gca, 'YTick'); yticks(yticks ~= 0) = -yticks(yticks ~= 0); set(gca, 'YTickLabel', yticks) </code></pre> <p><img src="https://i.stack.imgur.com/t9vUw.png" alt="enter image description here"></p>
<p>Let <code>mat</code> by your binary matrix. You can get the coordinates of the non-zero elements using <code>find</code>:</p> <pre><code>[I,J] = find(mat) plot(I, J); </code></pre> <p><strong>Note</strong> that the convention for axes is different between images and plots in Matlab. The above code assumes <code>I</code> is the rows index (top to bottom) and <code>J</code> is the columns index (left to right).</p> <p>A working example:</p> <pre><code>mat=eye(10); [I, J]=find(mat); subplot(1,2,1), imshow(mat) subplot(1,2,2), plot(I, J); </code></pre> <p>Result:</p> <pre><code> Binary image Non-zero pixels location </code></pre> <p><img src="https://i.stack.imgur.com/XEO7z.png" alt="enter image description here"></p>
