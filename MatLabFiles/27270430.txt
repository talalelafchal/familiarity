<p>Let your matrix be denoted as <code>M</code>. Some possible approaches are:</p> <ol> <li><p>Use <a href="http://es.mathworks.com/help/matlab/ref/num2str.html" rel="nofollow"><code>num2str</code></a> with a format specifier to avoid spaces:</p> <pre><code>result = num2str(reshape(M.',1,[]), '%i'); </code></pre> <p>Change the format specifier <code>'%i'</code> to something else if needed.</p></li> <li><p>Use <a href="http://es.mathworks.com/help/matlab/ref/mat2str.html" rel="nofollow"><code>mat2str</code></a> and remove unwanted characters with <a href="http://es.mathworks.com/help/matlab/ref/regexprep.html" rel="nofollow"><code>regexprep</code></a>:</p> <pre><code>result = regexprep(mat2str(M),'(\[|\]|\;|\s)','') </code></pre></li> <li><p>If your matrix contains only single digits: convert each number into its ASCII code and then cast to <code>char</code>:</p> <pre><code>result = char(reshape(M.',1,[])+'0'); </code></pre></li> </ol>
