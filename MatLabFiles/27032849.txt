How can I change the background color of the image?
<p>I want to <code>remove/change</code> the background color of an image in Matlab.</p> <p>Anyone know how to do this?</p> <p>Here is an example image, I want to remove the green background color. </p> <p><img src="http://cdn4.frip.in/wp-content/uploads/2013/05/custom-widget-area-for-wordpress.jpg" alt=""></p>
<p>Simplest answer would be,</p> <pre><code>c = [70 100 70]; thresh = 50; A = imread('image.jpg'); B = zeros(size(A)); Ar = A(:,:,1); Ag = A(:,:,2); Ab = A(:,:,3); Br = B(:,:,1); Bg = B(:,:,2); Bb = B(:,:,3); logmap = (Ar &gt; (c(1) - thresh)).*(Ar &lt; (c(1) + thresh)).*... (Ag &gt; (c(2) - thresh)).*(Ag &lt; (c(2) + thresh)).*... (Ab &gt; (c(3) - thresh)).*(Ab &lt; (c(3) + thresh)); Ar(logmap == 1) = Br(logmap == 1); Ag(logmap == 1) = Bg(logmap == 1); Ab(logmap == 1) = Bb(logmap == 1); A = cat(3 ,Ar,Ag,Ab); imshow(A); </code></pre> <p><img src="https://i.stack.imgur.com/wQ3Bj.png" alt="enter image description here"></p> <p>You should change <code>c</code> (background color) and <code>thresh</code> (threshold for <code>c</code>) and find the best that fits your background.</p> <p>You can define <code>B</code> as your new background image. Fr example adding <code>Bb(:,:) = 255;</code>will give you a blue background.</p> <p><img src="https://i.stack.imgur.com/wc3AT.png" alt="enter image description here"></p> <p>You can even define <code>B</code> as an image.</p> <p>In order to detect background you could find the color that is most used in the image, but that is not necessarily background I think. </p>
