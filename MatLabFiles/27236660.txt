matlab sort 2 columns: column 1 descending and column 2 ascending
<p>I have a 3 column array like this: </p> <pre><code>A = [6 -1 0; 6 0 3; 1 4 3; 1 2 5]; </code></pre> <p>I need the first column to be in descending order and the second column to be in ascending order with the third column changing with the other two columns (i.e. they are not sorted) like this: </p> <pre><code>B = [6 -1 0; 6 0 3; 1 2 5; 1 4 3]; </code></pre>
<p>If the third column is to be unchanged,</p> <pre><code> B = A; B(:,1) = sort(B(:,1),'descend'); B(:,2) = sort(B(:,2),'ascend'); </code></pre> <p>If you want the third column to change with the second then,</p> <pre><code>[B(:,2),ind] = sort(B(:,2),'ascend'); B(:,3) = B(ind,3); B = 6 -1 0 6 0 3 1 2 5 1 4 3 </code></pre>
<p>If you're looking to keep the rows intact as records, and sort them based on column 1 and then column 2, use <code>sortrows()</code>, with negative column indexes to indicate columns whose values are to be sorted in descending order.</p> <pre><code>B = sortrows(A, [-1 2]) </code></pre>
